# Python 内置函数

> 原文: [https://www.geeksforgeeks.org/python-built-in-functions/](https://www.geeksforgeeks.org/python-built-in-functions/)

Python 提供了许多内置函数，简化了代码的编写。在本文中，您将找到 Python 所有内置函数的列表。

## Python 内置函数列表

| 函数名 | 描述 |
| --- | --- |
| `abs()` | 返回一个数字的绝对值 |
| `all()` | 如果给定表(列表、字典、元组、集合等)的所有元素都为真，则返回真，否则返回假 |
| `any()` | 如果给定表(列表、字典、元组、集合等)的任何元素为真，则返回真，否则返回假 |
| `ascii()` | 返回包含对象的可打印表示形式的字符串 |
| `bin()` | 将整数转换为二进制字符串 |
| `bool()` | 将值返回或转换为布尔值，即真或假 |
| `bytearray()` | 返回一个字节数组对象 |
| `bytes()` | 将对象转换为给定大小和数据的不可变字节表示的对象 |
| `callable()` | 如果传递的对象看起来是可调用的，则返回真 |
| `chr()` | 返回一个字符串，该字符串表示 Unicode 代码点为整数的字符 |
| `classmethod()` | 返回给定函数的类方法 |
| `compile()` | 返回一个 Python 代码对象 |
| `complex()` | 创建复数 |
| `delattr()` | 从对象中删除命名属性 |
| `dict()` | 创建 Python 字典 |
| `dir()` | 返回任何对象的属性和方法列表 |
| `divmod()` | 取两个数字并返回一对由它们的商和余数组成的数字 |
| `enumerate()` | 向 iterable 添加计数器，并以枚举对象的形式返回它 |
| `eval()` | 解析传递给它的表达式，并在程序中运行 python 表达式(代码) |
| `exec()` | 用于程序的动态执行 |
| `filter()` | 在测试序列中每个元素是否为真的函数的帮助下过滤给定的序列 |
| `float()` | 从数字或字符串返回浮点数 |
| `format()` | 格式化指定的值 |
| `frozenset()` | 返回不可变的 frozenset |
| `getattr()` | 访问对象的属性值 |
| `globals()` | 返回当前全局符号表的字典 |
| `hasattr()` | 检查对象是否具有给定的命名属性，如果存在，则返回 true |
| `hash()` | 将数据编码为不可识别的值 |
| `help()` | 显示模块、函数、类、关键字等的文档 |
| `hex()` | 将整数转换为相应的十六进制形式 |
| `id()` | 返回对象的标识 |
| `input()` | 将用户输入作为字符串 |
| `int()` | 将给定基数的数字转换为十进制 |
| `isinstance()` | 检查对象是否属于某个类 |
| `issubclass()` | 检查一个类是否是另一个类的子类 |
| `iter()` | 将 iterable 转换为迭代器 |
| `len()` | 返回对象的长度 |
| `list()` | 用 Python 创建列表 |
| `locals()` | 返回当前本地符号表的字典 |
| `map()` | 将给定函数应用于给定可迭代表的每一项后，返回结果的 map 对象(迭代器) |
| `max()` | 返回可迭代表中最大的项或两个或多个参数中最大的一个 |
| `memoryview()` | 返回参数的内存视图 |
| `min()` | 返回可迭代表中的最小项或两个或多个参数中的最小项 |
| `next()` | 从迭代器接收下一项 |
| `object()` | 返回一个新对象 |
| `oct()` | 以字符串格式返回整数的八进制表示。 |
| `open()` | 打开一个文件并返回它的对象 |
| `ord()` | 返回传递的参数的等效 Unicode 值 |
| `pow()` | 计算一个数的幂 |
| `print()` | 将输出打印到控制台 |
| `property()` | 创建类的属性 |
| `range()` | 生成一系列数字 |
| `repr()` | 返回对象的表版本 |
| `reversed()` | 返回以相反顺序访问给定序列的迭代器 |
| `round()` | 舍入到给定的位数并返回浮点数 |
| `set()` | 将任何可迭代元素转换为具有不同元素的可迭代元素序列 |
| `setattr()` | 为对象属性赋值 |
| `slice()` | 返回切片对象 |
| `sorted()` | 以排序方式返回包含元素的列表，而不修改原始序列 |
| `staticmethod()` | 将消息转换为静态消息 |
| `str()` | 返回对象的字符串版本 |
| `sum()` | 总结列表中的数字 |
| `super()` | 返回超类的临时对象 |
| `tuple()` | 用 Python 创建一个元组 |
| `type()` | 返回对象的类型 |
| `vars()` | 返回模块、类、实例或任何其他对象的 `__dict__` 属性 |
| `zip()` | 映射多个容器的相似索引 |
| `__import__()` | 在运行时导入模块 |
