# 使用 Python 中的街机库显示降雪

> 原文: [https://www.geeksforgeeks.org/display-snowfall-using-arcade-library-in-python/](https://www.geeksforgeeks.org/display-snowfall-using-arcade-library-in-python/)

在本文中，我们将使用`arcade` Python 包来显示降雪。`arcade`库是一个现代 Python 模块，广泛用于开发具有引人注目的图形和声音的 2D 视频游戏。它是一个面向对象的库，可以像任何其他 Python 包一样安装。

## 分步方法

*   第一步也是最重要的一步是将`arcade`库导入到你的`Python IDE`中。同时导入`random`和`math`模块，因为降雪是随机显示在屏幕上的。

```py
# Import required modules
import random
import math
import arcade
```
