# Golang 和 Rust 的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-golang-and-rust/](https://www.geeksforgeeks.org/difference-between-golang-and-rust/)

## 介绍

`Golang` 是一种开源编程语言，是一种静态类型的编译语言。`Go` 语言是由谷歌的罗伯特·格里森、罗布·派克和肯·汤普森开发的。它是由谷歌在 2007 年推出的，并于 2009 年首次推出。它支持并发性，这为成功构建并行系统提供了很高的机会。使用 `Go` 语言的公司有谷歌、优步、`Twitch`、`Dailymotion`、`Dropbox` 等。

## Go 语言优势

*   `easy to use`
*   `Support Google`
*   `Good documentation`
*   `Simple code`
*   `Good interoperability with C language`

## Go 语言的劣势

*   `Poor versatility`
*   `Not a system language.`
*   `Insufficient invariance`
*   `Poor performance`
*   `Generic is not supported.`

## Rust 语言介绍

`Rust` 是一种多范式开源系统编程语言，特别安全并发。`Rust` 编程语言是由 `Mozilla research` 的 `Graydon Hoare` 开发的，它早在 2010 年就已经存在了。它缺乏并发性，因此在开发并行系统时不太受欢迎。使用 `Rust` 语言的公司有 `Mozilla`、`Dropbox`、`Sentry`、`Postmates`、`Brilliant` 等。

## Rust 语言优势

*   它提供快速调试和测试
*   这是理想的内存安全
*   它具有可预测的运行时行为
*   这样可以避免崩溃
*   它节省了调试时间

## Rust 语言的劣势

*   学锈比去
*   `Difficult to compile`
*   `Slow down and be less efficient.`
*   `Maintenance is more difficult.`
*   `Lack of efficient garbage collection`

## Go 与 Rust 的区别

| **Go** | **Rust** |
| :--- | :--- |
| `Golang` 是一种开源编程语言和静态类型编译语言。 | `Rust` 是一种多范式开源系统编程语言，它特别安全并发。 |
| 它可以用来更快地开发应用程序。 | 你不能像 `Go` 那样用它来更快地开发应用程序。 |
| `Go` 编程的编译速度比 `Rust` 快。 | `Rust` 编程的编译速度比 `Go` 慢。 |
| 这是一种非常简单的语言，易于理解。 | `Rust` 语言有点复杂，难以理解。 |
| `Go` 语言的功能比 `Rust` 语言少。 | 与 `Go` 语言相比，`Rust` 语言有很多特性和功能。 |
| `Go` 是一种比 `Rust` 更流行的语言。 | `Rust` 是一种不如 `Go` 流行的语言。 |
| `Go` 语言支持并发，这为成功构建并行系统提供了很高的机会。 | `Rust` 语言缺乏并发性，因此在开发并行系统时不太受欢迎。 |
