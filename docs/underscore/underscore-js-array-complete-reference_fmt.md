# Underscore.js 数组完整引用

> 原文：[https://www.geeksforgeeks.org/underscore-js-array-complete-reference/](https://www.geeksforgeeks.org/underscore-js-array-complete-reference/)

Underscore.js 是一个轻量级的 JavaScript 库，由 **Jeremy Ashkenas** 编写，为我们日常常见编程任务中的各种用例提供实用函数。

数组是用于存储不同元素的单个变量。当我们想要存储元素列表并通过单个变量访问它们时，通常会用到它。Underscore.js 提供了广泛的数组函数，用于操作给定的数组值。
