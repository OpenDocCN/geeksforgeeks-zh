# 如何用 PHP 写评论？

> 原文：[https://www.geeksforgeeks.org/how-to-write-comments-in-php/](https://www.geeksforgeeks.org/how-to-write-comments-in-php/)

在本文中，我们将通过一些简单的基础语法来介绍如何在 PHP 代码中添加注释。注释有助于理解复杂的代码，在代码调试期间也十分有用。我们使用注释来阐明代码的功能。

基本上，单行注释以 `//` 开头。

## 例 1

```php
<?php

// 输出 "Hello GeeksforGeeks"
echo "Hello GeeksforGeeks";

?>
```

**输出：**

```
Hello GeeksforGeeks
```

## 示例 2

多行注释以 `/*` 开头，以 `*/` 结尾。

```php
<?php

echo "Hello Geeks"; 
/* 这将打印信息
   "Hello geeks" */

?>
```

**输出：**

```
Hello Geeks
```
