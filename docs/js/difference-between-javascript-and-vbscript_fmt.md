# JavaScript 和 VBScript 的区别

> 原文: [https://www.geeksforgeeks.org/difference-between-javascript-and-vbscript/](https://www.geeksforgeeks.org/difference-between-javascript-and-vbscript/)

[JavaScript](https://www.geeksforgeeks.org/introduction-to-javascript/) 是一种轻量级、面向对象的脚本语言，用于在网页内创建具有交互效果的动态 HTML 页面。它是一种解释的脚本语言，其代码仅在网络浏览器中运行。它也被称为浏览器语言，可以用于客户端开发和服务器端开发。它是由网景公司的布兰登·艾奇开发的，于 1995 年首次发布。

**JavaScript 示例:**

## JavaScript 示例

```
<script type="text/javascript">
    document.write("Hello geeks, greeting from GeeksforGeeks")
</script>
```

[VBScript](https://www.geeksforgeeks.org/vbscript-introduction/) 是微软推出的一种主动脚本语言。它是基于 Visual Basic 开发的，也称为 Visual Basic 脚本语言。它不支持所有浏览器，只支持 Internet Explorer。它通常用于微软环境，并使用组件对象模型来访问系统对象和环境变量。

**VBScript 示例:**

## VBScript 示例

```
<script type="text/vbscript">
    document.write("Hello geeks, greeting from GeeksforGeeks")
</script>
```

**JavaScript 与 VBScript 的区别:**

| 序号 | JavaScript | VBScript |
| :--- | :--- | :--- |
| 1. | 由 Netscape 开发。 | 由 Microsoft 开发。 |
| 2. | 作为客户端脚本语言运行。 | 既可作为服务器端脚本语言，也可作为客户端脚本语言运行。 |
| 3. | 支持所有 Web 浏览器，并且是大多数浏览器的默认脚本语言。 | 不支持所有浏览器，仅支持 Internet Explorer。 |
| 4. | 对各种操作使用相同的运算符。 | 相反，它对各种操作使用不同的运算符。 |
| 5. | 使用花括号 `{}` 声明函数。 | 使用 `Function` 和 `End Function` 声明函数。 |
| 6. | JavaScript 文件的扩展名为 `.js`。 | VBScript 文件的扩展名为 `.vbs` 或 `.vba`。 |
| 7. | 学习 JavaScript 更简单、更容易。 | 与 JavaScript 相比，初学者学习 VBScript 更困难。 |
