# TypeScript `valueOf()` 方法

> 原文：[https://www.geeksforgeeks.org/typescript-number-valueof-method/](https://www.geeksforgeeks.org/typescript-number-valueof-method/)

TypeScript 中的 `valueOf()` 方法用于返回指定数字对象的基元值。

**语法：**
```
number.valueOf()
```

**参数值：** 此方法不接受任何参数。

**返回值：** TypeScript 中的 `valueOf()` 方法返回指定数字对象的基元值。

以下示例说明了 `valueOf()` 函数在 TypeScript 中的工作：

## JavaScript 示例
```
//valueOf()
var num3 = new Number(30);
console.log(num3.valueOf());
```
