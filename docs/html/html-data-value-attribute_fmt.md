
# HTML 数据值属性

> 原文: [https://www.geeksforgeeks.org/html-data-value-attribute/](https://www.geeksforgeeks.org/html-data-value-attribute/)

**HTML 数据值属性**用于指定元素内容的机器可读翻译。
**语法:**

```html
<data attribute> Contents... </data> 
```

**示例:**

## 超文本标记语言

```html
<!DOCTYPE html>
<html>

<head>
    <title>data 标签</title>
</head>

<body>
    <h1 style="color:green;">GeeksforGeeks</h1>
    <h2> HTML 数据值属性 </h2>

<p><b><i>GeeksforGeeks 主题列表:</i></b></p>

<ul>
        <li>
            <data value="009">
              数据结构
          </data>
        </li>
        <li>
            <data value="010">
              算法
          </data>
        </li>
        <li>
            <data value="011">
              HTML
          </data>
        </li>
        <li>
            <data value="019">
              操作系统
          </data>
        </li>
        <li>
            <data value="110">
              计算机网络
          </data>
        </li>
        <li>
            <data value="111">
              数据库管理系统
          </data>
        </li>
    </ul>
</body>

</html>
```
