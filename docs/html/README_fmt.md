
# HTML

HTML (HyperText Markup Language) 是一种用于创建网页的标准标记语言。它描述了一个网站的结构和内容。

## 基本结构

HTML 文档通常由以下部分组成：

```html
<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>
```

## 标签

HTML 使用标签来定义网页内容。标签通常成对出现，例如 `<p>` 和 `</p>`。

## 属性

HTML 标签可以包含属性，用于提供额外的信息。例如，`<a>` 标签的 `href` 属性用于指定链接的目标。

```html
<a href="https://www.example.com">This is a link</a>
```

## 语义化标签

HTML5 引入了许多语义化标签，如 `<header>`, `<footer>`, `<article>`, `<section>` 等，用于更好地描述网页内容。

## 响应式设计

响应式设计是指网页能够根据不同的设备屏幕尺寸自动调整布局和内容。

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## 总结

HTML 是构建网页的基础，掌握 HTML 是成为一名网页开发者的第一步。
