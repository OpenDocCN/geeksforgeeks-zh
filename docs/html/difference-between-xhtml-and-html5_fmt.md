# XHTML 和 HTML5 的区别

> 原文: [https://www.geeksforgeeks.org/difference-between-xhtml-and-html5/](https://www.geeksforgeeks.org/difference-between-xhtml-and-html5/)

**1. [`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/):**
[`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) 代表可扩展超文本标记语言。它可以被认为是 [`XML`](https://www.geeksforgeeks.org/xml-introduction/) 标记语言的一部分，这是因为 [`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) 同时具有 [`XML`](https://www.geeksforgeeks.org/xml-introduction/) 和 [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 的特性。[`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) 是从 [`XML`](https://www.geeksforgeeks.org/xml-introduction/) 和 [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 扩展而来的。[`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) 可以被认为是 [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 的更好版本。

**2. [`HTML`](https://www.geeksforgeeks.org/html5-introduction/):**
[`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 是超文本标记语言，是互联网上使用最广泛的语言。[`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 用于创建网页并将它们从一个网页链接到另一个网页。请注意 [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 不是[编程语言](https://www.geeksforgeeks.org/introduction-to-programming-languages/)，它是一种标记语言。我们可以使用不同的其他技术，如 [`CSS`](https://www.geeksforgeeks.org/css-tutorial/) 和 [`JavaScript`](https://www.geeksforgeeks.org/javascript-tutorial/) 来赋予由 [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 开发的页面新的外观。

**[`HTML`](https://www.geeksforgeeks.org/html5-introduction/) 和 [`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) 的区别:**

| s No. | [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) | [`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) |
| --- | --- | --- |
| 1. | [`HTML`](https://www.geeksforgeeks.org/html5-introduction/) stands for hypertext markup language. | [`XHTML`](https://www.geeksforgeeks.org/xhtml-introduction/) stands for extensible hypertext markup language. |
| 2. | It was developed by Tim Berners Lee. | Developed by [`W3C`](https://www.w3.org/), the World Wide Web Consortium. |
| 3. | Developed in 1991. | Published in 2000. |
| 4. | Derived from [`SGML`](https://www.geeksforgeeks.org/sgml-full-form/). | It is extended from [`XML`](https://www.geeksforgeeks.org/xml-introduction/) and [`HTML`](https://www.geeksforgeeks.org/html5-introduction/). |
| 5. | The format is document file format. | Format is a markup language. |
| 6. | All labels and attributes do not have to be uppercase or lowercase. | In this case, each label and attribute should be lowercase. |
| 7. | [`Doctype`](https://www.geeksforgeeks.org/html-doctype/) does not need to be written at the top. | It is very necessary to write [`Doctype`](https://www.geeksforgeeks.org/html-doctype/) at the top of the file. |
| 8. | Labels don't have to be closed in the order of opening. | Tags need to be closed in the order in which they are opened. |
| 9. | There is no need to mention quotation marks when using attributes. For example, for `<font size=20>`. | When using attributes, quotation marks must be mentioned. For example, for `<font size="20">`. |
| 10. | The file extension used by is `.html`. | The file extension is `.xhtml`. |
