
# HTML 5 figcaption Tag

> [HTML 5 figcaption Tag](https://www.geeksforgeeks.org/html-5-figcaption-tag/)

HTML 中的 `<figcaption>` 标记用于为文档中的图形元素设置标题。这个标签在 HTML5 中是新的。

## 超文本标记语言

```html
<!DOCTYPE html>
<html>
<body>

<h1>GeeksforGeeks</h1>
<h2><figcaption Tag></h2>
<figure>
<img src="https://media.geeksforgeeks.org/wp-content/uploads/geeks-25.png" alt="gfglogo" style="width:50%">
<!--HTML figcaption tag starts here-->
<figcaption>
GeeksforGeeks Logo
</figcaption>
<!--HTML figcaption tag ends here-->
</figure>

</body>
</html>
```
