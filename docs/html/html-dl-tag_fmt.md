
# HTML dl 标记

> [HTML dl 标签](https://www.geeksforgeeks.org/html-dl-tag/)

HTML 中的 `<dl>` 标签用来表示描述列表。此标签与 `<dt>` 和 `<dd>` 标签一起使用。在 HTML4.1 中定义定义列表，在 HTML5 中定义描述列表。

## 超文本标记语言

```html
<!DOCTYPE html>

<html>

<body>

<h1>GeeksforGeeks</h1>
        <h2>dl Tag</h2>
        <!-- HTML dl tag -->
        <dl>
            <dt>GeeksforGeeks</dt>
            <dd>A Computer Science Portal For Geeks</dd>
        </dl>

</body>

</html>
```
