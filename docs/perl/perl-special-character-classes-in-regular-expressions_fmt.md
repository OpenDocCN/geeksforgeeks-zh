# Perl 正则表达式中的特殊字符类

> 原文: [https://www.geeksforgeeks.org/perl-special-character-classes-in-regular-expressions/](https://www.geeksforgeeks.org/perl-special-character-classes-in-regular-expressions/)

在 Perl 中实现了许多不同的字符类，其中一些使用非常频繁，以至于为它们创建了一个特殊的序列。创建特殊序列的目的是使代码更易读、更简短。Perl 中的**特殊字符类**如下:

## 1. Digit `\d[0-9]`

`\d` 用于匹配任何数字字符，等价于 `[0-9]`。正则表达式 `/\d/` 将匹配单个数字。`\d` 被标准化为“digit”。主要优点是用户可以更容易地以更短的形式书写和阅读。有两种方法可以使用这个特殊字符类。让我们通过一个例子来更好地理解如何匹配字符串。

**示例:**

```perl
    /#[MNOPQ]-\d\d\d/
```

上面给定的字符串将匹配如下。

```perl
    #M-12345
    #N-66666
```

在这里，我们也可以通过把它放在字符类上来使用量词。

**示例:**

```perl
    /#[MNOPQ]-\d{5}/
```

上面给出的例子与前面的正则表达式相同，它允许破折号后有任意数量的数字，可以写成 `/#[MNOPQ]-\d+/`。

第二种方法用于较大的字符类。`\d` 放在方括号内，匹配单字符数字。

**示例:**

```perl
    [\dABCDEFDEFGHIJKLMN]
```

可以匹配单个数字，也可以匹配任何大写字母 A、B、C、D、E、F、G、H、I、J、K、L、M 或 N。可以使用破折号(-)以较短的形式书写。然后会是这样的:

```perl
    [\dA-N]
```

## 2. POSIX 字符类

POSIX 是维护操作系统之间兼容性的标准，并定义了应用程序编程接口（API），包括命令行 Shell 和实用程序接口。它还指定了许多具有名称的**“字符组”**（如 alpha、alnum、ascii、blank 等）。POSIX 字符类总是以 `[:class:]` 的形式存在，其中 class 是名称，`[:` 和 `:]` 是分隔符。POSIX 字符类总是出现在带括号的字符类内部。这些类是列出一组字符的一种方便且具有解释性的方式。

**语法:**

```perl
    $string =~ /[[:class:]]/
```

这里的类可以是 alpha、alnum、ascii 等。

POSIX 字符类支持更大的括号字符类，如下所示:

```perl
    [01[:Class:]%]
```

这里它将匹配“0”、“1”和任何字符类以及百分号。Perl 为不同的 POSIX 字符类提供支持，如下表所示:

| 类 | 描述 |
| --- | --- |
| alpha | 任何字母字符（“[A-Za-z]”） |
| alnum | 任何字母数字字符（“[A-Za-z0-9]”）。 |
| ascii | ASCII 字符集中的任何字符。 |
| blank | 空格或水平制表符 |
| cntrl | 任何控制字符。 |
| digit | 任何十进制数字（“[0-9]”）。 |
| graph | 任何可打印字符，不包括空格 |
| lower | 任何小写字符（“[a-z]”） |
| print | 任何图形字符 |
| punct | 任何空白字符 |
| space | 任何空白字符 |
| upper | 任何大写字符（“[A-Z]”） |
| xdigit | 任何十六进制数字（“[0-9a-fA-F]”） |
| word | 一个 Perl 扩展（“[A-Za-z0-9_]”），相当于“\w” |

## 3. Word character `\w[0-9a-zA-Z_]`

`\w` 属于单词字符类。`\w` 匹配任何单个字母数字字符，可以是字母字符、十进制数字或标点符号字符，例如下划线（`_`）。它只匹配单字符单词，而不是整个单词。如果你想匹配整个单词，请使用 `\w+`。

## 4. Whitespace `\s[\t\n\f\r ]`

字符类 `\s` 将匹配单个字符，即空白字符。它还将匹配 5 个字符，即 `\t`（水平制表符）、`\n`（换行符）、`\f`（换页符）、`\r`（回车符）和空格。在 Perl v5.18 中，引入了一个新字符，它匹配 `\cK`（垂直制表符）。

## 5. 否定字符类 `\D`, `\W`, `\S`

世界上有超过 110,000 个 Unicode 字符。要否定一个字符类，只需使用脱字符（`^`）符号。它将否定符号后的指定字符或范围。在否定字符类中，我们使用 `[^\d]` 来否定数字 0 到 9。但我们可以简单地使用 `\D` 来代替 `[^\d]` 来否定数字 0 到 9。下表说明了特殊的否定字符类:

| 字符类 | 否定 | 意义 | 描述 |
| --- | --- | --- | --- |
| `\d` | `\D` | `[^\d]` | 匹配任何非数字字符 |
| `\s` | `\S` | `[^\s]` | 匹配任何非空白字符 |
| `\w` | `\W` | `[^\w]` | 匹配任何非“单词”字符 |

## 6. Unicode 字符类

Unicode 是对“所有”现有字符的定义，Unicode 标准为每个字符提供了一个唯一的编号，并且它是平台无关的。世界上有超过 100,000 个字符，每个字符被描述为一个字符点。但有些字符被分组在一起。

**语法:**

```perl
    \p{...any character...}
```

该语法用于匹配其中一个组中的单个字符。如果您需要匹配除指定字符以外的任何内容，那么您可以使用相应的 `\P{…任何字符…}` 表达式。
