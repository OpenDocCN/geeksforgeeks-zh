# jQWidgets jqxTreeMap 完整参考

> 原文: [https://www.geeksforgeeks.org/jqwidgets-jqxtreemap-complete-reference/](https://www.geeksforgeeks.org/jqwidgets-jqxtreemap-complete-reference/)

`jqxTreeMap` 将分层数据显示为一组嵌套的矩形。给树的每个分支一个矩形，然后用代表子分支的更小的矩形平铺。叶节点的矩形面积与数据上的指定尺寸成比例。
