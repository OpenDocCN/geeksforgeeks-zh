# jQWidgets jqxListMenu showNavigationArrows 属性

> 原文:[https://www . geesforgeks . org/jqwidgets-jqxlistmenu-shownavigationarrows-property/](https://www.geeksforgeeks.org/jqwidgets-jqxlistmenu-shownavigationarrows-property/)

jQWidgets 是一个 JavaScript 框架，用于为 PC 和移动设备制作基于 web 的应用程序。它是一个非常强大、优化、独立于平台并且得到广泛支持的框架。jqxListMenu 是一个 jQuery 小部件，用于显示有序和无序列表的集合。我们可以通过使用有序列表和无序列表来创建嵌套列表。

**showNavigationArrows 属性**用于指定导航箭头是否仅显示具有嵌套列表的列表项。它接受布尔类型值，默认值为真。

**语法:**

*   设置 showNavigationArrows 属性。

    ```html
    $('selector').jqxListMenu({ 
        showNavigationArrows: Boolean
    });
    ```

*   返回 showNavigationArrows 属性。

    ```html
    var showNavigationArrows = $('selector')
        .jqxListMenu('showNavigationArrows');
    ```
