# jQWidgets jqxSlider 完整参考

> 原文: [https://www.geeksforgeeks.org/jqwidgets-jqxslider-complete-reference/](https://www.geeksforgeeks.org/jqwidgets-jqxslider-complete-reference/)

**jqxSlider** 代表一个灵活的 jQuery 滑块，允许用户通过沿轨迹移动拇指从一系列值中进行选择。该小部件在外观方面是完全可定制的，并提供了许多配置选项，如鼠标滚轮和键盘支持、平滑或基于步长的滑块以及对范围滑块的支持。
