# jQWidgets jqxHeatMap 完整参考

> 原文：[https://www.geeksforgeeks.org/jqwidgets-jqxheatmap-complete-reference/](https://www.geeksforgeeks.org/jqwidgets-jqxheatmap-complete-reference/)

`jqxHeatMap` 用于显示数据的图形表示，该图形表示使用颜色编码系统来表示不同的值。此外，它还用于将表格数据值显示为梯度而不是数字。低值和高值是具有不同渐变的不同颜色。
