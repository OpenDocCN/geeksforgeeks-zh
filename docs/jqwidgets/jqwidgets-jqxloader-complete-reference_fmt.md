> 原文：[https://www.geeksforgeeks.org/jqwidgets-jqxloader-complete-reference/](https://www.geeksforgeeks.org/jqwidgets-jqxloader-complete-reference/)

`jqxloader`代表一个显示内置加载元素的 jQuery 小部件。加载元素可以是仅图标、仅文本或图标和文本的组合。它可以用来显示加载元素。
