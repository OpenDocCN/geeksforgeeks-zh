# MySQL 中的 ASIN() 和 ACOS() 函数

> 原文: [https://www.geeksforgeeks.org/asin-and-acos-function-in-mysql/](https://www.geeksforgeeks.org/asin-and-acos-function-in-mysql/)

## ASIN() 函数

MySQL 中的 `ASIN()` 函数用来返回 [-1, 1] 范围内任意数字的反正弦。如果数字不在 -1 到 1 的范围内，则返回空值。

### 语法

```sql
ASIN(X)
```

### 参数

`ASIN()` 函数接受一个参数，如下所述。

*   **X** – 我们要计算其反正弦的数字。它应该在 [-1, 1] 的范围内。

### 返回

返回给定数字 `x` 的反正弦。

### 示例-1

使用 `ASIN()` 函数求 1 的反正弦。

```sql
SELECT ASIN(1) AS Asin_Val ;
```

**输出:**

| Asin_Val |
| --- |
| 1.5707963267948966 |

### 示例-2

使用 `ASIN()` 函数对 (0, 1) 范围内的数字进行反正弦运算。

```sql
SELECT ASIN(0.25) AS Asin_Val;
```

**输出:**

| Asin_Val |
| --- |
| 0.25268025514207865 |

### 示例-3

使用 `ASIN()` 函数对 (0, -1) 范围内的数字进行反正弦运算。

```sql
SELECT ASIN(-0.35) AS Asin_Val;
```

**输出:**

| Asin_Val |
| --- |
| -0.35757110364551026 |

### 示例-4

使用 `ASIN()` 函数计算不在 [-1, 1] 范围内的数字的反正弦。

```sql
SELECT ASIN(2.75) AS Asin_Val ;
```

**输出:**

| Asin_Val |
| --- |
| NULL |

### 示例-5

表中数值列的反正弦值。

**表: Number**

| X |
| --- |
| -2 |
| -1 |
| -0.50 |
| 0 |
| 0.5 |
| 1 |
| 2 |

```sql
SELECT X, ASIN(X) AS ArcSin_X FROM Number ;
```

**输出:**

| X | ArcSin_X |
| --- | --- |
| -2 | NULL |
| -1 | -1.5707963267948966 |
| -0.50 | -0.5235987755982989 |
| 0 | 0 |
| 0.5 | 0.5235987755982989 |
| 1 | 1.5707963267948966 |
| 2 | NULL |

## ACOS() 函数

`ACOS()` 函数在 MySQL 中用于返回 [-1, 1] 范围内任意数字的反余弦。如果数字不在 -1 到 1 的范围内，则返回空值。

### 语法

```sql
ACOS(X)
```

### 参数

`ACOS(X)` 函数接受一个参数，在 `ACOS(X)` 函数中 `X` 为参数。让我们借助例子来理解。

*   **X** – 我们要计算其反余弦的数字。它应该在 [-1, 1] 的范围内。

### 返回

返回给定数字 `x` 的反余弦。

### 示例-1

使用 `ACOS()` 函数求 1 的反余弦。

```sql
SELECT ACOS(1) AS Acos_Val;
```

**输出:**

| Acos_Val |
| --- |
| 0 |

### 示例-2

使用 `ACOS()` 函数对 (0, 1) 范围内的数字进行反余弦运算。

```sql
SELECT ACOS(0.75) AS Acos_Val;
```

**输出:**

| Acos_Val |
| --- |
| 0.7227342478134157 |

### 示例-3

使用 `ACOS()` 函数对 (0, -1) 范围内的一个数进行反余弦运算。

```sql
SELECT ACOS(-0.75) AS Acos_Val ;
```

**输出:**

| Acos_Val |
| --- |
| 2.4188584057763776 |

### 示例-4

使用 `ACOS()` 函数对不在 [-1, 1] 范围内的数字进行反余弦运算。

```sql
SELECT ACOS(2.75) AS Acos_Val;
```

**输出:**

| Acos_Val |
| --- |
| NULL |

### 示例-5

表中数值列的反余弦值。

**表: Number**

| X |
| --- |
| -5 |
| -1 |
| -0.50 |
| 0 |
| 0.5 |
| 1 |
| 2 |

```sql
SELECT X, ACOS(X) AS Acos_Val FROM Number;
```

**输出:**

| X | Acos_Val |
| --- | --- |
| -5 | NULL |
| -1 | 3.141592653589793 |
| -0.50 | 2.0943951023931957 |
| 0 | 1.5707963267948966 |
| 0.5 | 1.0471975511965979 |
| 1 | 0 |
| 2 | NULL |
