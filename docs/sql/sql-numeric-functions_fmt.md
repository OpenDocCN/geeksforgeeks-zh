# SQL 数值函数

> 原文: [https://www.geeksforgeeks.org/sql-numeric-functions/](https://www.geeksforgeeks.org/sql-numeric-functions/)

数字功能用于对数字进行运算并返回数字。
以下是 SQL 中定义的数值函数：

## 函数列表

1.  **`ABS()`**: 返回数字的绝对值。
    ```sql
    SELECT ABS(-243.5);
    ```
    **输出:** `243.5`
    ```sql
    SQL> SELECT ABS(-10);
    +--------------------------------------+
    | ABS(10)                                                  |
    +--------------------------------------+
    | 10                                                       |
    +--------------------------------------+
    ```

2.  **`ACOS()`**: 返回数字的反余弦值。
    ```sql
    SELECT ACOS(0.25);
    ```
    **输出:** `1.318116071652818`

3.  **`ASIN()`**: 返回数字的反正弦值。
    ```sql
    SELECT ASIN(0.25);
    ```
    **输出:** `0.25268025514207865`

4.  **`ATAN()`**: 返回数字的反正切值。
    ```sql
    SELECT ATAN(2.5);
    ```
    **输出:** `1.1902899496825317`

5.  **`CEIL()`**: 返回大于或等于数字的最小整数值。
    ```sql
    SELECT CEIL(25.75);
    ```
    **输出:** `26`

6.  **`CEILING()`**: 返回大于或等于数字的最小整数值。
    ```sql
    SELECT CEILING(25.75);
    ```
    **输出:** `26`

7.  **`COS()`**: 返回数字的余弦值。
    ```sql
    SELECT COS(30);
    ```
    **输出:** `0.154251449887584`

8.  **`COT()`**: 返回数字的余切值。
    ```sql
    SELECT COT(6);
    ```
    **输出:** `-3.436353004180128`

9.  **`DEGREES()`**: 将弧度值转换为度数。
    ```sql
    SELECT DEGREES(1.5);
    ```
    **输出:** `85.94366926962348`
    ```sql
    SQL>SELECT DEGREES(PI());
    +------------------------------------------+
    | DEGREES(PI())                                           |
    +------------------------------------------+
    | 180.000000                                              |
    +------------------------------------------+
    ```

10. **`DIV()`**: 用于整数除法。
    ```sql
    SELECT 10 DIV 5;
    ```
    **输出:** `2`

11. **`EXP()`**: 返回 e 的 number 次方。
    ```sql
    SELECT EXP(1);
    ```
    **输出:** `2.718281828459045`

12. **`FLOOR()`**: 返回小于或等于数字的最大整数值。
    ```sql
    SELECT FLOOR(25.75);
    ```
    **输出:** `25`

13. **`GREATEST()`**: 返回表达式列表中的最大值。
    ```sql
    SELECT GREATEST(30, 2, 36, 81, 125);
    ```
    **输出:** `125`

14. **`LEAST()`**: 返回表达式列表中的最小值。
    ```sql
    SELECT LEAST(30, 2, 36, 81, 125);
    ```
    **输出:** `2`

15. **`LN()`**: 返回数字的自然对数。
    ```sql
    SELECT LN(2);
    ```
    **输出:** `0.6931471805599453`

16. **`LOG10()`**: 返回数字的以 10 为底的对数。
    ```sql
    SELECT LOG(2);
    ```
    **输出:** `0.6931471805599453`

17. **`LOG2()`**: 返回数字的以 2 为底的对数。
    ```sql
    SELECT LOG2(6);
    ```
    **输出:** `2.584962500721156`

18. **`MOD()`**: 返回 n 除以 m 的余数。
    ```sql
    SELECT MOD(18, 4);
    ```
    **输出:** `2`

19. **`PI()`**: 返回 PI 的值，显示 6 位小数。
    ```sql
    SELECT PI();
    ```
    **输出:** `3.141593`

20. **`POW()`**: 返回 m 的 n 次方。
    ```sql
    SELECT POW(4, 2);
    ```
    **输出:** `16`

21. **`RADIANS()`**: 将度数值转换为弧度。
    ```sql
    SELECT RADIANS(180);
    ```
    **输出:** `3.141592653589793`

22. **`RAND()`**: 返回一个随机数。
    ```sql
    SELECT RAND();
    ```
    **输出:** `0.33623238684258`

23. **`ROUND()`**: 返回四舍五入到指定小数位数的数字。
    ```sql
    SELECT ROUND(5.553);
    ```
    **输出:** `6`

24. **`SIGN()`**: 返回指示数字符号的值。
    ```sql
    SELECT SIGN(255.5);
    ```
    **输出:** `1`

25. **`SIN()`**: 返回数字的正弦值。
    ```sql
    SELECT SIN(2);
    ```
    **输出:** `0.9092974268256817`

26. **`SQRT()`**: 返回数字的平方根。
    ```sql
    SELECT SQRT(25);
    ```
    **输出:** `5`

27. **`TAN()`**: 返回数字的正切值。
    ```sql
    SELECT TAN(1.75);
    ```
    **输出:** `-5.52037992250933`

28. **`ATAN2()`**: 返回 x 和 y 坐标的反正切值，以弧度表示的角度。
    ```sql
    SELECT ATAN2(7);
    ```
    **输出:** `1.42889927219073`

29. **`TRUNCATE()`**: 此函数不适用于 SQL Server。它将数字截断到小数点右侧指定位数。
    ```sql
    SELECT TRUNCATE(7.53635, 2);
    ```
    **输出:** `7.53`
