# MariaDB 中的 DATE_FORMAT() 函数

> 原文: [https://www.geeksforgeeks.org/date_format-function-in-mariadb/](https://www.geeksforgeeks.org/date_format-function-in-mariadb/)

## DATE_FORMAT() 函数

在 MariaDB 中，`DATE_FORMAT()` 函数接受两个参数：一个日期和一个格式掩码。该函数将根据指定的格式掩码返回格式化后的日期字符串。

## 语法

```sql
DATE_FORMAT(date, format_mask)
```

## 参数

参数是必需的。

*   `date` – 要格式化的日期。
*   `format_mask` – 应用于日期的格式。

## 返回值

根据掩码格式返回转换后的日期字符串。

## 格式掩码

*   `%Y`: 年份，数字，4 位数
*   `%y`: 年份为 2 位数的数值
*   `%a`: 工作日名称缩写 (Sun 到 Sat)
*   `%b`: 月名缩写 (1 月至 12 月)
*   `%c`: 数值形式的月份 (0 到 12)
*   `%D`: 以数值形式表示的一个月中的某一天，后跟一个后缀 (1st, 2nd, 3rd, …)
*   `%d`: 一个月中的某一天的数值 (01 到 31)
*   `%e`: 一个月中的某一天的数值 (0 到 31)
*   `%f`: 微秒 (000000 到 999999)
*   `%H`: 小时 (00 到 23)
*   `%h`: 小时 (00 到 12)
*   `%I`: 小时 (00 到 12)
*   `%i`: 分钟 (00 到 59)
*   `%j`: 一年中的某一天 (001 到 366)
*   `%k`: 小时 (00 到 23)
*   `%l`: 小时 (1 到 12)
*   `%M`: 完整的月份名称 (1 月至 12 月)
*   `%m`: 数值形式的月份名称 (00 到 12)
*   `%p`: 上午 (AM) 或下午 (PM)
*   `%r`: 12 小时上午或下午格式的时间 (hh:mm:ss AM/PM)
*   `%S`: 秒 (00 到 59)
*   `%s`: 秒 (00 到 59)
*   `%T`: 24 小时制时间 (hh:mm:ss)
*   `%U`: 周日是一周的第一天的一周 (00 到 53)
*   `%u`: 一周，其中星期一是一周的第一天 (00 到 53)
*   `%V`: 周日是一周的第一天的一周 (01 到 53)
*   `%v`: 周一是一周的第一天的一周 (01 到 53)
*   `%W`: 完整的工作日名称 (Sunday 到 Saturday)
*   `%w`: 一周中的某一天，其中星期日=0，星期六=6
*   `%X`: 周日是一周的第一天的那一周的年份
*   `%x`: 一周中的年份，其中星期一是一周的第一天

## 示例

**示例 1:**
```sql
SELECT DATE_FORMAT('2020-04-09', '%M %d, %Y');
```
**输出:**
```
'April 09, 2020'
```

**示例 2:**
```sql
SELECT DATE_FORMAT('2020-10-18', '%W');
```
**输出:**
```
'Sunday'
```

**示例 3:**
```sql
SELECT DATE_FORMAT('2020-10-20', '%M %e %Y');
```
**输出:**
```
'October 20 2020'
```

**示例 4:**
```sql
SELECT DATE_FORMAT('2020-10-19', '%W, %M %e, %Y');
```
**输出:**
```
'Monday, October 19, 2020'
```

**示例 5:**
```sql
SELECT DATE_FORMAT('2014-05-17 08:44:21.000001', '%h');
```
**输出:**
```
'08'
```

**示例 6:**
```sql
SELECT DATE_FORMAT('2019-08-11 10:44:21', '%s');
```
**输出:**
```
'21'
```

**示例 7:**
```sql
SELECT DATE_FORMAT('2012-06-15 11:23:16', '%t');
```
**输出:**
```
'11:23:16'
```

**示例 8:**
```sql
SELECT DATE_FORMAT('2019-03-13', '%M');
```
**输出:**
```
'March'
```

**示例 9:**
```sql
SELECT DATE_FORMAT('2020-10-23', '%W');
```
**输出:**
```
'Friday'
```

**示例 10:**
```sql
SELECT DATE_FORMAT('2019-05-13', '%Y');
```
**输出:**
```
'2019'
```
