# 实时系统中事件的分类

> 原文：[https://www.geeksforgeeks.org/classification-of-events-in-real-time-system/](https://www.geeksforgeeks.org/classification-of-events-in-real-time-system/)

**事件**在[实时系统](https://www.geeksforgeeks.org/real-time-systems/)中是系统或环境产生的动作或动作的结果。实时系统中的事件要么是瞬间的，要么可能有一定的持续时间。实时系统中事件的分类基于不同的理论。一旦实时系统中的事件被分类，则定时约束被相应地分类。

## 事件分类

### 1. 基于生成的基础

实时系统中的事件可以由环境或系统生成。在此基础上，事件分为两类：

*   **刺激事件**
    在实时系统中，刺激事件由环境产生。刺激事件作用于计算机系统。刺激事件是非周期性和异步的。这些事件不是为响应生成的。刺激事件可能是瞬间的，也可能持续一段时间。

    **示例：**
    1.  在键盘上打字是作用于计算机系统的刺激事件。
    2.  测量植物的温度。

*   **响应事件**
    在实时系统中，响应事件由计算机系统生成。这些事件作用于环境。这些事件是为响应刺激事件而生成的。响应事件可以是周期性的或非周期性的。这些事件是为响应而生成的。响应事件通常是瞬时事件。

    **示例：**
    1.  4点钟闹铃是一个响应事件。
    2.  温度超过100度时关闭感应。

### 2. 基于持续时间

实时系统中的事件可能是瞬时的，也可能有一定的持续时间。在此基础上，实时系统中的事件分为两类：

*   **瞬时事件**
    在实时系统中，瞬时事件是持续时间为零的事件。这些事件可能由环境或计算机系统生成，但它们是在瞬间生成的。瞬时事件可以是刺激事件或响应事件。

    **示例：**
    1.  瞬间按下一个键。
    2.  随时显示警报通知。

*   **持续事件**
    在实时系统中，持续事件是持续时间大于零的事件。这些事件可能由环境或计算机系统生成，但它们具有一定的持续时间。持续事件可以是刺激事件或响应事件。

    **示例：**
    1.  测量温度是一个持续的事件。
    2.  计算机系统中的所有计算事件都是持续性事件。
