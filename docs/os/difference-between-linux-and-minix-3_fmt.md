# Linux 和 MINIX 3 的区别

> 原文:[https://www . geeksforgeks . org/Linux 和-minix-3 的区别/](https://www.geeksforgeeks.org/difference-between-linux-and-minix-3/)

## 1. [`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/)
[`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 是由 Linus Torvalds 开发的一组开源类 Unix 操作系统。它是 [`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 发行版的打包版本。一些最常用的 [`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 发行版是 Debian、Fedora 和 Ubuntu。基本上是用 [`C语言`](https://www.geeksforgeeks.org/c-language-set-1-introduction/)和汇编语言写的。[`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 使用的内核是 [`Monolithic内核`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/)。[`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 发行版的目标系统是云计算、嵌入式系统、移动设备、个人电脑、服务器、[大型机和超级计算机](https://www.geeksforgeeks.org/difference-between-supercomputer-and-mainframe-computer/)。[`Linux`](https://www.geeksforgeeks.org/introduction-to-linux-operating-system/) 的第一个版本于 1991 年推出。

## 2. [`MINIX 3`]
[`MINIX 3`](https://www.geeksforgeeks.org/difference-between-linux-and-minix-3/) 是由 Andrew S. Tanenbaum 开发的免费开源操作系统。它是根据伯克利软件分发许可证发布的。它是一个类似 Unix 的操作系统。它是专门为工作站设计的。它源自 [`MINIX 1`](https://www.geeksforgeeks.org/difference-between-linux-and-minix-3/) 和 [`MINIX 2`](https://www.geeksforgeeks.org/difference-between-linux-and-minix-3/) 操作系统。它基本上是用 [`C语言`](https://www.geeksforgeeks.org/c-language-set-1-introduction/)和汇编语言编写的。[`MINIX 3`](https://www.geeksforgeeks.org/difference-between-linux-and-minix-3/) 的第一个版本于 2005 年推出。这个操作系统使用的内核是微内核。
