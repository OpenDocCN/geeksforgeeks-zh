## Kali Linux 和 Parrot OS 的区别

> 原文:[https://www . geesforgeks . org/区别-kali-Linux-和-parrot-os/](https://www.geeksforgeeks.org/difference-between-kali-linux-and-parrot-os/)

**Kali Linux:**
Kali Linux 是一个用于渗透测试和数字取证的操作系统，以 `Linux` 为核心。它是根据 `Debian`(一个 `Linux` 发行版)标准开发的。它于 2013 年 3 月首次发布，旨在取代回溯操作系统。

**Parrot OS:**
Parrot OS 类似于 Kali Linux，是一个开源的基于 `Debian` 的操作系统。它用于云测试、计算机取证、黑客攻击和隐私/匿名。它于 2013 年 4 月首次发布。

这两种操作系统有一些相似之处:

*   两者对[渗透测试](https://www.geeksforgeeks.org/software-testing-penetration-testing/)都有用。
*   两者都是基于 `Debian` 标准开发的。
*   两者都可以支持 32 位和 64 位架构。

让我们看看 Kali Linux 和 Parrot OS 的区别:

| 序号 | Kali Linux | Parrot OS |
| --- | --- | --- |
| 1. | 它需要更多的内存，大约 1 GB。 | 它需要更少的内存，大约 320 兆字节。 |
| 2. | 在 `GPU` 方面，它需要一个图形卡，因为它需要图形加速。 | 它不需要图形加速，因此不需要显卡。 |
| 3. | 安装需要大约 20 GB 的可用空间。 | 安装需要大约 16 GB 的可用空间。 |
| 4. | 它的界面遵循 `Gnome` 桌面界面。 | 它的界面是建立在 `Ubuntu-mate-Desktop-Environment` 之上的。 |
| 5. | 它没有预装的编译器和 `IDEs`。 | 它预装了一堆编译器和 `IDEs`。 |
| 6. | 它有一个更简单的用户界面。 | 它有更好的用户界面。 |
| 7. | 它有重量级的要求，有点落后。 | 它很轻，也不会落后太多。 |
| 8. | 它拥有黑客攻击所需的所有基本工具。 | 它拥有 `Kali` 中所有可用的工具，并且还添加了它自己的工具。例如 `AnonSurf`, `Wifiphisher`, `Airgeddon`。 |
