# Node.js 和 AngularJS 的区别

> 原文: [https://www.geeksforgeeks.org/difference-between-node-js-and-angularjs/](https://www.geeksforgeeks.org/difference-between-node-js-and-angularjs/)

`AngularJS` 是谷歌开发的开源 web 应用开发框架。它为开发动态和单页 web 应用程序提供支持。

`Node.js` 是一个跨平台的运行环境，用于在浏览器之外运行 `JavaScript` 应用程序。它提供了丰富的各种 `JavaScript` 模块库，可以简化编码。

| AngularJS | Node.js |
| --- | --- |
| 是用于开发动态 web 应用程序的结构化框架。 | 它是一个用 `JavaScript` 语言编写的应用程序的**跨平台运行环境**。 |
| 完全用 `JavaScript` 编写。 | 用 `C`、`C++` 和 `JavaScript` 编写。 |
| 用于构建**单页客户端**应用程序。 | 用于构建**快速、可扩展的服务器端和客户端网络应用程序**。 |
| 非常适合开发高度活跃和交互式的网络应用程序。 | 适合开发小型项目。 |
| 开发人员只需**添加 `angularjs` 文件**即可在自己的应用程序中使用它。 | 开发人员需要在自己的计算机系统上安装 `Node.js`。 |
| `AngularJS` 中的模型和视图比其他 `JavaScript` 客户端框架中的要简单得多。 | 它利用 `JavaScript` 的事件驱动特性来支持非阻塞操作，这使得该平台非常高效。 |
| 它基于模型-视图-控制器设计模式并完全接受该模式。 | 它是单线程的，这意味着 web 请求在同一线程上处理和运行。 |
| `AngularJS` 是一个 **web 框架**。 | `Node.js` 提供不同的 web 框架，如 `Socket.io`、`Hapi.js`、`Meteor.js`、`Express.js` 和 `Sails.js`。 |
