## & (x: Short) 方法

`& (x: Short)` 方法用于返回短值对指定 `Int` 值进行按位 AND 运算的结果。

**方法定义:** `(Int_Value).&(short_value)`

**返回类型:** 通过 `Short` 值返回指定 `Int` 值的按位 AND 运算结果。

### 示例 1

```scala
// Scala program of Int &(x: Short)
// method

// Creating object
object GfG
{

    // Main method
    def main(args: Array[String])
    {

        // Applying Int &(x: short) function
        val result = (10).&(6)

        // Displays output
        println(result)

    }
}
```
