# Scala Int byteValue()方法示例

> 原文:``https://www.geeksforgeeks.org/scala-int-bytevalue-method-with-example/``

``byteValue()`` 方法用于返回指定 ``Int`` 值的字节值。

> **方法定义:** ``def byteValue(): Byte``
>
> **返回类型:** 返回给定 ``Int`` 值的字节值。

## 示例#1:

```scala
// Scala program of Int byteValue
// method

// Creating object
object GfG
{

    // Main method
    def main(args: Array[String])
    {

        // Applying byteValue method
        val result = (15).byteValue

        // Displays output
        println(result)

    }
}
```
