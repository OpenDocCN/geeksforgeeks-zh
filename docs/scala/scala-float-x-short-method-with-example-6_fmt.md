# Scala Float `+(x: Short)` 方法示例

> 原文：[`https://www.geeksforgeks.org/scala-float-x-short-method-with-example-6/`](https://www.geeksforgeks.org/scala-float-x-short-method-with-example-6/)

使用 `+(x: Short)` 方法返回指定的浮点值和短值之和。

> **方法定义：** `(Float_Value).+(Short_Value)`
>
> **返回类型：** 返回指定的浮点值和短值之和。

## 示例 #1

```scala
// Scala program of Float +(x: Short)
// method

// Creating object
object GfG
{

// Main method
    def main(args:Array[String])
    {

// Applying +(x: Short) function
        val result = (6.0).+(10)

// Displays output
        println(result)

}
}
```

**输出：**

```scala
16.0
```
