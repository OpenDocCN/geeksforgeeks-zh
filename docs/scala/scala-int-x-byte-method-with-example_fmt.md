# Scala Int `+(x: Byte)` 方法，示例

> 原文: [https://www.geeksforgeeks.org/scala-int-x-byte-method-with-example/](https://www.geeksforgeeks.org/scala-int-x-byte-method-with-example/)

利用 `+(x: Byte)` 方法返回指定的 `Int` 值和 `Byte` 值之和。

> **方法定义:** `(Int_Value).+(Byte_Value)`
>
> **返回类型:** 返回指定的 `Int` 值和 `Byte` 值之和。

## 示例 #1:

```scala
// Scala program of Int +(x: Byte)
// method

// Creating object
object GfG
{

// Main method
    def main(args:Array[String])
    {

// Applying Int +(x: Byte) function
        val result = (100).+(60)

// Displays output
        println(result)

}
}
```
