# Scala Int 一元 _-方法示例

> 原文: [https://www.geeksforgeeks.org/scala-int-unary_-method-with-example-2/](https://www.geeksforgeeks.org/scala-int-unary_-method-with-example-2/)

`unary_-` 方法用于返回相同的带负号的指定 int 值。

> **方法定义:** `(Int_Value).unary_-`
> **返回类型:** 返回相同的带负号的指定 int 值。

## 示例#1:

```scala
// Scala program of Int unary_-
// method

// Creating object
object GfG
{

// Main method
    def main(args:Array[String])
    {

// Applying unary_- function
        val result = (65).unary_-

// Displays output
        println(result)

    }
}
```
