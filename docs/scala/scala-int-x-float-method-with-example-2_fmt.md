# Scala Int `*(x: Float)` 方法

> 原文: [https://www.geeksforgeeks.org/scala-int-x-float-method-with-example-2/](https://www.geeksforgeeks.org/scala-int-x-float-method-with-example-2/)

使用 `*(x: Float)` 方法返回指定的 `Int` 值和 `Float` 值的乘积。

> **方法定义:** `(Int_Value).*(float_value)`
>
> **返回类型:** 返回指定的 `Int` 值和 `Float` 值的乘积。

## 示例 #1

```scala
// Scala program of Int *(x: Float)
// method

// Creating object
object GfG
{

// Main method
    def main(args: Array[String])
    {

// Applying Int *(x: Float) function
        val result = (100).*(5.0)

// Displays output
        println(result)

    }
}
```
