# Scala Int `- (x: Short)` 方法示例

> 原文: [https://www.geeksforgeeks.org/scala-int-x-short-method-with-example-10/](https://www.geeksforgeeks.org/scala-int-x-short-method-with-example-10/)

利用 `- (x: Short)` 方法返回指定的 `Int` 值与 `Short` 值之差。

> **方法定义:** `(Int_Value).-(short_value)`
>
> **返回类型:** 返回指定的 `Int` 值和 `Short` 值的差值。

## 示例 #1

```scala
// Scala program of Int -(x: Short)
// method

// Creating object
object GfG
{

// Main method
    def main(args: Array[String])
    {

// Applying Int -(x: Short) function
        val result = (100).-(50)

// Displays output
        println(result)

    }
}
```
