# 为什么大家更喜欢有蟒蛇的硒？

> 原文:[https://www . geesforgeks . org/为什么人们更喜欢用蟒蛇来补硒/](https://www.geeksforgeeks.org/why-do-people-prefer-selenium-with-python/)

`Selenium` 是一套强大的工具，坚定地支持 web 应用测试自动化的快速发展。它提供了一组测试功能，这些功能是专门为 web 应用程序的测试需求而设计的。这些功能是可靠的，有助于定位用户界面元素的各种选项，并将预期结果与实际应用程序行为进行比较。它利用各种脚本语言来实现测试自动化。

`Python` 是一种高级的基于对象的脚本语言，以用户友好的方式设计。它使用简单的英语关键词，需要非常容易的解释。与其他编程语言相比，它几乎没有复杂的语法。

## 为什么和 Python 一起使用硒？

这里列出了一些原因:

*   易于编码和阅读。
*   与另一个程序相比，运行速度更快。
*   提供动态类型特性。
*   许多程序员习惯使用 `Python` 语言。
*   `Python` 中使用的 `API` 帮助你通过 `Selenium` 连接到浏览器。`Python` 和 `Selenium` 之间的绑定提供了一个简单的 `API`，可以通过 `Selenium WebDriver` 以可感知的方式编写功能测试。此外，该绑定还提供了一个非常方便的 `API` 来联系 [Selenium web driver](https://www.browserstack.com/guide/selenium-webdriver-tutorial)，例如 `Firefox`、`Remore` 等。
*   与另一种编程语言相比，`python` 语言不冗长且易于使用。正是 `Python API` 帮助你通过 `Selenium` 连接到浏览器。`Selenium` 可以轻松地向各种浏览器发送 `python` 标准命令，而不考虑浏览器设计的所有变化。
*   `Python` 作为一种脚本语言，运行一个编译器将代码从代码行转换为可以实现和利用的任何东西。
*   `Python` 运行非常快，使用缩进来开始和结束块。与其他编程语言相比，它简单而紧凑。
*   `Python` 语言是围绕一个庞大的社区开发的，它为这个社区提供了一个非常好的支持系统。因此，在使用 `Selenium` 和 `python` 进行自动化测试的过程中，对于那些完全不熟悉这门语言的人来说，这个社区就像一个受欢迎的马车。
*   编程也是免费的，并且可以是开源的。任何需要它的人都可以轻松下载并在任何环境中自由使用它。整个文化以及围绕该语言的社区都是免费软件的粉丝。
*   除了上述所有支持在 `python` 中使用 `Selenium` 的原因之外，在 `Python` 中使用 `selenium` 的另一个主要原因是存在各种工具来扩展它。简化用户界面的最重要工具是网络驱动程序，它与 `Python` 强烈绑定。

因此，这就是为什么现在 `Selenium` 主要使用 `Python` 作为脚本语言的几个重要原因，尽管大量其他语言也有相同的特性。
