# 刮网入门

> 原文：[https://www.geeksforgeeks.org/introduction-to-web-scraping/](https://www.geeksforgeeks.org/introduction-to-web-scraping/)

网页抓取是一种从网站获取数据的技术。在网上冲浪时，许多网站不允许用户保存数据供个人使用。一种方法是手动复制粘贴数据，这既繁琐又耗时。网页抓取是从网站提取数据过程的自动化。这项活动是在称为刮网器的刮网软件的帮助下完成的。他们根据用户需求自动从网站加载和提取数据。这些可以定制为一个网站，也可以配置为任何网站。

## 刮网的用途

刮网在专业和个人层面都有很多用途。不同层次有不同的需求，网页抓取的一些流行用法是：

*   `Brand monitoring and competition analysis`：网页抓取用于获取客户对特定服务或产品的反馈，以了解客户对特定事物的感受。它也用于以结构化和可用的格式提取竞争对手的数据。
*   `Machine learning`：机器学习是人工智能的一个过程，其中机器被允许通过其经验来学习和改进，而不是被显式编程。因此，有必要通过网页爬取软件从数百万个网站中提取大量数据。
*   `Financial data analysis`：网页爬取用于以可用的格式保存股票市场的记录，因此它被用于洞察。
*   `Social media analysis`：它用于从社交媒体网站提取数据，以衡量客户趋势及其对活动的反应。
*   `SEO 监控`：搜索引擎优化是一个网站在 Google、Yahoo、Bing 等不同搜索引擎中的可见性和排名的优化。网页抓取是用来了解内容的排名是如何随时间变化的。

还有很多其他的原因可以使用网络报废。

## 网页抓取技术

从网站中提取数据有两种方式，手动提取技术和自动提取技术。

*   `Manual extraction technology`：手动复制和粘贴网站内容属于这种技术。虽然繁琐、耗时且重复，但对于清除具有良好防刮措施（如机器人检测）的网站数据来说，它是一种有效的方法。
*   `Automatic extraction technology`：网页捕获软件用于根据用户要求自动从网站提取数据。
    *   `HTML parsing`：解析意味着逐部分分析可理解的部分。换句话说，它意味着将一种形式的信息转换为另一种更易于处理的形式。HTML 解析指的是接收代码并根据用户要求从中提取相关信息。主要使用 JavaScript，顾名思义，目标是 HTML 页面。
    *   `DOM parsing`：文档对象模型是万维网联盟的官方推荐。它定义了一个接口，使用户能够修改和更新 XML 文档的样式、结构和内容。
    *   `Web page crawling software`：如今，有许多网页爬取工具可用，或者根据用户需求定制，以从数百万个网站中提取所需信息。

## 刮网工具

刮网工具是专门为从互联网上提取数据而开发的。也被称为网络收集工具或数据提取工具，它们对任何试图从网站收集特定数据的人都很有用，因为它们为用户提供了从多个网站提取数据的结构化数据。一些最流行的网页抓取工具有：

*   `Import.io`
*   `Webhose.io`
*   `Dexi.io`
*   `剪贴簿`
*   `Parsehub`

## 刮网合法化

刮网合法化是一个敏感话题，取决于它的使用方式，它可能是好事，也可能是坏事。一方面，用好的 bot 进行网页抓取，使搜索引擎能够索引网页内容，提供比价服务，为客户节省资金和价值。但是网页抓取可以被重新定位，以满足更多的恶意和滥用的目的。网页抓取可以与其他形式的恶意自动化相结合，被命名为`“坏机器人”`，这使得其他有害活动成为可能，如`拒绝服务攻击`、`竞争数据挖掘`、`账户劫持`、`数据窃取`等。

网页抓取的合法性是一个灰色区域，随着时间的推移，它会不断发展。尽管网络抓取器在技术上提高了数据浏览、加载、复制和粘贴的速度，但网络抓取也是侵犯版权、违反使用条款和其他对公司业务造成高度破坏的活动案件增加的罪魁祸首。

## 对刮网的挑战

除了对刮网合法性的挑战，还有其他问题对刮网提出了挑战。

*   `Data warehouse`：大规模数据提取将产生大量信息用于存储。如果数据仓库基础设施构建不当，搜索、存储和导出这些数据将成为一项麻烦的任务。因此，对于大规模数据提取，需要一个完美无缺的数据仓库系统。
*   `Changes in website structure`：每个网站都会定期更新用户界面以增强其吸引力和体验。这也需要各种结构上的改变。因为网络爬虫是根据当时网站的代码元素设置的，所以它也需要修改。因此，它们也需要每周更改以捕获正确的网站数据，因为关于网站结构的不完整信息将导致不正确的数据捕获。
*   `Scratch-proof technology`：一些网站使用防刮技术来防止任何刮取尝试。它们应用动态编码算法来防止任何机器人干扰，并使用 IP 封锁机制。要绕过这种防刮技术需要大量的时间和金钱。
*   `Data quality extracted`：不符合信息质量要求的记录会影响数据的整体完整性。确保收集的数据符合质量标准是一项艰巨的任务，因为它需要实时完成。

## 数据刮擦的未来

由于数据刮擦存在一些挑战和机遇，可以公平地认为，非预期的数据刮擦从业者倾向于制造道德风险，他们将公司作为目标并检索其数据。然而，由于我们正处于数据转型的边缘，数据抓取与大数据相结合可以为公司提供市场情报，帮助他们识别关键趋势和模式，并确定最佳机会和解决方案。因此，说数据清理可以很快升级到更好的状态并没有错。
