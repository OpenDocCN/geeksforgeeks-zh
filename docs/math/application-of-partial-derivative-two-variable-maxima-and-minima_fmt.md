# 偏导数的应用–两个变量的最大值和最小值

> 原文: [https://www.geeksforgeeks.org/application-of-partial-derivative-two-variable-maxima-and-minima/](https://www.geeksforgeeks.org/application-of-partial-derivative-two-variable-maxima-and-minima/)

偏导数可以用来求双变量函数的最大值和最小值(如果存在的话)。我们试图找到一个斜率为零的静止点，然后追踪它附近的最大值和最小值。最大值/最小值的实际应用是给定曲线的利润最大化或损失最小化。

设 `f(x, y)` 为实值函数，设 `(pt, pt')` 为 `f(x, y)` 域内的内点，则

*   `Pt, pt'` 被称为局部最大值点。如果存在 `h > 0`，那么 `f(pt, pt') ≥ f(x, y)`。对于所有在 `(pt–h, pt'+h)` 中的 `x` 值，`x ≠ a`，`f(pt, pt')` 被称为 `f(x, y)` 的局部最大值。
*   `Pt, pt'` 如果 `h < 0` 使得 `f(pt, pt') ≥ f(x, y)`，对于所有在 `(pt–h, pt'+h)` 中的 `x`，`x ≠ a`，`f(pt, pt')` 的值被称为 `f(x, y)` 的局部最小值。

**求双变量函数最大值和最小值的算法:**

1.  令 `F_XX = 0` 和 `F_YY = 0`。**注意**: `F_XX` 和 `F_YY` 分别是...
2.  获得的结果将被视为曲线的**稳定/转折点**。
3.  创建三个新变量 `R`、`T` 和 `S`。
4.  使用 `r = f_xx`、`t = f_yy` 找到 `r`、`t`、`s` 的值，`s = f_xy`。
5.  **如果**
    `rt - s^2 |_(固定 pts) > 0`
    则**(最大值/最小值)存在**。
6.  如果 `rt - s^2 |_(固定 pts) < 0`，则**不存在**。

## 示例-1

函数 `f(x, y) = x^2y - 3xy + 2y + x` 有

*   (a) 无局部极值
*   (b) 有局部最小值但无局部最大值
*   (c) 有局部最大值但无局部最小值
*   (d) 有局部最小值和局部最大值

**说明:**

回答: A

```
r = ∂²f/∂x² = 2y
s = ∂²f/∂x∂y = 2x - 3
t = ∂²f/∂y² = 0
```

因为 `rt - s² ≤ 0`，(如果 `rt - s² < 0`，那么我们没有最大值或最小值，如果 `= 0`，那么我们什么也不能说)。

当 `rt - s² > 0` 和 `r < 0` 时，最大值将存在。

当 `rt - s² > 0` 和 `r > 0` 时，将存在最小值。

**由于 `rt - s²` 永远不大于 0，所以我们没有局部极值。**

## 示例-2

求函数 `f(x, y) = 2x² + 2xy + 2y² - 6x` 的局部最小值

```
fx(x, y) = 4x + 2y - 6 = 0    (1)
fy(x, y) = 2x + 4y = 0        (2)
```

在求解 `(1)` 和 `(2)` 时，我们得到，

```
x = 2, y = -1
r = ∂²f/∂x² = 4
s = ∂²f/∂x∂y = 2
t = ∂²f/∂y² = 4
rt - s² = 12
```

**由于 `rt - s² > 0` 和 `r > 0`。因此，`(2, -1)` 是局部极小点。**

## 示例-3

求 `f(x, y) = x² + y² + 6x + 12` 的最大值/最小值

```
fx(x, y) = 2x + 6 = 0     (1)
fy(x, y) = 2y = 0         (2)
```

在求解 `(1)` 和 `(2)` 时，我们得到，

```
x = -3, y = 0
r = ∂²f/∂x² = 2
s = ∂²f/∂x∂y = 0
t = ∂²f/∂y² = 2
```

**由于 `rt - s² > 0` 和 `r > 0`。因此，`(-3, 0)` 是局部极小点。**
