# 群同态和正规子群

> 原文: [https://www.geeksforgeeks.org/group-homomorphisms-and-normal-subgroup/](https://www.geeksforgeeks.org/group-homomorphisms-and-normal-subgroup/)

**先决条件:** [群 (Group)](https://www.geeksforgeeks.org/groups-discrete-mathematics/)

## 同态

一个映射（函数）`f` 从一个群 `(G, *)` 到一个群 `(G', +)` 被称为从 `G` 到 `G'` 的同态，如果对于所有 `a, b ∈ G`，满足 `f(a * b) = f(a) + f(b)`。

### 示例

1.  函数 `f(x) = a^x` 从 `(R, +)` 到 `(R_{o}, *)`，这里 `R` 是所有实数的集合，`R_{o}` 是除 0 以外的所有实数的集合。对于任意 `n, m ∈ R`，`f(n+m) = a^{n+m} = a^{n} * a^{m} = f(n) * f(m)`，为同态。

2.  对于循环群 `Z_{3} = {0, 1, 2}` 和带加法的 `Z`（整数集），函数 `f(x) = x mod 3` 从 `Z` 到 `(Z, +)` 是群同态。

### 注

对于同态 `f: G → G'`：

*   如果 `f` 是单射的（一对一），那么 `f` 就是**单态**。
*   如果 `f` 是满射（到上），那么 `f` 就是**满态**。
*   如果 `G = G'`，那么 `f` 是**自同态**。
*   `f(G)` 被称为 `G` 群的同态像。

## 同态相关定理

### 定理 1

如果 `f` 是从一个群 `(G, *)` 到 `(G', +)` 的同态，并且如果 `e` 和 `e'` 是它们各自的恒等元，那么 `f(e) = e'`。对于 `n ∈ G`，`f(n^{-1}) = f(n)^{-1}`。

**证明：**

1.  设 `n ∈ G`，那么 `n * e = n = e * n`。
    => `f(n * e) = f(n) = f(e * n)`
    => `f(n) + f(e) = f(n) = f(e) + f(n)`，因为 `f` 是同态。
    => `f(e)` 是 `G'` 中的恒等元。
    => `f(e) = e'`。因此证明了群态射 `f` 下 `G` 的恒等元的像就是 `G'` 的恒等元。

2.  设 `n'` 是 `n ∈ G` 的逆元，那么 `n * n' = e = n' * n`。
    => `f(n * n') = f(e) = f(n' * n)`
    => `f(n) + f(n') = e' = f(n') + f(n)`，因为 `f` 是同态。
    => `f(n') = f(n)^{-1}`。
    证明了群态射 `f` 下 `G` 的任意元素的逆元的像是 `G'` 中该元素像的逆元。

### 定理 2

如果 `f` 是群 `(G, *)` 到群 `(G', +)` 的同态，那么：

*   `H` 是 `G` 的子群 => `f(H)` 是 `G'` 的子群。
*   `H'` 是 `G'` 的子群 => `f^{-1}(H') = { x ∈ G | f(x) ∈ H' }` 是 `G` 的子群。

**证明：**

1.  因为 `f(H)` 是 `G'` 的子群且 `f(H) ≠ ∅`，因为 `e ∈ H` => `f(e) = e' ∈ f(H)`，其中 `e'` 是 `G'` 中的恒等元。
    如果 `a', b' ∈ f(H)`，那么 `a' + b' ∈ f(H)`
    => 存在 `a, b ∈ H` 使得 `f(a) = a'` 和 `f(b) = b'`
    => `a' + b' = f(a) + f(b) = f(a * b)`，因为 `f` 是同态。
    但 `a ∈ H, b ∈ H` => `a * b ∈ H`
    => `f(a * b) ∈ f(H)`
    => `f(a * b) = a' + b' ∈ f(H)`。因此，`f(H)` 是 `G'` 的一个子群。

2.  因为 `f^{-1}(H')` 是 `G` 的子群且 `f^{-1}(H') ≠ ∅`，因为至少 `e ∈ f^{-1}(H')`。
    如果 `a, b ∈ f^{-1}(H')`，那么 `f(a) ∈ H'` 和 `f(b) ∈ H'`
    => `f(a) + f(b) ∈ H'`，因此 `f(a * b) ∈ H'`
    => `a * b ∈ f^{-1}(H')`。所以 `f^{-1}(H')` 是 `G` 的子群。

## 正规子群

群 `(G, *)` 的一个子群 `(N, *)` 称为 `(G, *)` 的**正规子群**，如果对于 `g ∈ G` 和 `n ∈ N`，我们有 `g * n * g^{-1} ∈ N`。
我们把它写成 `N ⊴ G`。

### 示例

1.  群 `N = ({1}, *)` 是群 `G = ({1, -1}, *)` 的正规子群，因为对于 `g = 1` 和 `n = 1`，`g * n * g^{-1} = 1 * 1 * (1^{-1}) = 1 ∈ N`。同样，对于 `g = -1` 和 `n = 1`，这里 `g * n * g^{-1} = 1 ∈ N`。
2.  群 `N = ({1, -1}, *)` 是群 `G = ({1, -1, i, -i}, *)` 的正规子群。
3.  包含任意 `G` 群的恒等元 `{e}` 的群，是 `G` 的正规子群。

### 注

*   如果 `N` 是 `G` 的子群，我们可以说 `g * N = N * g`，`∀g ∈ G`，其中 `g * N` 是 `N` 的左陪集，`N * g` 是 `N` 的右陪集。
*   如果一个非阿贝尔群的所有子群都是正规的，则称之为**哈密顿群**。
*   对于任何 `G` 群，1. `G` 本身和 2. `{e}`（其中 `e` 是恒等元）称为**平凡正规子群**，除此之外的两个称为**非平凡正规子群**。
*   没有非平凡正规子群的群称为**简单群**。
*   两个正规子群的交集也是正规子群。
*   循环群的每个子群都是正规子群。

## 与正规子群相关的定理

### 定理 1

阿贝尔群的所有子群都是正规的。

**证明：**

设 `N` 为任意阿贝尔群 `G` 的任意子群。
如果 `g ∈ G`（所以我们也有 `g^{-1}` 作为 `G` 的逆元）和 `n ∈ N`，那么，`g * n * g^{-1} = (n * g) * g^{-1}`，因为 `G` 是阿贝尔的所以可交换。
= `n * (g * g^{-1}) = n * e = n ∈ N`。所以 `N` 是 `G` 的正规子群。

### 定理 2

群 `(G, *)` 的子群 `(N, *)` 是正规的当且仅当 `g * N * g^{-1} = N` 对于所有 `g ∈ G`。

**证明：**

让 `N` 成为 `G` 的正规子群，那么，对于每个 `g ∈ G`，`n ∈ N` => `g * n * g^{-1} ∈ N`
=> `g * N * g^{-1} ⊆ N` ... (1)
=> `g^{-1} * N * (g^{-1})^{-1} ⊆ N`，因为 `g^{-1} ∈ G`。
=> `g^{-1} * N * g ⊆ N`
=> `g * (g^{-1} * N * g) * g^{-1} ⊆ g * N * g^{-1}`
=> `N ⊆ g * N * g^{-1}` ... (2)

根据等式 (1) 和 (2)，
`g * N * g^{-1} = N`，`∀ g ∈ G`。

**反过来：**

让 `g * N * g^{-1} = N`，`∀ g ∈ G`
那么 `g * n * g^{-1} ∈ N` => `g * n * g^{-1} ⊆ N`
=> `g * n * g^{-1} ∈ N`，对于 `n ∈ N`。
因此，证明了当且仅当 `g * N * g^{-1} = N` 时，`N` 是 `G` 的正规子群。
