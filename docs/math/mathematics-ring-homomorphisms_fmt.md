# 数学|环同态

> 原文: [https://www.geeksforgeeks.org/mathematics-ring-homomorphisms/](https://www.geeksforgeeks.org/mathematics-ring-homomorphisms/)

## 先决条件
`Ring`

## 环同态
对集合`R`进行任意两次二元运算的集合`R`由`+`和`*`表示的 let 称为表示为`(R, +, *)`的环，如果`(R, +)`是阿贝尔群，`(R, *)`是半群，它们也遵循左右分配律。

对于两个环`(R,+,*)`和`(S,⨁, \times)`来说，映射`f : R → S`被称为环同态，如果

1.  `f (a + b) = f (a) ⨁ f (b)`，`∀a, b ∈ R`。
2.  `f(a * b) = f(a) \times f(b)`，`∀a, b ∈ R`。
3.  `f(I_R) = I_S`，如果 `I_R` 和 `I_S` 分别是集`R`于`*`和集`S`于`\times`操作的身份(如果存在的话)。

注：环`(S,⨁, \times)`称为环`(R,+,*)`的同形像。

## 示例

1.  函数 `f(x) = x mod(n)` 从组(`Z`，+，*)到(`Z_n`，+，*) `∀x ∈ Z`，`Z`是一组整数。`+`和`*`分别是简单的加法和乘法运算。
2.  对于任意两个群 `(r，+，*)` 和 `(S,⨁，\times)`，`∀x ∈ R`，函数 `f(x) = x`，称为同态环同态。
3.  对于 `∀x ∈ N`，函数 `f(x) = 0`。
4.  函数 `f(x) =` 它本身是复共轭形式群 `(C，+，*)`，这里 `C` 是复数的集合。`+`和`*`分别是简单的加法和乘法运算。

注：如果 `f` 是来自 `(r，+，*)` 和 `(S,⨁，\times)` 的同态，那么 `f(O_R) = f(O_S)`，其中 `O_R` 和 `O_S` 分别是集合 `R` over `+` 和集合 `S` over `⨁` 运算的恒等式。

注：如果 `f` 是来自 `(r，+，*)` 和 `(S,⨁，\times)` 的环同态，那么 `f : (R，+) → (S,⨁)` 就是群同态。

## 环同构
从环`R`到环`S`的一一并到同态称为环同构，`R`和`S`是同构的。

## 环自同构
从环到自身的同态称为环自同构。

## 场同态
对于两个场 `(F,+,*)` 和 `(K,⨁, \times)`，一个映射 `f : F → K` 被称为场同态，如果

1.  `f(a + b) = f(a) ⨁ f(b)`，`∀a, b ∈ F`。
2.  `f(a * b) = f(a) \times f(b)`，`∀a, b ∈ F`。
3.  `f(I_F) = I_K`，其中 `I_F` 和 `I_K` 分别是设置`F`于`*`和设置`K`于`\times`操作的身份。
4.  `f(O_F) = O_K`，其中 `O_F` 和 `O_K` 分别是设置`F`于`+`和设置`K`于`⨁`操作的标识。
