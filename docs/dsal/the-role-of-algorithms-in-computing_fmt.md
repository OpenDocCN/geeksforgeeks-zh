# 算法在计算中的作用

> 原文：[https://www.geeksforgeeks.org/the-role-of-algorithms-in-computing/](https://www.geeksforgeeks.org/the-role-of-algorithms-in-computing/)

**先决条件：** [算法简介](https://www.geeksforgeeks.org/introduction-to-algorithms/)

*   单词 [algorithm](https://www.geeksforgeeks.org/fundamentals-of-algorithms/) 的意思是“在计算或其他操作中为解决问题而遵循的一套过程或规则”。因此，算法指的是一套规则/指令，它们逐步定义了如何执行工作以获得预期结果。
*   算法是当代计算机中使用的大多数技术的核心，并与其他技术协同工作。

## 如何决定哪种算法最适合？

*   这取决于当输入规模增大时算法的效率。
*   对数值可能存在的限制/约束。
*   计算机的架构和要使用的存储设备类型。
*   另一个重要方面是，算法的正确性意味着如果算法对每个实例都产生正确的输出，那么它就是正确的。不正确的算法可能在某些输入实例上根本不终止，或者给出错误的输出。

## 算法的实际应用：

*   **互联网**：很难想象没有它一天的结果会是什么，这要归功于聪明高效的算法。借助这些算法，互联网上的各种网站可以管理和操作大量数据。找到良好的数据传输路线，并使用搜索引擎查找包含特定信息的页面。
*   另一个伟大的里程碑是 [人类基因组计划](https://en.wikipedia.org/wiki/Human_Genome_Project)，它在识别人类DNA中的10万个基因、确定构成人类DNA的30亿个化学碱基对的序列、将这些海量信息存储在数据库中以及开发数据分析工具方面取得了巨大进展。这些步骤中的每一步都需要复杂而高效的算法。
*   日常的**电子商务**活动严重依赖我们的个人信息，如信用卡/借记卡号码、密码、银行对账单、一次性密码等。使用的核心技术包括基于数字算法和[数论](https://en.wikipedia.org/wiki/Number_theory)的公钥、[加密货币](https://www.geeksforgeeks.org/what-is-a-cryptocurrency/)和数字签名。
*   **[线性规划方法](https://www.geeksforgeeks.org/linear-programming/)** 也是这样一种技术，它：
    *   被广泛用于**制造业**和其他商业企业，这些企业几乎需要以最有利的方式分配资源。
    *   或者一个组织可能希望决定在哪里花钱购买广告，以**最大化**其组织成长的机会。
*   **[最短路径算法](https://www.geeksforgeeks.org/dijkstras-shortest-path-algorithm-greedy-algo-7/)** 也被广泛使用，例如：
    *   在**运输**公司中，如卡车运输或铁路公司，通过道路或铁路网络找到最短路线可能会带来经济效益，因为采用最短路线将降低人工或燃料成本。
    *   或者互联网上的一个**路由节点**可能需要在网络中找到最短路径，以便快速路由消息。
*   即使在应用层不需要算法内容的应用程序也严重依赖算法，因为这些应用程序依赖于**硬件、GUI、网络或面向对象**，而这些都广泛使用算法。T65】
