# 亚马逊面试体验|第 332 集(校外)

> 原文：[https://www.geeksforgeeks.org/amazon-interview-experience-set-332-off-campus/](https://www.geeksforgeeks.org/amazon-interview-experience-set-332-off-campus/)

## 亚马逊面试体验(校外)

### 第一轮：线上测试

1.  [`circular tour`](https://practice.geeksforgeeks.org/problems/circular-tour/1)。
2.  [`flip bits`](https://practice.geeksforgeeks.org/problems/flip-bits/0)。

### 第二轮：电话面试 1

1.  [`mirror tree`](https://practice.geeksforgeeks.org/problems/mirror-tree/1)（需要递归求解）。
2.  [`mirror tree`](https://practice.geeksforgeeks.org/problems/mirror-tree/1)（需要迭代求解）。
3.  [`mirror of n-ary tree`](https://www.geeksforgeeks.org/mirror-of-n-ary-tree/)。
4.  [`print all combinations of given length`](https://www.geeksforgeeks.org/print-all-combinations-of-given-length/)。

### 第三轮：电话面试 2

1.  [`square root`](https://practice.geeksforgeeks.org/problems/square-root/1)（预期的复杂性–logn，并且只能创建一个函数来处理所有情况）。
2.  [`merge two BST's`](https://practice.geeksforgeeks.org/problems/merge-two-bst-s/1)。

### 第 4 轮：F2F

1.  [`check if tree is isomorphic`](https://practice.geeksforgeeks.org/problems/check-if-tree-is-isomorphic/1)。
2.  [`total decoding messages`](https://practice.geeksforgeeks.org/problems/total-decoding-messages/0)。
3.  给定由购买的产品和客户 id 组成的日志文件，随时打印前 k 个购买项目。如果有数十亿个项目，而一台机器不可能处理它们，那么你将如何处理这种情况。讨论了几种方法。

### 第 5 轮：F2F

1.  [`top view of binary tree`](https://practice.geeksforgeeks.org/problems/top-view-of-binary-tree/1)。
2.  给定一个表达式树，不同的方式对表达式求值。例如表达式 `1+2 * 3`，可以评估为 `(1+2)*3` 或 `1+(2*3)`。后来他让我也找出所有可能的答案。

### 第 6 轮：F2F（技术+ hr）

*   [`kth largest element in BST`](https://practice.geeksforgeeks.org/problems/kth-largest-element-in-bst/1)。
*   [`Array vs Linked list`](https://www.geeksforgeeks.org/linked-list-vs-array/)。

几个行为问题。

### 第 7 轮：F2F

1.  酒店预订问题。
2.  [`next permutation`](https://practice.geeksforgeeks.org/problems/next-permutation/0)（需要 O(n) 解法）。
3.  关于 TRIE 的讨论。
