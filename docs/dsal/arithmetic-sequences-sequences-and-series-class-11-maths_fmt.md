序列是一个对象的枚举集合，其中重复是允许的，有序的事物形成了这样一种模式，通过这种模式我们可以识别整个序列。我们可以把这整个系列概括为序列。

**例 1. 差为 4 的偶数序列。**

**解决方案:**

> 2, 6, 10, 14,……………..`a_n`。
>
> 在上面的例子中，序列的第一项是 `a_1` = 2。而上学期是 `a_n`。

**例 2. 数字的排列，如 1，1，2，3，5，8，…没有可见的模式，但序列是由给出的递归关系生成的。**

**解决方案:**

> `a_1` = 1，`a_2` = 1，`a_3` = 2
>
> `a_3` = `a_1` + `a_2`。
>
> `a_n` = `a_{n-2}` + `a_{n-1}` 其中 `n > 2`
>
> 这叫做斐波那契数列。

**例 3. 一个数字的排列，比如 2，8，14，20，…有一个可见的模式，它的顺序是由这个关系产生的。**

**解决方案:**

> `a_1` = 2，`a_2` = 6，`a_3` = 14。元素间的公差为常数，等于 `d = a_2 - a_1` 或取任意两个相邻的数字。
>
> 所以序列变成 `a_n = a_1 + (n-1)*d`。

**系列:** A 系列可以高度概括为一个序列中所有项的和。让一个序列作为 `a_1`，`a_2`，`a_3`，……，`a_n`。则该表达式称为与给定序列相关联的序列。有限级数或无限级数的依赖性取决于序列的性质，是有限的还是无限的。数列用 `∑` (σ) 符号表示。因此，系列 `a_1 + a_2 + a_3 + ... + a_n = ∑_{k=1}^{n} a_k`。

## 算术序列

在算术序列中，一项和下一项之间的绝对差是常数。

**解释:**

如果 `a_{n+1} – a_n = d`，其中 `d` 为常数，则序列 `a_1, a_2, …, a_n` 被称为算术序列或算术级数。而且是共性差异。

让我们用第一项 `A` 和公共差 `d` 做一个算术级数

`{A, A+D, A+2D, A+3D, ….}`

A.P. 的第 `n` 个通用术语由 `a_n = a + (n-1)*d` 给出

其中，`A` 是 A.P. 的第一项，`d` 是公差，`n` 是项数。

## 显式公式

### 编写显式公式

让我们来看一个序列 6，16，26，36 …76。

序列的第一项是 6，公差是 10。

我们可以通过取第一项 6 得到序列中的任意项，公差为 10。

**第 n 项的计算**。

| 项数 | 表达式 | 计算 |
| :--- | :--- | :--- |
| 1. | six | `6 + 0*10 = 6` |
| 2. | `6+10` | `6 + 1*10 = 16` |
| 3. | `6+10+10` | `6 + 2*10 = 20` |
| 4. | `6+10+10+10` | `6 + 3*10 = 36` |
| 5. | `6+10+10+10+10` | `6 + 4*10 = 46` |
| 6. | `6+10+10+10+10+10` | `6 + 5*10 = 56` |
| 7. | `6+10+10+10+10+10+10` | `6 + 6*10 = 66` |

> 第 `n` 项很容易找到。第一项是 6，每一步我们得到的差是 10。
>
> 以上表述可以概括为 `6 + (n-1)*d`。
>
> 一般来说，这是一个算术数列的标准显式公式，它的第一项是，`A`，end，公共差是 `d`。
>
> **`A_n = A + (n-1)* d`**

### 等差数列的重要性质

*   如果一个常数被加到每个项上，那么得到的序列也是一个项
*   如果一个常数被减去每一项，那么得到的序列也是一个
*   如果 A.P 的每一项都乘以一个常数。那么得到的序列也是一个 A.P
*   如果一个 A.P 的每个项都除以非零常数，那么得到的序列也是一个 A.P

## 等差数列之和

咱们的顺序是给定为 `a, a+d, a+2d, a+3d, …..a+(n-1)*d`。

> **`S_n = (n/2)(a + l)`**
>
> *在哪里，*
>
> *`a` 是第一个术语*
>
> *`l` 为级数的最后一项*
>
> *`n` 是系列中的项数*
>
> 用等式 3 中的第 `n` 个项代替最后一个项 `l`，得到
>
> *第 `n` 项 = `a + (n–1)d`*
>
> *`S_n = (n/2)(a + a + (n–1)d)`*
>
> **`S_n = (n/2)(2a + (n–1) * d)`**

### 例 1. 按照下列顺序写出前三个术语

**(i) `A_n = 5n + 2(n-1)`**

**解决方案:**

> 求解: (I) 放 `n=1`，我们得到 `a_1 = 5*1 + 2(1-1) = 5 + 0 = 5`
>
> 放 `n=2`，我们得到 `a_2 = 5*2 + 2(2-1) = 10 + 2 = 12`
>
> 放 `n=3`，我们得到 `a_3 = 5*3 + 2(3-1) = 15 + 4 = 19`
>
> 所以前三项是 5，12，19。

**(ii) `A_n = 2n + 4(n-2)`**

**解决方案:**

> 放 `n=1`，我们得到 `a_1 = 2*1 + 4(1-2) = 2 - 4 = -2`
>
> 放 `n=2`，我们得到 `a_2 = 2*2 + 4(2-2) = 4 + 0 = 4`
>
> 放 `n=3`，我们得到 `a_3 = 2*3 + 4(3-2) = 6 + 4 = 10`
>
> 所以前三项是 -2，4，10。

### 例 2. 找出给定表达式的第 20 个项。

**解决方案:**

> `A_n = (n-1)(2-n)(3+n)`
>
> 求解: 在给定的表达式中放入 `n=20`，
>
> `a_{20} = (20-1)(2-20)(20+3) = 19 * -18 * 23 = -7886`。
>
> **注: 如果问的问题是找到算术序列的第 `n` 项，则认为该序列是 `a_n = a + (n-1)*d`。**

### 例 3. 求所有介于 100 和 1000 之间的自然数的和，这些自然数是 5 的倍数。

**解决方案:**

> 求解: 第一项为 100，最后一项为 1000，公差为 5。
>
> 所以我们的公式是 `S_n = (n/2)[2a + (n-1)*d]`。
>
> 我们需要找到项数，所以项数由 `(1000-100)/5 = 900/5 = 180` 给出。
>
> `s_{180} = (180/2)[2*100 + (180-1)*5]`。
>
> `s_{180} = 90 * [200 + 179*5]`
>
> `S_{180} = 90 * 1095 = 98,550`。
