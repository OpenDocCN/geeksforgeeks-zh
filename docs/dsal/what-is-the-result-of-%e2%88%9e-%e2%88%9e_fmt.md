# 什么是∞–∞？

> 原文: [https://www.geeksforgeeks.org/what-is-the-result-of-%e2%88%9e-%e2%88%9e/](https://www.geeksforgeeks.org/what-is-the-result-of-%e2%88%9e-%e2%88%9e/)

众所周知，从自身减去一个数会得到值 `0`，但是有一个混淆，就是从`无穷大`减去`无穷大`是否为`零`。但事实并非如此。因为`无穷大`不是一个[实数](https://www.geeksforgeeks.org/real-numbers/)。

## 基本假设与矛盾

**假设:**

*   首先，假设无穷减去无穷等于零，即 `∞-∞= 0`。
*   然后在等式两边同时加上数字 1，得到 `∞-∞+1 = 0+1`。
*   因为 `∞+1 = ∞` 且 `0+1 = 1`，所以等式两边简化为 `∞-∞ = 1`。

> 无穷减去无穷等于一和零是**不可能的**。使用这种类型的数学，将更容易得到无穷大减去无穷大等于任何实数。因此，从无穷大中减去无穷大就是`未定义`的。

## 黎曼悖论与π

现在用我们著名的数学家（黎曼悖论）的概念，从`∞`中减去`∞`，得到一个精确的`π`。

```
1-1/2+1/3-1/4+1/5-1/6+1/7-1/8+...+∞
```

从这个级数中分离正项和负项：
*   `1+1/3+1/5+1/7+……`
*   `-1/2–1/4–1/6–1/8–……`

现在，如果你只加正项，你会得到 `∞`，而如果加负项，你会得到 `-∞`。

**黎曼**的重排定理指出，如果存在一个收敛级数，其正项之和为 `∞` 且其负项之和为 `-∞`，那么它可以将该级数重排成一个具有任意和的级数。因此，对这个特定级数的 `π (π)` 执行了相同的操作。

`π (π)` 的值是正的（3.14159...）。因此，我们新级数的第一项将是 1，然后持续添加正项直到它接近 `π`。例如，我们使用 `1/151` 来加，使其达到 `3.1471`。

现在用户将使用负项来使其刚好下降。
所以使用 `-1/2`。现在 `π` 变成了 `2.6471`，这并不精确等于 `π`。

像这样添加一些正项，加加减减，你最终一定会得到精确的 `π`。

这是因为在这个过程的任何阶段，剩余正项的和是 `∞`，而剩余负项的和是 `-∞`。因此，无论当前和是低还是高，我们总是可以确信，能够通过添加足够的项来达到目标值。

所以，`π = ∞–∞`。这就是为什么数学家们决定将其保留为未定义，因为它不存在，并且很可能没有任何与之相关的有价值的意义。
