# 美术馆问题

> 原文: [https://www.geeksforgeeks.org/art-gallery-problem/](https://www.geeksforgeeks.org/art-gallery-problem/)

## 问题描述

**美术馆问题**在几何学中被表述为需要放置在一个 `n` 顶点简单多边形中的最小数量的守卫，这样内部的所有点都是可见的。简单多边形是一个连通的封闭区域，其边界由有限数量的线段定义。它是计算几何中一系列相关的问题。如果连接两个点的线段位于多边形内，则可见性定义为两个点 `u` 和 `v` 相互可见。这里，我们讨论几个变体。下面讨论的变体中常用的术语：

*   一个简单（不一定是凸的）多边形 `p` 被用来描述美术馆。
*   一组点 `s` 被用来描述守卫，其中每个守卫由 `p` 中的一个点来表示。
*   一个点 `A ∈ S` 可以守卫另一个点 `B ∈ P` 当且仅当 `A ∈ S`、`B ∈ P` 和线段 `AB` 完全位于多边形 `p` 内部。

这个**美术馆问题**的很多变体被归类为 [NP 难问题](https://www.geeksforgeeks.org/difference-between-np-hard-and-np-complete-problem/)。在这里，我们将重点介绍那些承认多项式解的：

*   变体 1：确定集合 `s` 最小大小的上限。
*   变体 2：确定一个关键点 `C` 和多边形 `P` 中的另一个点 `D`，如果守卫位于 `C` 的位置，则 `D` 不可见。
*   变体 3：确定多边形 `p` 是否可以仅由一个守卫守卫。
*   变体 4：如果守卫只能放置在多边形 `p` 的顶点并且只需要守卫顶点，则确定集合 `s` 的最小大小。

*请注意，还有很多变体，至少已经为其编写了一本书。*

## 解

*   变体 1 的解决方案是 **Vaclav Chvatal** 的美术馆定理的理论工作。他指出，对于一个有 `n` 个顶点的简单多边形，`⌊n/3⌋` 个守卫总是足够且有时是必要的。
*   变体 2 的解决方案包括测试多边形 `p` 是否是凹的（因此有一个关键点）。我们可以使用 `isconvex` 函数的否定形式。
*   如果你以前没见过变体 3 的解决方案，它可能很难。我们可以使用**切割多边形**的函数。我们用 `p` 中所有边形成的线逆时针切割多边形 `p`，并始终保留左侧部分。如果最后我们还有一个非空的多边形，我们可以在那个非空多边形中放置一个守卫来保护整个多边形 `p`。
*   变体 4 的解决方案涉及计算多边形 `p` 的“可见图”的最小顶点覆盖。一般来说，这是另一个 **NP-hard 问题**。
