# 图形的基本属性

> 原文：[https://www.geeksforgeeks.org/basic-properties-of-a-graph/](https://www.geeksforgeeks.org/basic-properties-of-a-graph/)

图是由节点和边组成的非线性数据结构。节点有时也称为顶点，边是连接图中任意两个节点的直线或圆弧。

根据图的结构，图的性质基本上用于描述图的特征。我们用属于图论领域的特定术语定义了这些性质。在本文中，我们将讨论图的一些性质，如下所示：

## 1. 两个顶点之间的距离

它基本上是顶点 A 和顶点 B 之间最短路径中可用的边的数量。如果有多条边用于连接两个顶点，那么我们基本上将最短路径视为这两个顶点之间的距离。

```
使用的表示法：
d(A, B)
这里的函数 d 基本上表示顶点 A 和顶点 B 之间的距离。
```

让我们用一个例子来理解这一点：
![](img/08dbc601e31f48522d98a3988b67e3f2.png)
在上图中，让我们试着找出顶点 b 和 d 之间的距离。

```
d(b, c)
我们可以从顶点 b 到顶点 d 有不同的方式，例如
1. ba, af, fe, ed 这里 d(b, c) 将是 4。
2. ba, af, fc, cd 这里 d(b, c) 将是 4。
3. bc, cf, fe, ed 这里 d(b, c) 将是 4。
4. bc, cd 这里 d(b, c) 将是 2。
因此，顶点 b 和顶点 d 之间的最小距离是 2。
```

## 2. 顶点的离心率

从一个顶点到所有其他顶点的最大距离被认为是该顶点的离心率。

```
使用的表示法：
e(V)
这里的 e(v) 确定顶点 V 的离心率。
```

让我们试着用下面的例子来理解这一点。
![](img/08dbc601e31f48522d98a3988b67e3f2.png)

```
从上图中，让我们试着找出顶点 b 的离心率。
e(b)
d(b, a)=1
d(b, c)=1
d(b, d)=2
d(b, e)=3
d(b, f)=2
d(b, g)=2
因此，顶点 b 的离心率是 3。
```

## 3. 连通图的半径

所有顶点的离心率中的最小值基本上被认为是连通图的半径。

```
使用的表示法：
r(G)
这里的 G 是连通图。
```

让我们试着用下面的例子来理解这一点。
![](img/08dbc601e31f48522d98a3988b67e3f2.png)

```
从上图中：
r(G) 是 2。
因为所有顶点的离心率的最小值是 2。
```

## 4. 连通图的直径

与连通图的半径不同，这里我们基本上使用所有顶点的离心率的最大值来确定图的直径。

```
使用的表示法：
d(G)
其中 G 是连通图。
```

让我们试着用下面的例子来理解这一点。
![](img/08dbc601e31f48522d98a3988b67e3f2.png)

```
从上图中：
d(G) 是 3。
因为所有顶点的离心率的最大值是 3。
```

## 5. 中心点与中心

具有最小离心率的顶点被认为是图的中心点。所有中心点的集合被认为是图的中心。

```
如果
e(V)=r(G)
那么 v 是中心点。
```

让我们试着用下面的例子来理解这一点。
![](img/08dbc601e31f48522d98a3988b67e3f2.png)

```
在上图中，中心点将是 f。
因为
e(f)=r(G)=2
因此 f 被认为是图的中心点。
因此 f 也是图的中心。
```
