# Java 中的重要关键词

> 原文: [https://www.geeksforgeeks.org/important-keywords-java/](https://www.geeksforgeeks.org/important-keywords-java/)

## 非访问修饰符

1.  `abstract`: 适用于类和方法的非访问修饰符。它用于实现抽象。更多信息，请参考 Java 中的[抽象关键字](https://www.geeksforgeeks.org/abstract-keyword-in-java/)。
2.  `enum`: 用于在 Java 中定义[枚举](https://www.geeksforgeeks.org/enum-in-java/)。
3.  `instanceof`: 用于知道对象是否为指定类型(类或子类或接口)的实例。
4.  `private`: 是访问修饰符。任何声明为私有的东西都不能在它的类之外看到。
5.  `protected`: 如果您希望允许在当前包之外看到一个元素，但只允许看到直接子类化您的类的类，那么请声明该元素受保护。
6.  `public`: 任何声明为公共的东西都可以从任何地方访问。

有关访问修饰符的更多信息，请参考 Java 中的[访问修饰符](https://www.geeksforgeeks.org/access-modifiers-java/)。

## 其他重要关键词

1.  `static`: 用于创建成员（块、方法、变量、嵌套类），该成员可以在不引用特定实例的情况下自行使用。更多信息，请参考 Java 中的 [`static` 关键字](https://www.geeksforgeeks.org/static-keyword-java/)。
2.  `strictfp`: 用于限制浮点计算，并确保在浮点变量中执行操作时在每个平台上获得相同的结果。更多信息请参见 Java 中的 [`strictfp` 关键字](https://www.geeksforgeeks.org/strictfp-keyword-java/)。
3.  `synchronized`: 适用于块和方法。它用于在 Java 中实现同步。更多信息请参考 Java 中的 [`synchronized`](https://www.geeksforgeeks.org/synchronized-in-java/)。
4.  `transient`: `transient` 是一个变量修饰符，用于[序列化](https://www.geeksforgeeks.org/serialization-in-java/)。在序列化时，如果我们不想在文件中保存特定变量的值，那么我们使用 `transient` 关键字。更多信息请参考 Java 中的 [`transient` 关键字](https://www.geeksforgeeks.org/transient-keyword-java/)。
5.  `volatile`: `volatile` 修饰符告诉编译器，被 `volatile` 修改的变量可能会被程序的其他部分意外更改。更多信息，请参考 Java 中的 [`volatile` 关键字](https://www.geeksforgeeks.org/volatile-keyword-in-java/)。
