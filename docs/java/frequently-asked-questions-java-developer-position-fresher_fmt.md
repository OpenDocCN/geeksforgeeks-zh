# 新鲜出炉的 Java 开发人员职位常见问题

## 基于 Java:

1.  `OOPs` 概念（主要是多态，以及封装与抽象的区别）
2.  `abstract class` 和 `interface` 的区别。
3.  `==` 和 `.equals()` 函数的区别。
4.  什么是 `hash map`？
5.  什么是 `hash table`？
6.  类型异常。
7.  异常（`exceptions`）和错误（`errors`）的区别。
8.  `throw`、`throws` 和 `throwing` 的区别。
9.  `final`、`finally` 和 `finalize` 的区别。
10. 什么是 `class`？
11. 什么是 `object`？
12. `class` 和 `object` 的区别。
13. 方法重载（`Method overload`）。
14. 什么是 `thread`？
15. 创建线程的不同方法。
16. `C` 和 `Java` 的区别。
17. `C++` 和 `Java` 的区别。
18. `Java` 中的文件处理。

## 算法:

1.  二分法（`method of bisection`）
2.  快速排序（复杂度和逻辑）
3.  归并排序（复杂度和逻辑）
4.  归并排序和快速排序的区别
5.  哪个更好，归并排序还是快速排序，为什么？
6.  广度优先搜索（`BFS`）
7.  深度优先搜索（`DFS`）

## 数据结构:

1.  什么是二叉树（`binary tree`）？
2.  什么是链表（`linked list`）及其类型。
3.  使用链表实现栈/队列（`stack/queue`）。
4.  内存管理（堆 `heap` 和栈 `stack`）
5.  `AVL` 树
6.  数组列表（`ArrayList`）。
7.  反转链表（`Reverse linked list`）。
8.  查找链表的中间元素。
9.  如何检测链表中的环？
10. 二叉树的俯视图/左视图/右视图/对角线视图
11. 将矩阵向右旋转。
12. 检查一个二叉树是否是另一个二叉树的子树。

## 操作系统:

1.  进程（`process`）和线程（`thread`）的区别。
2.  什么是死锁（`deadlock`）？
3.  临界区问题（`Critical section problem`）
4.  什么是颠簸（`thrashing`）？
5.  分页（`Paging`）。
6.  缺页中断（`Page fault`）。

## DBMS:

1.  主键（`primary key`）和候选键（`candidate key`）的区别。
2.  什么是连接（`connection`）以及连接的类型。
3.  从员工列表中找出第二高的薪水。
4.  `ACID`。
