# 十大顺风 CSS 插件

> 原文:[https://www.geeksforgeeks.org/top-10-tailwind-css-plugins/](https://www.geeksforgeeks.org/top-10-tailwind-css-plugins/)

`Tailwind CSS`：这被称为实用优先的 CSS 框架，用于构建自定义 UI。这是一个非常可定制的低级 CSS 框架，它为您提供了构建模块，帮助您构建出色的设计，而没有任何烦人的风格。

排名前十的顺风 CSS 插件如下：

## 1. 顺风 CSS 排版

*   如果你想设计文章或文档的样式，或者可能是一篇博客文章。借助 `Tailwind CSS`，这非常困难，因为需要更多地关注排版和大量复杂的自定义 CSS。
*   这个功能有助于在前端构建应用程序，因为你花在组件建模上的时间非常少。

可以通过以下命令下载：

```
// Using npm
npm install @tailwindcss/typography

// Using Yarn
yarn add @tailwindcss/typography
```

## 2. 顺风 CSS 自定义表单

*   `Tailwind CSS` 自定义表单用于提供更好的外观，并通过添加一些实用程序。
*   帮助提供更好的自定义，以便轻松定制和制作更好的表单。

可以通过以下命令下载：

```
// Using npm
npm install @tailwindcss/custom-forms --save-dev

// Using Yarn
yarn add @tailwindcss/custom-forms -D
```

## 3. 顺风 CSS 表格插件

*   这个插件用于在 `Tailwind CSS` 中创建指导表格。
*   继承自 `Bootstrap`，借助它，可以使嵌套表格与父表格相同。

可以通过以下命令下载：

```
// With using NPM
npm install tailwindcss-tables --save

// With using Yarn
yarn add tailwindcss-tables
```

## 4. 顺风 CSS 主题化插件

*   `tailwindcss-theming` 是一个 CSS 插件，负责更改网站中的多个主题。
*   通常用于在网站中制作深色主题。

可以通过以下命令下载：

```
yarn add tailwindcss-theming@next --dev
```

## 5. 顺风 CSS 截断多行

*   当我们想要截断某些卡片内的文本时，它很有帮助，并且意识到仅用 CSS 可能很难做到这一点。
*   为此，我们可以使用 `Tailwind CSS`，这是一个用于自定义的插件。它用来截断多行文本的元素是添加类 `truncate-key-line`，其中 `key` 是截断前显示的行数。

您可以通过以下命令下载它：

```
// With using npm
npm install tailwindcss-truncate-multiline --save

// With using yarn
yarn add tailwindcss-truncate-multiline
```

## 6. 顺风 CSS-Elevation

*   这个插件提供了两个不同图像或组件的 `elevation` 角度。
*   每当您想应用 `elevation` 时，您需要使用 `elevation` 实用程序。
*   这是一个向按钮添加 CSS `elevation` 的基本示例：`<button class="elevation-1">Click me!</button>`

可以通过以下命令下载：

```
npm install tailwindcss-elevation
```

## 7. 顺风 CSS 长宽比

*   这种类型的插件是在 `aspect-ratio` 之前创建的，并且受任何浏览器支持。
*   这个插件需要 `Tailwind CSS` 1.2 或更高版本。它在不同形状中提供自定义尺寸。
*   这个插件使用配置的主题和变体对象来创建 `aspect-ratio` 实用程序。

可以通过以下命令下载：

```
npm install tailwindcss-aspect-ratio
```

## 8. 顺风 CSS 渐变

*   这个插件可以帮助您在两种添加的颜色之间提供非常平滑的过渡。
*   这些插件由开发者定期更新。

可以通过以下命令下载：

```
npm install tailwindcss-gradients
```

## 9. 顺风 CSS 调试屏幕

*   `tailwindcss-debug-screens` 是一个显示响应式断点的插件。
*   您可以轻松自定义主题并调试 `tailwind.config.js` 文件。
*   您可以设置调试器选项以轻松查找断点。

可以通过以下命令下载：

```
npm install tailwindcss-debug-screens --save-dev
```

## 10. 顺风 CSS Spinner

*   所有网络应用程序都需要一个加载器来加载数据，因此 `tailwindcss-spinner` 是一个加载器，可以根据用户的需求轻松自定义。您还可以根据需要更改其颜色、大小、宽度等。
*   可以与单个实用程序类一起使用。

```
// With using yarn
yarn add -D tailwindcss-spinner

// With using npm
npm install --save-dev tailwindcss-spinner
```
