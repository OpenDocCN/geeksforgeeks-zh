# 阿里巴巴云AnalyticDB for MySQL与亚马逊Neptune的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-alibaba-cloud-analyticdb-for-mysql-and-amazon-neptune/](https://www.geeksforgeeks.org/difference-between-alibaba-cloud-analyticdb-for-mysql-and-amazon-neptune/)

## 1. 阿里巴巴云AnalyticDB for MySQL
它是阿里巴巴云上的在线分析处理数据库系统。它同时提供`PostgreSQL`和`MySQL`版本。`MySQL`的`AnalyticDB`是一个实时数据仓库服务，能够以高并发和低延迟处理数十亿字节的数据。它与`MySQL`协议和`SQL:2003`语法完全兼容。是阿里巴巴开发的。

## 2. 亚马逊Neptune
这是`Amazon.com`为云产品建立的快速、可靠的图形数据库。它是亚马逊网络服务的一部分。2017年11月29日公布。它支持开源的`Apache TinkerPop Gremlin`图遍历语言。它允许用户使用通过`AWS Key Management Service`创建的密钥来加密数据库。
