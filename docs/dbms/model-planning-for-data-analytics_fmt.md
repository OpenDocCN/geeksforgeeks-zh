# 数据分析的模型规划

> 原文: [https://www.geeksforgeeks.org/model-planning-for-data-analytics/](https://www.geeksforgeeks.org/model-planning-for-data-analytics/)

在本文中，我们将讨论数据分析的模型规划，其中我们将逐一介绍所有过程步骤。

模型规划是数据分析生命周期阶段的第 3 阶段，在此阶段，团队确定其打算为后续模型构建阶段遵循的方法、技术和工作流。

在此阶段，该团队参考在发现过程中开发的假设，在此阶段，他们首先熟悉数据并了解业务问题或领域。

## 模型规划阶段的常用工具

### `R`

*   其基本优势是它可以轻松开发高质量的绘图，包括所需的数学公式。
*   `SQL` 最著名的用途是作为构建其仪表板的基础架构，与报告工具配合使用非常方便。
*   为了更快地创建和与数据库交互，`SQL` 已被适配到各种工具中，每个工具都有自己的利基市场，包括 `Microsoft Access` 和 `PostgreSQL`。

### `SQL`

*   它易于访问，可用于构建复杂模型和快速分析，并提供强大的数据操作能力。
*   应用管理器的 `SQL` 服务器监控部分采用表格格式，便于在实时数据屏幕和分析功能访问之间切换。
*   通过直观的界面广泛获取数据，无论数据存储在何处。
*   您可以轻松访问数据，对 `SQL` 的知识要求很少，适用于数据或表面数据。
*   通过与加载器的无缝界面轻松工作，无需详细了解每个加载器即可使用。
*   通过基本存储选项（包括物化视图、临时表和分区表）来提高效率。

### `Tableau Public`

*   它是一个免费软件，可以连接到任何数据源以获取公司的基于网络的数据。
*   它允许访问不同格式的下载文件。
*   数据可以通过社交媒体共享。
*   如果有人想看看 `Tableau` 的优越性，这是一个非常好的数据源。

### `SAS`

*   它是一个用于数据操作的编程环境和语言。
*   `SAS` 易于管理和访问，用于从各种来源观察数据。
*   `SAS` 组织了用于网络、社交媒体和营销分析的模块，广泛应用于客户前景分析。
*   它还用于预测客户行为和沟通。

### `RapidMiner`

*   它是一个在数据科学领域高度集成的平台，执行预测分析。
*   它包括高级分析，如数据挖掘和机器学习，无需任何编程。
*   这个工具非常强大，可以基于真实数据生成分析。
