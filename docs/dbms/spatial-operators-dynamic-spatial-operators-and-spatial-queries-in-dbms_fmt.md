# DBMS 中的空间算子、动态空间算子和空间查询

> 原文: [https://www.geeksforgeeks.org/spatial-operators-dynamic-spatial-operators-and-spatial-queries-in-dbms/](https://www.geeksforgeeks.org/spatial-operators-dynamic-spatial-operators-and-spatial-queries-in-dbms/)

## 1. 空间运算符
空间运算符应用于对象的几何属性。然后在物理空间中使用它来捕捉它们以及它们之间的关系，也用于进行空间分析。

空间运算符分为三类：

### 1.1 拓扑算子
拓扑性质在应用拓扑操作（如平移或旋转）时不会改变。

拓扑算子在许多层次上是分层结构的。基本层为操作员提供了检查宽边界区域之间详细拓扑关系的能力。更高的级别提供了更抽象的操作符，允许用户独立于几何数据模型查询不确定的空间数据。

**示例：** `open(region)`、`close(region)` 和 `inside(point, ring)`。

### 1.2 投影算子
投影算子，如凸包（`convex hull`），用于建立关于对象凹凸性的谓词。

**示例：** 物体内部的凹度。

### 1.3 度量算子
度量算子的任务是提供对象几何形状的更精确描述。它们通常用于测量单个对象的全局属性，以及测量不同对象之间的相对位置（距离和方向）。

**示例：** （弧的）长度和（点到点的）距离。

## 2. 动态空间操作符
动态操作改变操作符所应用的对象。创建、销毁和更新是基本的动态操作。

**示例：** 通过平移、旋转、缩放、反射和剪切来移动空间对象。

## 3. 空间查询
对需要使用空间操作的空间数据的请求称为空间查询。它可以分为：

### 3.1 范围查询
它查找在给定空间区域内的特定类型的所有对象。

**示例：** 查找西里古里地区的所有医院。这个查询的一个变体是对于给定的位置，查找特定距离内的所有对象，例如，查找 5 公里范围内的所有银行。

### 3.2 最近邻查询
它查找距离给定位置最近的特定类型的对象。

**示例：** 找到离事故地点最近的派出所。

### 3.3 空间连接或叠加
它基于空间条件连接两种类型的对象，例如相交或空间上重叠的对象。

**示例：** 在两个城市之间的国道上找到所有的达巴斯。它在空间上将乡镇对象和公路对象连接起来。查找火车站 5 公里范围内的所有酒店。它在空间上连接火车站对象和酒店对象。
