# 甲骨文与 SQL Server 的比较

> 原文：[https://www.geeksforgeeks.org/comparisons-between-oracle-vs-sql-server/](https://www.geeksforgeeks.org/comparisons-between-oracle-vs-sql-server/)

在几个电子数据库管理系统中，最受欢迎和广泛使用的两个是：`Oracle` 和 `SQL Server`。

`Oracle` 支持的操作系统有：`Windows`、`Linux`、`Solaris`、`HP-UX`、`OS X` 等。

而 `SQL Server` 只支持 `Windows` 和 `Linux` 操作系统。`Oracle` 和 `SQL Server` 之间有几个基本的差异，其中最重要的差异是它们使用的语言。`Oracle` 使用过程语言，而 `SQL Server` 使用事务处理语言。

## Oracle 和 SQL Server 之间的区别

| S.NO | Oracle | SQL Server |
| :--- | :--- | :--- |
| 1. | `PL/SQL` 由 `Oracle` 使用。 | 而 `T-SQL` 是由 `SQL Server` 使用的。 |
| 2. | 就复杂性而言，它是复杂的。 | 虽然简单，但就复杂性而言。 |
| 3. | `Oracle` 不好用。 | 虽然它比 `Oracle` 更容易使用。 |
| 4. | `Oracle` 支持的操作系统有：`Windows`、`Linux`、`Solaris`、`HP-UX`、`OS X` 等。 | 而 `SQL Server` 只支持 `Windows` 和 `Linux` 操作系统。 |
| 5. | 在 `Oracle` 中，用户共享数据库。 | 在 `SQL Server` 中，用户不共享数据库。 |
| 6. | 还有就是软件包都是在 `Oracle` 发生的。 | 而有包则不发生在 `SQL Server` 中。 |
| 7. | 它比 `SQL Server` 更强大。 | 虽然它不如 `Oracle` 强大。 |
