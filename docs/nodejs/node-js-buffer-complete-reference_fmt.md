> 原文：[https://www.geeksforgeeks.org/node-js-buffer-complete-reference/](https://www.geeksforgeeks.org/node-js-buffer-complete-reference/)

缓冲区是 Node.js 中 `Buffer` 类的实例。缓冲区设计用于处理二进制原始数据。缓冲区在 V8 堆之外分配原始内存。`Buffer` 类是一个全局类，因此可以在应用程序中不导入 `buffer` 模块的情况下使用它。
