# 用海鸟和熊猫制作时间序列图

> 原文:[https://www . geesforgeks . org/creating-a-time-series-plot-with-seaborn-and-pandas/](https://www.geeksforgeeks.org/creating-a-time-series-plot-with-seaborn-and-pandas/)

## 概述

在这篇文章中，我们将学习如何创建一个与海鸟和熊猫的时间序列图。让我们讨论一些概念:

*   `Panda` 是一个基于 `NumPy` 库的开源库。这是一个 Python 包，提供了各种数据结构和操作来处理数值数据和统计信息。主要是因为它更容易导入和分析数据。Panda 的速度和高性能对用户来说是富有成效的。
*   `seaborn` 是一个用于在 Python 中绘制统计图形的巨大可视化库。它提供了漂亮的默认样式和调色板，以形成更具吸引力的统计图表。它构建在 `matplotlib` 库的最高级别，并且也与 `pandas` 的信息结构紧密集成。
*   一个 **时间序列图**（有时称为统计图）显示与时间对应的数值。它们几乎像 x-y 图，但尽管 x-y 图可以绘制“X”变量（例如，身高、体重和年龄）的分布，时间序列图只能在 X 轴上显示时间。与饼图和条形图不同，这些图表没有类别。时间序列图有助于显示数据如何随时间变化。例如，如果你随机抽样数据，这个图表将非常有效。

### 所需步骤

1.  导入包
2.  导入/加载/创建数据。
3.  使用线图绘制数据的时间序列图（因为自 2020 年 9 月以来，`tsplot` 已被 `lineplot` 取代）。
