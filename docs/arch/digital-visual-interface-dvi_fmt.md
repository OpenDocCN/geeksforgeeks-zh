# 数字视觉接口(DVI)

> 原文：[https://www.geeksforgeeks.org/digital-visual-interface-dvi/](https://www.geeksforgeeks.org/digital-visual-interface-dvi/)

`数字视觉接口`是用于传输模拟或数字视频信号的接口。我们还使用 `VGA` 和 `HDMI` 发送视频信号。`DVI` 仅用于传输显示器中的视频信号，而不是音频信号。它们在改善任何电子设备的照片或视觉质量方面有着非凡的作用。它通常与液晶显示器、投影仪和电视等设备相关联。这些电缆提供清晰优质的图片，这显然是一个良好的用户体验。

## 数字视频接口的类型

这些解释如下。

### 1. DVi-A

`DVi-A` 是 `DVI` 的旧版本，如今使用的人非常少。这里的 `A` 代表模拟信号。`DVi-A` 的全称是 `Digital Visual Interface- Analogue`。这意味着它只能向显示器传输模拟信号，而不能传输数字信号。它的最大分辨率是 `1920 * 1080`。

`DVi-A` 的标识是，它的右上角有一个杆，杆的上方有 2 个引脚，杆的下方有 2 个引脚，另外还有 12 个引脚可供使用。

### 2. DVi-D

`DVi-D` 代表 `Digital Visual Interface- Digital`。它非常常见，被大多数人使用。它传输的视频信号是数字信号，而不是模拟信号。我们不能将 `DVi-A` 和 `DVi-D` 连接在一起，因为 `DVi-A` 读取模拟信号，而 `DVi-D` 读取数字信号。因此，由于格式不同，我们不能同时使用这两种接口。

此外，它有两种类型：`Single Link` 和 `Dual Link`。

`Single Link` 有一个杆，其侧面有两个块，每个块有 9 个销。`DVi-D(Single Link)` 给出的最大分辨率为 `1920 * 1080`。`Dual Link` 也有一个杆，有一个 3 行，每行有 8 个引脚。`Dual Link` 给出的最大分辨率为 `2048 * 1536`。

这是现在非常常见的接口。`Single Link` 可以向监视器传输一个信号，`Dual Link` 可以向监视器传输两个信号，这就是为什么与 `Single Link` 相比，`Dual Link` 支持最大分辨率的原因。

### 3. DVi-i

`DVi-i` 可以传输模拟或数字两种视频信号。由于它支持两种信号，因此也可以将其与 `DVi-A` 或 `DVi-D` 连接。它也有两种进一步的类型：`Single Link` 和 `Dual Link`。

`Single Link` 有一个杆，杆上有两个销，杆下有两个销。带杆，它有 2 个块，每个块有 9 个销。它支持的最大分辨率是 `1600 * 1200`。`Dual Link` 有一个杆，杆上有两个销，杆下有两个销。它也有 3 行，每行 8 个引脚。`Dual Link` 支持的最大分辨率为 `2048 * 1536`。
