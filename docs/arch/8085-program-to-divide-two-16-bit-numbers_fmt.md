# 8085 程序划分两个 16 位数字

> 原文: [https://www.geeksforgeeks.org/8085-program-to-divide-two-16-bit-numbers/](https://www.geeksforgeeks.org/8085-program-to-divide-two-16-bit-numbers/)

## 问题
在 8085 微处理器中编写汇编语言程序，对两个 16 位数字进行除法运算。

## 假设
- 程序起始地址: `2000`
- 输入存储位置: `2050`、`2051`、`2052`、`2053`
- 输出内存位置: `2054`、`2055`、`2056`、`2057`

## 示例
```
INPUT:
       (2050H) = 04H
       (2051H) = 00H 
       (2052H) = 02H 
       (2053H) = 00H
OUTPUT:
        (2054H) = 02H
        (2055H) = 00H
        (2056H) = FEH
        (2057H) = FFH 
```
结果: 因此我们划分了两个 16 位数字。

## 算法
1. 将寄存器 `BC` 初始化为商的 `0000H`。
2. 载入 `HL` 对中的除数，保存在 `DE` 寄存器对中。
3. 将红利装入 `HL` 对。
4. 用 `E` 寄存器减去累加器的内容。
5. 将内容 `A` 移动到 `C`，将内容 `H` 移动到 `A`。
6. 用 `D` 减去借 `A` 的内容。
7. 将累加器的值移动到 `H`。
8. 如果 `CY=1`，转到步骤 10，否则转到下一步。
9. 递增寄存器 `B` 并跳至步骤 4。
10. 添加 `DE` 和 `HL` 的内容。
11. 将剩余部分存储在内存中。
12. 将 `C` 到 `L` & `B` 的内容移到 `H`。
13. 将商存储在内存中。

## 程序
| 存储地址 | 记忆术 | 评论 |
| --- | --- | --- |
| `2000` | `LXI B, 0000H` | 将商初始化为 `0000H` |
| `2003` | `LHLD 2052H` | 在 `HL` 中加载除数 |
| `2006` | `XCHG` | `HL` 和 `DE` 交换 |
| `2007` | `LHLD 2050H` | 加载红利 |
| `200A` | `MOV A, L` | `A <- L` |
| `200B` | `SUB E` | `A <- A - E` |
| `200C` | `MOV L, A` | `L <- A` |
| `200D` | `MOV A, H` | `A <- H` |
| `200E` | `SBB D` | `A <- A - D - CY` |
| `200F` | `MOV H, A` | `H <- A` |
| `2010` | `JC 2017` | 进位时跳跃 |
| `2013` | `INX B` | `B <- B + 1` |
| `2014` | `JMP 200B` | 跳转到 `200B` |
| `2017` | `DAD D` | `HL <- HL + DE` |
| `2018` | `SHLD 2056H` | `HL` 存储在存储器中 |
| `201B` | `MOV L, C` | `L <- C` |
| `201C` | `MOV H, B` | `H <- B` |
| `201D` | `SHLD 2054H` | `HL` 存储在存储器中 |
| `2020` | `HLT` | 终止程序 |

## 解释
1. `LXI B, 0000H`: 将 `BC` 寄存器初始化为 `0000H`。
2. `LHLD 2052H`: 加载地址为 `2052` 的 `HL` 对。
3. `XCHG`: 用 `DE` 对寄存器交换 `HL` 对的内容。
4. `LHLD 2050H`: 加载地址为 `2050` 的 `HL` 对。
5. `MOV A, L`: 将寄存器 `L` 的内容移入寄存器 `A`。
6. `SUB E`: 用累加器的内容减去寄存器 `E` 的内容。
7. `MOV L, A`: 将寄存器 `A` 的内容移入寄存器 `L`。
8. `MOV A, H`: 将寄存器 `H` 的内容移入寄存器 `A`。
9. `SBB D`: 用带进位的累加器的内容减去寄存器 `D` 的内容。
10. `MOV H, A`: 将寄存器 `A` 的内容移入寄存器 `H`。
11. `JC 2017`: 有进位跳转到地址 `2017`。
12. `INX B`: 将 `BC` 寄存器递增 1。
13. `JMP 200B`: 跳转到地址 `200B`。
14. `DAD D`: 添加 `DE` 和 `HL` 对的内容。
15. `SHLD 2056H`: 将 `HL` 对的内容存储到内存地址 `2056` 和 `2057` 中。
16. `MOV L, C`: 将寄存器 `C` 的内容移入寄存器 `L`。
17. `MOV H, B`: 将寄存器 `B` 的内容移入寄存器 `H`。
18. `SHLD 2054H`: 将 `HL` 对的内容存储到内存地址 `2054` 和 `2055` 中。
19. `HLT`: 终止程序的执行。
