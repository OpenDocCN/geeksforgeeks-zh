# 树莓派和橘子派的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-raspberry-pi-and-orange-pi/](https://www.geeksforgeeks.org/difference-between-raspberry-pi-and-orange-pi/)

**1. 树莓派：**
树莓派可以被称为一台小型电脑，它是一个信用卡大小的设备，可以插入电脑，电脑有标准的键盘和鼠标。它还具有计算能力，可以被教如何用语言编程。一般来说，这些设备用于制造 Ariel 无人机，并与天气探测程序一起使用。众所周知，它的功能是以可承受的价格提供的，尽管这取决于您购买的变体。说到功能，它是一台迷你电脑，可以像台式电脑一样为您服务。从上网看高清视频到玩轻模式游戏。它支持开箱即用的 `Python` 语言。

**2. 橘子派：**
橘子派也是市场上另一款拥有所有功能的迷你电脑，与树莓派一起考虑。它是在对树莓派有巨大需求的时候开发的，结果像电脑一样的克隆体被开发出来，并用橘子派命名。用一些功能较少的 `Pi` 支持不同的操作系统如 `Linux`、`Ubuntu`、`Android`。与原厂 `Pi` 相比，`Orange Pi` 的 `USB` 端口更少，成本有所降低。它有更快的网络和一些更重的功能，这使得它很难像 `Pi` 一样快。通过新的集成，它已经开发了板载无线网络，但缺乏树莓派那样的社区支持。

## 树莓派和橘子派的区别

| 基础 | 树莓派 | 橘子派 |
| --- | --- | --- |
| 中央处理器 | 它使用 1.2 千兆赫 64 位四核处理器 | 它采用 `H5` 四核 `Cortex-A53` 处理器 |
| 随机存取存储 | 就其功能而言，它使用 `1GB DDR2` 内存 | 为了获得更好的性能，它使用了 `2GB` 的 `DDR3` 内存 |
| 图形处理器 | 为了处理图形需求，它使用博通 `VideoCore IV` 图形处理器。 | 为了增强图形功能，它使用了 `Mali-450` 图形处理器。 |
| 储存 | 对于存储要求，它提供了板载存储模块。 | 为了更好的存储升级，它有 `SD` 卡插槽用于存储。 |
| 重量 | 它重约 45g | 它重约 68 克 |
| 视频兼容 | 它支持视频，但不支持 4K。 | 不支持 4K 视频 |
| `USB` 连接埠 | 它有 4 个 `USB 2.0` 和 1 个微 `OTG` 接口。 | 它有 3 个 `USB 2.0` 和 1 个 `USB 2.0 OTG` 接口 |
| 以太网/无线网络 | 是的，它能够通过互联网发送数据，并且支持以太网和 `WiFi`。 | 是的，它能够通过互联网发送数据，并配有以太网和无线网络 |
| 蓝牙 | 是的，支持。 | 是的，它支持。 |
| 操作系统支持 | `Raspbian`、`Windows IoT Core`、`OSMX`、`Linux`、`Ubuntu`、`Debian` 等。 | `Android`、`Ubuntu`、`Raspbian`、`Debian` |
