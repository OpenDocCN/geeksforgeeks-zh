# 浮点表示法介绍

> 原文: [https://www.geeksforgeeks.org/introduction-of-floating-point-representation/](https://www.geeksforgeeks.org/introduction-of-floating-point-representation/)

## 1. 将浮点转换为十进制

在 `32位` 浮点表示中，有 3 个元素：
i) 符号
ii) 指数
iii) 尾数

*   **符号**位是二进制表示的第一位。“1”表示负数，“0”表示正数。
    **例:** `11000001110100000000000000000000`
*   **指数**由二进制表示的接下来的 `8位` 决定。`127` 是 `32位` 浮点表示的唯一数字，称为偏置。它由 `2^(k-1) - 1` 确定，其中 `k` 是指数字段的位数。

`8位` 表示中有 `3` 个指数位，`32位` 表示中有 `8` 个指数位。

因此：
`8位` 转换偏置 = `3` (`2^(3-1)-1 = 4-1 = 3`)
`32位` 转换偏置 = `127` (`2^(8-1) -1 = 128-1 = 127`)

**例:** `0100000111010000000000000000000`
`100000011` = `131` (十进制)
`131 - 127 = 4`

因此 `2` 的指数将是 `4`，即 `2^4 = 16`。

*   **尾数**从二进制表示的剩余 `23位` 计算得出。它由隐含的“1”和一个小数部分组成，小数部分由下式确定：

**示例:**

尾数的小数部分由下式给出:
`1*(1/2) + 0*(1/4) + 1*(1/8) + 0*(1/16) + ... = 0.625`

因此尾数将是 `1 + 0.625 = 1.625`

因此给出的十进制数为: 符号 * 指数 * 尾数 = `(-1)^0 * 16 * 1.625 = 26`

## 2. 将十进制转换为浮点

在 `32位` 浮点表示中，有 3 个元素：
i) 符号 (MSB)
ii) 指数 (MSB 后 `8位`)
iii) 尾数 (剩余 `23位`)

*   **符号位**是二进制表示的第一位。“1”表示负数，“0”表示正数。
    示例: 将 `-17` 转换为 `32位` 浮点表示，符号位 = `1`
*   **指数**由最接近且小于或等于 `2^n` 的数决定。对于 `17`，`16` 是最接近的 `2^n`。因此 `2` 的指数将是 `4`，因为 `2^4 = 16`。`127` 是 `32位` 浮点表示的唯一数字，称为偏置。它由 `2^(k-1) - 1` 确定，其中 `k` 是指数字段的位数。

因此，`32位` 偏置 = `127` (`2^(8-1) -1 = 128-1 = 127`)

现在，`127 + 4 = 131`，即二进制表示中的 `10000011`。

*   **尾数:** `17` 的二进制 = `10001`。

移动二进制点，使其只剩下一位。调整 `2` 的指数，使数值不变。这是标准化的数字。`1.0001 × 2^4`。现在，考虑小数部分，通过添加零表示为 `23位`。
