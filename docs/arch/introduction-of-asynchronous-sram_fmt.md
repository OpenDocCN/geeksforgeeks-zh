# 异步 SRAM 介绍

> 原文：[https://www.geeksforgeeks.org/introduction-of-asynchronous-sram/](https://www.geeksforgeeks.org/introduction-of-asynchronous-sram/)

随机存取存储器，即[随机存取存储器](https://www.geeksforgeeks.org/random-access-memory-ram/)是计算机存储设备的主要数据来源。该内存有两种形式，命名为：

*   [静态随机存取存储器](https://www.geeksforgeeks.org/different-types-ram-random-access-memory/)
*   [动态随机存取存储器](https://www.geeksforgeeks.org/difference-between-sram-and-dram/)

而静态随机存取存储器也大致分为两种形式，分别是：

*   同步静态随机存取存储器
*   异步随机存取存储器

在本文中，我们将研究什么是异步静态随机存取存储器。这里需要注意的一点是，同步动态随机存取存储器不同于异步静态随机存取存储器。

## 异步 SRAM

*   首先，让我们看看异步在这里是什么意思。不依赖外部时钟脉冲工作的器件称为异步器件，不依赖外部时钟工作的静态随机存取存储器称为异步静态随机存取存储器。
*   它使用锁存电路，尤其是触发器。由于该静态随机存取存储器不依赖于任何外部时钟脉冲，因此一旦接收到指令，它就可以根据指令要求进行读取、写入。
*   通常优选用作小型高速缓存嵌入式处理器。它位于易失性存储器之下，因为只要持续向系统供电，它就可以存储该值。所以异步静态随机存取存储器中的数据本质上是非常不稳定的。动态随机存取存储器将数据作为电荷存储在电容器上。
*   用于构建异步静态随机存取存储器的电路架构本质上是复杂的，因此用于构建其拓扑的组件数量会更多，这一因素使得静态随机存取存储器比动态随机存取存储器更昂贵。在许多家用电脑中，都使用 `DRAM`，如 `DDR3`、`DDR5` 等。
*   现在，由于数据被静态存储在这个静态随机存取存储器中，所以静态随机存取存储器所需的功率比动态随机存取存储器少。静态随机存取存储器是静态存储数据的，因此与动态随机存取存储器相比，它存储数据的速度更快。

## 异步静态随机存取存储器的状态

它可以处于 3 种状态，即：

*   **待机（Standby）**：这是 `SRAM` 的一种状态，此时 `SRAM` 电路空闲，特定的 `SRAM` 不工作。它可能没有充分利用可用资源的容量。
*   **读取（Read）**：这是在 `SRAM` 中请求数据的情况。它使 `SRAM` 的电路进入工作状态。
*   **写入（Write）**：当必须更新或覆盖异步 `SRAM` 中存储的数据时，这是 `SRAM` 中的一种状态。此状态也会使 `SRAM` 的电路进入工作状态。

## 异步 SRAM 的应用

*   广泛用于制造 `CPU` 缓存、硬盘缓冲区等，这在小型存储器应用中更为常见。
*   用于制造网络设备，异步静态随机存取存储器常用于网络设备的硬件实现。例如交换机、路由器等。
*   可编程逻辑控制器的设计采用异步静态随机存取存储器。
*   用于多功能打印机。

## 异步静态随机存取存储器的优势

*   由于异步静态随机存取存储器具有三种操作状态，其通信本质上变得非常有效。
*   异步随机存取存储器的快速执行功能使其成为网络架构中的首选。

## 异步 SRAM 的缺点

*   实施异步 `SRAM` 的成本较高，这使得其在我们系统中的嵌入成本更高。
*   异步双端口通常比同步双端口慢。
*   异步 `SRAM` 的带宽比同步 `SRAM` 少。
