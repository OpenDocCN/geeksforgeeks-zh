## 数据操作指令

数据操作指令对数据进行操作，为计算机提供计算能力。典型计算机中的数据操作指令通常分为以下三种基本类型。

1.  算术指令
2.  逻辑和位操作指令
3.  移位指令

我们一个一个来讨论。

### 算术指令

算术指令：算术指令对数据执行算术运算。基本的算术运算是加法、减法、乘法和除法。大多数计算机都提供所有这四种运算的指令。

**典型算术指令**

| 名字 | 助记符 | 例子 | 说明 |
| :--- | :--- | :--- | :--- |
| 增量 | `INC` | `INC B` | 它将寄存器 `B` 增加 `1`。 |
| 减量 | `DEC` | `DEC B` | 它会将寄存器 `B` 减 `1`。 |
| 加法 | `ADD` | `ADD B` | 它会将寄存器 `B` 的内容添加到累加器的内容中，并将结果存储在累加器中。 |
| 减法 | `SUB` | `SUB B` | 它将从累加器的内容中减去寄存器 `B` 的内容，并将结果存储在累加器中。 |
| 乘法 | `MUL` | `MUL B` | 它会将寄存器 `B` 的内容乘以累加器，并将结果存储在累加器中。 |
| 除法 | `DIV` | `DIV B` | 它会将累加器的内容除以寄存器 `B` 的内容，并将商存储在累加器中。 |
| 带进位加法 | `ADDC` | `ADDC B` | 它会将寄存器 `B` 的内容和进位标志与累加器的内容相加，并将结果存储在累加器中。 |
| 带借位减法 | `SUBB` | `SUBB B` | 它将从累加器的内容中减去寄存器 `B` 的内容和进位标志，并将结果存储在累加器中。 |
| 求反（2的补码） | `NEG` | `NEG B` | 它将通过寻找其单个操作数的2的补码来否定一个值。这意味着简单地将操作数乘以 `-1`。 |

### 逻辑和位操作指令

逻辑和位操作指令：逻辑指令对存储在寄存器中的位串执行二进制操作。它们对于操作单个位或一组位很有用。

**典型逻辑和位操作指令**

| 名字 | 助记符 | 例子 | 说明 |
| :--- | :--- | :--- | :--- |
| 清除 | `CLR` | `CLR A` | 它会将累加器设置为 `0`。 |
| 求补 | `COM` | `COM A` | 它将对累加器的内容求补。 |
| 与 | `AND` | `AND B` | 它将寄存器 `B` 的内容与累加器的内容进行逻辑与运算，并将结果存储在累加器中。 |
| 或 | `OR` | `OR B` | 它将寄存器 `B` 的内容与累加器的内容进行逻辑或运算，并将结果存储在累加器中。 |
| 异或 | `XOR` | `XOR B` | 它会将寄存器 `B` 的内容与累加器的内容进行异或运算，并将其储存在累加器中。 |
| 清除进位 | `CLC` | `CLC` | 它会将进位标志设置为 `0`。 |
| 设定进位 | `SETC` | `SETC` | 它会将进位标志设置为 `1`。 |
| 补码进位 | `CMC` | `CMC` | 它将对进位标志求补。 |
| 启用中断 | `EI` | `EI` | 它将启用中断。 |
| 禁止中断 | `DI` | `DI` | 它将禁用中断。 |

### 移位指令

移位指令：移位操作是将一个字的位向左或向右移动的操作。移位指令可以指定逻辑移位、算术移位或循环类型的操作。

**典型移位指令**

| 名字 | 助记符 |
| :--- | :--- |
| 逻辑右移 | `SHRL` |
| 逻辑左移 | `SHLL` |
| 算术右移 | `SHRA` |
| 算术左移 | `SHLA` |
| 右循环 | `ROR` |
| 左循环 | `ROL` |
| 通过进位右循环 | `RORC` |
| 通过进位左循环 | `ROLC` |

关于移位指令，请参考本[移位指令参考](https://www.geeksforgeeks.org/shift-micro-operations-in-computer-architecture/)。
