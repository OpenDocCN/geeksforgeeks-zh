# Arduino与树莓派的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-arduino-and-raspberry-pi/](https://www.geeksforgeeks.org/difference-between-arduino-and-raspberry-pi/)

我们可以在硬件项目中使用各种各样的控制器板。其中最受欢迎的两个是：`Arduino`和`树莓派`。

[`Arduino`](https://www.geeksforgeeks.org/what-is-arduino/)基于`ATmega`家族，设计和软件结构相对简单。树莓派，基本上是单板电脑。它们都有一个执行指令、定时器、内存和输入输出引脚的中央处理器。两者之间的主要区别在于，`Arduino`往往具有强大的`I/O`能力，可以直接驱动外部硬件。而树莓派的输入输出很弱，需要晶体管来驱动硬件。

让我们看看`Arduino`和树莓派的区别：

| 序列号 | Arduino | 树莓派 |
| :--- | :--- | :--- |
| 1. | `Arduino`的控制单元来自`Atmega`家族。 | 而树莓派的控制单元来自`ARM`家族。 |
| 2. | `Arduino`基于微控制器。 | 而树莓派是基于微处理器的。 |
| 3. | 它被设计用来控制系统中连接到电路板的电气元件。 | 而树莓派计算数据并产生有价值的输出，并根据其计算结果控制系统中的组件。 |
| 4. | `Arduino`板具有简单的硬件和软件结构。 | 而树莓派板具有复杂的硬件和软件架构。 |
| 5. | `CPU`架构：8位。 | `CPU`架构：64位。 |
| 6. | 它使用非常少的内存，2 kB。 | 而树莓派需要更多的内存，1 GB。 |
| 7. | 它的时钟处理速度为16兆赫。 | 而树莓派时钟的处理速度为1.4千兆赫。 |
| 8. | 它在成本上更便宜。 | 而树莓派贵。 |
| 9. | 它具有更高的输入/输出电流驱动强度。 | 而树莓派的输入输出电流驱动强度较低。 |
| 10. | 它消耗大约200兆瓦的电力。 | 而它消耗大约700兆瓦的功率。 |
