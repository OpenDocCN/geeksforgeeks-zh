# 功能完备操作

> 原文：[https://www.geeksforgeeks.org/functionally-complete-operations/](https://www.geeksforgeeks.org/functionally-complete-operations/)

先决条件 – [功能完备性](https://www.geeksforgeeks.org/digital-logic-functionality-completeness/)

开关函数由二进制变量、逻辑运算符号以及常数 `0` 和 `1` 表示。当每个开关函数都可以用其中的操作来表示时，那么只有一组操作被称为功能完备的。

1.  集合（`AND`、`OR`、`NOT`）是一个功能完备的集合。
2.  集合（`AND`，`NOT`）被认为是功能完备的。
3.  集合（`OR`，`NOT`）也被认为是功能完备的。

这里，集合（`AND`，`NOT`）被认为是功能完备的，因为 `OR` 可以使用 `AND` 和 `NOT` 操作导出。

**例：**

```
(X + Y) = (X'.Y')'
X'= X 的补数。
Y'= Y 的补数。
```

集合（`OR`，`NOT`）在功能上是完备的，因为 `AND` 可以用 `OR` 和 `NOT` 运算导出。

**例：**

```
(X.Y) = (X' + Y')'
```

**注：**
一个函数可以是完全功能完备的，也可以是部分功能完备的，或者，根本不是功能完备的。

## Example-1

如果一个函数 `f(X, Y, Z)= (X’ + YZ’)`，那么检查它是否功能完备？

在函数中令 `Z = Y`，因此：

```
f(X, Y, Y)= (X' + YY')
          = (X' + 0) 因为，Y.Y'=0
          = X' （这是补数，即 NOT）
```

再次，在函数中令 `X= X'` 和 `Z= Y'`，因此：

```
f(X', Y, Y')= (X')'+ Y(Y')'
           = (X + Y.Y) 因为，(X')'= X 且 (Y')'= Y
           = (X + Y) 因为，Y.Y= Y （这是 OR 运算符）
```

因此，您可以从上面的函数中导出 `NOT` 和 `OR` 运算符，因此该函数在功能上是完全完备的。

## Example-2

如果一个函数 `f(X, Y)= (X’Y)`，那么检查它是否功能完备？

令 `X= (X’)`，因此：

```
f(X', Y)= (X')'.Y = X.Y （这是 AND 运算符）
```

这里，`AND` 运算符是派生出来的，现在你需要派生 `NOT` 运算符来使它在功能上完备。

如果你令 `Y= 1`，

```
那么 f(X, 1)= (X')，这是 NOT 运算符。
```

因此，这个函数在功能上是部分完备的，因为您需要借助常数 `1` 导出 `NOT` 运算符。

**注：** 每当你借助常数（`1` 和 `0`）使一个函数功能完备时，这个函数就被称为部分完备函数。

## Example-3

如果一个函数 `f(X, Y, Z)= (XY + YZ + ZX)`，那么检查这个函数是否功能完备？

为了使一个函数功能完备，导出 `NOT` 运算符是必要的，但这里，函数中没有补数，因此不可能导出 `NOT` 运算符。

因此，该功能在功能上根本不完备。
