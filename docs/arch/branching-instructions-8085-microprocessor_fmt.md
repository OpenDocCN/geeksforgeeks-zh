# 8085 微处理器中的分支指令

> 原文：[https://www.geeksforgeeks.org/branching-instructions-8085-microprocessor/](https://www.geeksforgeeks.org/branching-instructions-8085-microprocessor/)

分支指令是指由于执行分支指令而将执行切换到不同指令序列的行为。

分支指令的三种类型是：

1.  跳转（无条件和有条件）
2.  调用（无条件和有条件）
3.  返回（无条件和有条件）

## 1. 跳转指令

跳转指令根据指定的标志将程序序列转移到操作数中给定的内存地址。跳转指令有两种类型：无条件跳转指令和条件跳转指令。

### (a) 无条件跳转指令

将程序序列转移到所述存储器地址。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `JMP` | `地址` | 跳到地址 | `JMP 2050` |

### (b) 条件转移指令

仅当满足条件时，将程序序列转移到所描述的存储器地址。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `JC` | `地址` | 如果进位标志为 1，则跳转到该地址 | `JC 2050` |
| `JNC` | `地址` | 如果进位标志为 0，则跳转到该地址 | `JNC 2050` |
| `JZ` | `地址` | 如果零标志为 1，则跳转到该地址 | `JZ 2050` |
| `JNZ` | `地址` | 如果零标志为 0，则跳转到该地址 | `JNZ 2050` |
| `JPE` | `地址` | 如果奇偶校验标志为 1，跳转到该地址 | `JPE 2050` |
| `JPO` | `地址` | 如果奇偶校验标志为 0，跳转到该地址 | `JPO 2050` |
| `JM` | `地址` | 如果符号标志为 1，跳转到地址 | `JM 2050` |
| `JP` | `地址` | 如果符号标志为 0，跳转到地址 | `JP 2050` |

## 2. 调用指令

调用指令将程序序列转移到操作数中给定的内存地址。在传送之前，`CALL` 之后的下一条指令的地址被推到堆栈上。调用指令有两种类型：无条件调用指令和条件调用指令。

### (a) 无条件调用指令

它将程序序列转移到操作数中给定的内存地址。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `CALL` | `地址` | 无条件地调用 | `CALL 2050` |

### (b) 条件调用指令

只有满足条件，指令才执行。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `CC` | `地址` | 如果进位标志为 1，则调用 | `CC 2050` |
| `CNC` | `地址` | 如果进位标志为 0，则调用 | `CNC 2050` |
| `CZ` | `地址` | 如果零标志为 1，则调用 | `CZ 2050` |
| `CNZ` | `地址` | 如果零标志为 0，则调用 | `CNZ 2050` |
| `CPE` | `地址` | 如果奇偶校验标志为 1，则调用 | `CPE 2050` |
| `CPO` | `地址` | 如果奇偶校验标志为 0，则调用 | `CPO 2050` |
| `CM` | `地址` | 如果符号标志为 1，则调用 | `CM 2050` |
| `CP` | `地址` | 如果符号标志为 0，则调用 | `CP 2050` |

## 3. 返回指令

返回指令将程序序列从子程序转移到调用程序。返回指令有两种类型：无条件返回指令和条件返回指令。

### (a) 无条件返回指令

程序序列从子程序无条件转移到调用程序。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `RET` | 无 | 无条件地从子程序返回 | `RET` |

### (b) 条件返回指令

只有满足条件时，程序序列才会从子程序转移到调用程序。

| 操作码 | 操作数 | 说明 | 例子 |
| :--- | :--- | :--- | :--- |
| `RC` | 无 | 如果进位标志为 1，则从子程序返回 | `RC` |
| `RNC` | 无 | 如果进位标志为 0，则从子程序返回 | `RNC` |
| `RZ` | 无 | 如果零标志为 1，则从子程序返回 | `RZ` |
| `RNZ` | 无 | 如果零标志为 0，则从子程序返回 | `RNZ` |
| `RPE` | 无 | 如果奇偶校验标志为 1，则从子程序返回 | `RPE` |
| `RPO` | 无 | 如果奇偶校验标志为 0，则从子程序返回 | `RPO` |
| `RM` | 无 | 如果符号标志为 1，则从子程序返回 | `RM` |
| `RP` | 无 | 如果符号标志为 0，则从子程序返回 | `RP` |
