# 8086 微处理器中程序执行转移指令

> 原文: [https://www.geeksforgeeks.org/program-execution-transfer-instructions-8086-microprocessor/](https://www.geeksforgeeks.org/program-execution-transfer-instructions-8086-microprocessor/)

## 先决条件
[8085 微处理器中的分支指令](https://www.geeksforgeeks.org/branching-instructions-8085-microprocessor/)

## 概述
程序执行转移指令类似于分支指令，是指由于执行分支指令而将执行切换到不同指令序列的行为。

程序执行转移指令的两种类型是:
1.  无条件的
2.  有条件的

## 无条件程序执行转移指令
这些指令始终执行。

| 操作码 | 操作数 | 说明 | 例子 |
| --- | --- | --- | --- |
| `CALL` | 地址 | 调用子程序并将返回地址保存在堆栈上 | `CALL 2050` |
| `RET` | 没有人 | 从子程序返回主程序 | `RET` |
| `JUMP` | 地址 | 将执行控制转移到指定的地址 | `JUMP 2050` |
| `LOOP` | 地址 | 循环执行一系列指令，直到 `CX=0` | `LOOP 2050` |

这里可以直接或间接指定地址。

## 条件程序执行转移指令
这些指令仅在指定条件为真时执行。

| 操作码 | 操作数 | 说明 | 例子 |
| --- | --- | --- | --- |
| `JC` | 地址 | `CF = 1` 时跳转 | `JC 2050` |
| `JNC` | 地址 | 如果 `CF = 0`，则跳转 | `JNC 2050` |
| `JZ` | 地址 | 如果 `ZF = 1`，则跳转 | `JZ 2050` |
| `JNZ` | 地址 | 如果 `ZF = 0`，则跳转 | `JNZ 2050` |
| `JO` | 地址 | 如果 `OF = 1`，则跳转 | `JO 2050` |
| `JNO` | 地址 | 如果 `OF = 0`，跳转 | `JNO 2050` |
| `JP` | 地址 | 如果 `PF = 1`，则跳转 | `JP 2050` |
| `JNP` | 地址 | 如果 `PF = 0`，则跳转 | `JNP 2050` |
| `JPE` | 地址 | 如果 `PF = 1`，则跳转 | `JPE 2050` |
| `JPO` | 地址 | 如果 `PF = 0`，则跳转 | `JPO 2050` |
| `JS` | 地址 | 如果 `SF = 1`，则跳转 | `JS 2050` |
| `JNS` | 地址 | 如果 `SF = 0`，则跳转 | `JNS 2050` |
| `JA` | 地址 | 如果 `CF=0` 且 `ZF=0`，则跳转 | `JA 2050` |
| `JNBE` | 地址 | 如果 `CF=0` 且 `ZF=0`，则跳转 | `JNBE 2050` |
| `JAE` | 地址 | 如果 `CF=0`，则跳转 | `JAE 2050` |
| `JNB` | 地址 | 如果 `CF=0`，则跳转 | `JNB 2050` |
| `JBE` | 地址 | 如果 `CF = 1` 或 `ZF = 1`，则跳转 | `JBE 2050` |
| `JNA` | 地址 | 如果 `CF = 1` 或 `ZF = 1`，则跳转 | `JNA 2050` |
| `JE` | 地址 | 如果 `ZF = 1`，则跳转 | `JE 2050` |
| `JG` | 地址 | 如果 `ZF = 0` 且 `SF = OF`，则跳转 | `JG 2050` |
| `JNLE` | 地址 | 如果 `ZF = 0` 且 `SF = OF`，则跳转 | `JNLE 2050` |
| `JGE` | 地址 | 如果 `SF = OF`，则跳转 | `JGE 2050` |
| `JNL` | 地址 | 如果 `SF = OF`，则跳转 | `JNL 2050` |
| `JL` | 地址 | 跳如果 `SF != OF` | `JL 2050` |
| `JNGE` | 地址 | 跳如果 `SF != OF` | `JNGE 2050` |
| `JLE` | 地址 | 如果 `ZF = 1` 或 `SF != OF`，跳 | `JLE 2050` |
| `JNG` | 地址 | 如果 `ZF = 1` 或 `SF != OF`，跳 | `JNG 2050` |
| `JCXZ` | 地址 | 如果 `CX = 0`，则跳转 | `JCXZ 2050` |
| `LOOP` | 地址 | 当 `ZF = 1` 和 `CX = 0` 时循环 | `LOOP 2050` |
| `LOOPE` | 地址 | 当 `ZF = 1` 和 `CX = 0` 时循环 | `LOOPE 2050` |
| `LOOPNE` | 地址 | 当 `ZF = 0` 和 `CX = 0` 时循环 | `LOOPNE 2050` |
| `LOOPNZ` | 地址 | 当 `ZF = 0` 和 `CX = 0` 时循环 | `LOOPNZ 2050` |

这里可以直接或间接指定地址。

## 标志寄存器说明
- `CF` 是进位标志
- `ZF` 是零标志
- `OF` 是溢出标志
- `PF` 是奇偶标志
- `SF` 是符号标志
- `CX` 是寄存器
