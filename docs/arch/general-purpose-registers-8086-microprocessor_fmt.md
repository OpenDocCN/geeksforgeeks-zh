# 8086 微处理器中的通用寄存器

> 原文：[https://www.geeksforgeeks.org/general-purpose-registers-8086-microprocessor/](https://www.geeksforgeeks.org/general-purpose-registers-8086-microprocessor/)

通用寄存器用于在微处理器中存储临时数据。8086 微处理器中有 8 个通用寄存器。

![General purpose registers](img/ffff213b961c7f8600d2edbb6a19a1fb.png)
*图 – 通用寄存器*

## 1. `AX` – 累加器
这是累加器。它是 16 位的，并被划分为两个 8 位寄存器 `AH` 和 `AL`，以便也能执行 8 位指令。
它通常用于算术和逻辑指令，但在 8086 微处理器中，累加器并非必须是目的操作数。

示例：
```
ADD AX, AX (AX = AX + AX)
```

## 2. `BX` – 基址寄存器
这是基址寄存器。它是 16 位的，并被划分为两个 8 位寄存器 `BH` 和 `BL`，以便也能执行 8 位指令。
它用于存储偏移量的值。

示例：
```
MOV BL, [500] (BL = 500H)
```

## 3. `CX` – 计数寄存器
这是计数寄存器。它是 16 位的，并被划分为两个 8 位寄存器 `CH` 和 `CL`，以便也能执行 8 位指令。
它用于循环和移位操作。

示例：
```
MOV CX, 0005
LOOP
```

## 4. `DX` – 数据寄存器
这是数据寄存器。它是 16 位的，并被划分为两个 8 位寄存器 `DH` 和 `DL`，以便也能执行 8 位指令。
它用于乘法和输入/输出端口寻址。

示例：
```
MUL BX (DX, AX = AX * BX)
```

## 5. `SP` – 堆栈指针
这是堆栈指针。它是 16 位的。
它指向堆栈的最顶层。如果堆栈是空的，堆栈指针将是 `(FFFE)H`。它是相对于堆栈段的偏移地址。

## 6. `BP` – 基址指针
这是基址指针。它是 16 位的。
主要用于访问堆栈传递的参数。它是相对于堆栈段的偏移地址。

## 7. `SI` – 源变址寄存器
这是源索引寄存器。它是 16 位的。
用于数据的指针寻址和一些字符串相关操作的源。它的偏移量是相对于数据段的。

## 8. `DI` – 目的变址寄存器
这是目标索引寄存器。它是 16 位的。
用于数据的指针寻址和一些字符串相关操作的目的地。它的偏移量是相对于附加段的。
