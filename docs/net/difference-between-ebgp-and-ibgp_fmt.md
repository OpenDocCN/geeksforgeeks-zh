# EBGP 和 IBGP 的区别

> 原文: [https://www.geeksforgeeks.org/difference-between-ebgp-and-ibgp/](https://www.geeksforgeeks.org/difference-between-ebgp-and-ibgp/)

## 外部边界网关协议 (EBGP)

`EBGP` 在自治系统之间使用。它在为两个或多个自治系统提供互连的边缘或边界路由器上使用和实现。它的功能是负责不同组织或互联网的网络互连的协议。

## 内部边界网关协议 (IBGP)

`IBGP` 用于自治系统内部。它用于向您的内部路由器提供信息。它要求同一自治系统中的所有设备形成全网状拓扑，或者路由反射器和联盟来进行前缀学习。

## EBGP 和 IBGP 的区别

| SR.NO | EBGP | IBGP |
| :--- | :--- | :--- |
| 1 | `EBGP` 代表外部边界网关协议。 | `IBGP` 代表内部边界网关协议。 |
| 2 | 它运行在不同自治系统的两台 `BGP` 路由器之间。 | 它运行在同一自治系统中的两台 `BGP` 路由器之间。 |
| 3 | 它的默认管理距离是 `20`。 | 它的默认管理距离是 `200`。 |
| 4 | 从 `EBGP` 对等点收到的 `EBGP` 路由可以通告给 `EBGP` 和 `IBGP` 对等点。 | 从一个 `IBGP` 对等体接收的 `IBGP` 路由不能通告给另一个 `IBGP` 对等体，但是可以通告给 `EBGP` 对等体。 |
| 5 | 它不需要全网状拓扑。 | 它需要全网状拓扑。 |
| 6 | 它用于组织之间或组织与互联网服务提供商之间。 | 它在同一组织内使用。 |
| 7 | 它用作防止环路的路径。 | 它使用 `BGP` 水平分割来防止环路。 |
| 8 | 它的默认对等体设置为 `TTL = 1`。 | 它的默认对等体设置为 `TTL = 255`。 |
| 9 | 在 `EBGP` 同行中，不发送本地偏好等属性。 | 在 `IBGP` 对等体中，像本地偏好这样的属性被发送。 |
| 10 | 当向 `EBGP` 对等方通告路由时，下一跳将更改为本地路由器。 | 当路由通告给 `IBGP` 对等体时，下一跳保持不变。 |
