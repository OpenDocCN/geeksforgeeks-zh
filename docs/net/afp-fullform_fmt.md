# 法新社全文

> 原文：`https://www.geeksforgeeks.org/afp-fullform/`

法新社代表**苹果备案协议**。这是一种用于在服务器和客户端之间共享文件的苹果操作系统网络协议。它允许用户访问属于外部系统的文件。它是应用层和会话层协议。

## 功能:

*   支持`Unicode`文件名。
*   提供可移植操作系统接口（`POSIX`）。
*   还提供[访问控制列表（`ACL`）](https://www.geeksforgeeks.org/access-lists-acl/)权限，指定哪些系统进程和用户可以访问对象并执行必要的操作。
*   使用资源分支为结构化数据提供存储设施，使用数据分支为非结构化数据提供存储设施。
*   支持[`TCP/IP`](https://www.geeksforgeeks.org/tcp-ip-model/)和`AppleTalk`进行通信并提供各种服务。
*   在`AFP`协议中，可以执行诸如创建目录、关闭目录、复制文件、删除文件和关闭卷等命令。

## 优势:

*   通过使用高级文件锁定机制，为系统提供安全功能，并限制用户访问危险文件。
*   还包括命名的扩展属性，用户可以用它来将未解释的计算机文件与元数据进行映射。
*   除了本地文件访问外，还通过建立远程文件服务器连接来支持服务器文件访问。
*   与`Mac`文件系统（`HFS+`）提供100%兼容性，并且也是原生文件共享协议。
*   提供内置功能，如`spotlight`搜索、`time machine`、`Mac`别名和`Bonjour`服务。

## 缺点:

*   与使用`Apple`文件系统（`APFS`）格式化的存储设备不兼容。
*   其顺序读写速度低于服务器消息块（`SMB`）协议。
