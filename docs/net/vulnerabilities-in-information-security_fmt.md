# 信息安全漏洞

> 原文：[https://www.geeksforgeeks.org/vulnerabilities-in-information-security/](https://www.geeksforgeeks.org/vulnerabilities-in-information-security/)

**漏洞**是系统中的弱点，给威胁提供了危害资产的机会。所有系统都有漏洞。尽管技术在进步，但漏洞的数量却在增加，例如数千万行代码、许多开发人员、人为弱点等。漏洞主要是由硬件、软件、网络和程序漏洞造成的。

## 1. 硬件漏洞
硬件漏洞是一种弱点，可用于通过物理或远程方式攻击系统硬件。例如：
- 系统或设备的旧版本
- 非保护存储区
- 未加密的设备等。

## 2. 软件漏洞
在开发或配置过程中发生的软件错误，如其执行可能违反安全策略。例如：
- 缺乏输入验证
- 未经验证的上传
- `跨站点脚本`
- 未加密的数据等。

## 3. 网络漏洞
网络中出现的一个弱点，可能是硬件，也可能是软件。例如：
- 无保护通信
- 恶意软件或恶意软件（例如：`病毒`、`键盘记录程序`、`蠕虫`等）
- 社会工程攻击
- 配置错误的防火墙

## 4. 程序漏洞
组织运作方法中出现的一个弱点。例如：
- 密码程序——密码应遵循标准`密码策略`。
- 培训程序——员工必须知道应该采取哪些措施以及如何处理安全问题。决不能要求员工在线提供用户凭据。让员工了解`社会工程`和`网络钓鱼`威胁。
