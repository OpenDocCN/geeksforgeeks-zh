# Amazon Drive and RapidShare

> 原文: [https://www.geeksforgeeks.org/difference-between-amazon-drive-and-rapidshare/](https://www.geeksforgeeks.org/difference-between-amazon-drive-and-rapidshare/)

## Amazon Drive
`Amazon Drive` is a file hosting service and cloud storage provided by `Amazon`. It allows for cloud storage, file hosting, file sharing, and file backup. Files and folders on this drive can be streamed from a variety of devices such as desktop, mobile, and tablet. `Amazon Drive` was initially known as `Amazon Cloud Drive`. It is not available worldwide. It is available in a select few countries. It was launched in `2011`.

## RapidShare
`RapidShare` is a file hosting service, cloud storage service, and online backup service provided by `RapidShare AG`. It has the ability to handle a large number of users simultaneously. It is available in multiple languages such as English, French, German, and Spanish. It was developed by `Christian Schmid`. It was initially named `ezShare`. It was one of the earliest cloud storage services. It was launched in `2002`.

## Differences Between Amazon Drive and RapidShare

<center>

| Amazon Drive | RapidShare |
| --- | --- |
| Owned by `Amazon`. | It is owned by `RapidShare AG`. |
| It was launched in `2011`. | It was launched in `2002`. |
| Developed by `Amazon`. | It was developed by `Christian Schmid`. |
| It offers `5 GB` of free storage and up to `30 TB` of maximum paid storage. | It offered limited storage space for both free and paid accounts. |
| Files can be downloaded an unlimited number of times. | Files could only be downloaded `10` times with a free account. |
| Used in all countries except `Mexico` and `China`. | It was used worldwide. |
| It is still in use. | The service was suspended in `2015`. |
| It is specifically used to store photos. | It was used to store all types of files. |
| Maximum file size is `2 GB` via web upload and `50 GB` via the Drive app. | The maximum file size was `500 MB` per file. |
| It is a file hosting and cloud storage service. | It was a file hosting and online backup service. |
| Links do not expire after a few days of inactivity. | `RapidShare` links expired after `60` days of inactivity. |
| It has `Amazon S3` traffic or bandwidth limits. | It had variable traffic or bandwidth limits based on the paid plan. |
