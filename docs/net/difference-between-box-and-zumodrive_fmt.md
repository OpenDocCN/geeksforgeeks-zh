# Box 和 ZumoDrive 的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-box-and-zumodrive/](https://www.geeksforgeeks.org/difference-between-box-and-zumodrive/)

## Box
`Box` 是由 `Box Incorporation` 提供的云存储和文件托管服务。它是由亚伦·列维和迪伦·史密斯开发的。它基本上是面向企业的云内容管理和文件共享服务。它提供 `10 GB` 的存储空间。它由 `Box Incorporation` 于 `2005` 年推出。它适用于 `Windows`、`macOS` 等平台。

## ZumoDrive
`ZumoDrive` 是一个文件托管服务、文件同步服务和基于云的文件存储服务，最初由 `Zecter Incorporation` 提供，但后来归 `Motorola Mobility` 所有。由 `Zecter Incorporation` 开发，适用于 `Windows`、`macOS`、`iOS`、`webOS`、`Linux` 和 `Android`。它提供了 `2 GB` 的免费存储空间，付费时可扩展至 `500 GB`。它于 `2009` 年推出。

### 区别对比

| Box | ZumoDrive |
| --- | --- |
| 它属于 `Box Incorporation`。 | 最初由 `Zecter Incorporation` 拥有，但后来被 `Motorola Mobility` 拥有。 |
| 它于 `2005` 年推出。 | 它于 `2009` 年推出。 |
| 它提供 `10 GB` 的可用存储空间。 | 它提供了 `2 GB` 的可用存储空间。 |
| 它是由亚伦·列维和迪伦·史密斯开发的。 | 由 `Zecter Incorporation` 开发。 |
| 个人账户最大存储容量为 `100 GB`，企业账户不受限制。 | 提供的最大存储大小为 `500 GB`。 |
| 它不使用增量编码技术。 | 它使用了增量编码技术。 |
| 它仍在使用中。 | 这项服务于 `2012` 年暂停。 |
| 最大文件大小免费为 `250 MB`，付费为 `5 GB`。 | 最大文件在这里没有限制。 |
| 它不提供智能预测缓存。 | 它提供了智能预测缓存。 |
| 它支持 `Windows`、`OS X`、`Android`、`BlackBerry` 和 `iOS`。 | 它支持 `Windows`、`OS X`、`Linux`、`Android`、`iPhone` 和 `webOS`。 |
| 它有有限的免费软件版本。 | 它有免费软件版本。 |
