# AES 和 DES 密码的区别

> 原文: [https://www.geeksforgeeks.org/difference-between-aes-and-des-ciphers/](https://www.geeksforgeeks.org/difference-between-aes-and-des-ciphers/)

`AES` 和 `DES` 都是对称分组密码的例子，但有一定的不同。

## 主要区别

| 特性 | AES | DES |
| :--- | :--- | :--- |
| 全称 | `AES` 代表高级加密标准 | `DES` 代表数据加密标准 |
| 创作日期 | 创作日期是 1999 年。 | 创作日期是 1976 年。 |
| 设计 | 面向字节的。 | 面向比特。 |
| 密钥长度 | 密钥长度可以是 128 位、192 位和 256 位。 | `DES` 中密钥长度为 56 位。 |
| 回合数 | 回合数取决于密钥长度:10(128 位)、12(192 位)或 14(256 位) | `DES` 涉及 16 轮相同的操作 |
| 结构 | 该结构基于置换网络。 | 该结构基于费斯特网络。 |
| 设计原理 | `AES` 的设计原理是开放的。 | `DES` 的设计原理是封闭的。 |
| 选择过程 | 这个项目的选择过程是秘密的，但接受公开的公众意见。 | 这个的选择过程是秘密的。 |
| 安全性 | `AES` 比 `DES` 密码更安全，是事实上的世界标准。 | `DES` 很容易被破解，因为它有已知的漏洞。`3DES`(三重 `DES`)是 `DES` 的变体，比通常的 `DES` 更安全。 |
| 轮次操作 | `AES` 中的轮次为:字节替换、移位行、混合列和键添加 | `DES` 中的轮次有:展开、轮键异或运算、替换和置换 |
| 明文块大小 | `AES` 可以加密 128 位明文。 | `DES` 可以加密 64 位明文。 |
| 密码来源 | `AES` 密码是从旁道方形密码中推导出来的。 | `DES` 密码源于路西法密码。 |
| 设计者 | `AES` 是由文森特·里曼和琼·代蒙设计的。 | `DES` 是由 `IBM` 设计的。 |
| 已知攻击 | 没有已知的针对 `AES` 的加密分析攻击，但是针对 `AES` 实现的侧信道攻击是可能的。比暴力攻击更复杂，但仍然无效。 | 针对 `DES` 的已知攻击包括暴力破解、线性加密分析和差分加密分析。 |

## 总结

`AES` 比 `DES` 密码更安全，是事实上的世界标准。`DES` 很容易被破解，因为它有已知的漏洞。`3DES`(三重 `DES`)是 `DES` 的变体，比通常的 `DES` 更安全。
