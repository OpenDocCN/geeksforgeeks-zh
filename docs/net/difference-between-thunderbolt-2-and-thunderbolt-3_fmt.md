# Thunderbolt 2 和 Thunderbolt 3 的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-thunderbolt-2-and-thunderbolt-3/](https://www.geeksforgeeks.org/difference-between-thunderbolt-2-and-thunderbolt-3/)

## Thunderbolt 2

Thunderbolt 2 发布于 2013 年，是 Thunderbolt 标准的第二次迭代。像 Thunderbolt 一样，它基于苹果公司开发的 `Mini DisplayPort` 连接器，这意味着它可以兼容任何 `USB` 端口，并且它的电缆也不适合 `Thunderbolt 3` 端口。`Thunderbolt 2` 的最大吞吐量为 `20 Gbps`，是 `Thunderbolt` 吞吐量的两倍，是 `Thunderbolt 3` 吞吐量的一半。

## Thunderbolt 3

`Thunderbolt 3` 于 2015 年发布。这是 Thunderbolt 最新最快的版本，最大吞吐量可达 `40 Gbps`。与 `Thunderbolt 2` 不同，在 `Thunderbolt 3` 中使用的是 `USB Type-C` 连接器，向后兼容 `USB 3.1`。它还为这些设备提供 `100` 瓦的功率。

我们可以区分 `Thunderbolt 2` 和 `Thunderbolt 3` 端口。如果端口形状像 `Mini DisplayPort`，那么肯定是 `Thunderbolt 1` 或 `2`。如果端口形状像 U 盘，那就是 `Thunderbolt 3` 端口。

## Thunderbolt 2 和 Thunderbolt 3 的区别

| S.NO | Thunderbolt 2 | Thunderbolt 3 |
| :--- | :--- | :--- |
| 1. | 在 `Thunderbolt 2` 中，使用了苹果的 `Mini DisplayPort` 连接器。 | 而在 `Thunderbolt 3` 中，使用的是 `Type-C` 连接器。 |
| 2. | 它提供 `20 Gbps` 的最大吞吐量。 | 而它可以提供 `40 Gbps`，是 `Thunderbolt 2` 吞吐量的两倍。 |
| 3. | 它提供两个双向 `10 Gbps` 通道。 | 而在 `Thunderbolt 3` 中，提供两个双向 `20 Gbps` 通道。 |
| 4. | `Thunderbolt 2` 与 `USB-C` 不向后兼容。 | 而 `Thunderbolt 3` 与 `USB-C` 向后兼容。 |
| 5. | `Thunderbolt 2` 与 `USB 3.1` 不向后兼容。 | 而 `Thunderbolt 3` 与 `USB 3.1` 向后兼容。 |
| 6. | `Thunderbolt 2` 比 `Thunderbolt 3` 慢。 | 而 `Thunderbolt 3` 比 `Thunderbolt 2` 快。 |
| 7. | 与 `Thunderbolt 3` 端口相比，`Thunderbolt 2` 端口为任何连接的设备提供非常低的功率。 | 而 `Thunderbolt 3` 端口可以为任何连接的设备提供最大 `100` 瓦的功率。 |
| 8. | `Thunderbolt 2` 端口不支持 `USB 3.1 Gen 2` 和 `Thunderbolt 3`。 | 而 `Thunderbolt 3` 端口支持 `USB 3.1 Gen 2` 以及 `Thunderbolt 3`。 |
| 9. | `Thunderbolt 2` 可以支持每个端口一个 `4K` 显示器。 | 而它可以支持每个端口一个 `120Hz` 的 `4K` 监视器或两个 `60Hz` 的监视器。 |
| 10. | 在 `Thunderbolt 2` 协议中，使用 `4x PCI Express 2.0` 和 `DisplayPort 1.2`。 | 而在 `Thunderbolt 3` 中，使用了协议 `4x PCI Express 3.0`、`HDMI 2.0`、`USB 3.1 Gen 2` 和 `DisplayPort 1.2`。 |
