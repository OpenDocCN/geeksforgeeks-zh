# 如何解决 RSA 算法问题？

> 原文: [https://www.geeksforgeeks.org/how-to-solve-rsa-algorithm-problems/](https://www.geeksforgeeks.org/how-to-solve-rsa-algorithm-problems/)

[RSA 算法](https://www.geeksforgeeks.org/rsa-algorithm-cryptography/)是一种非对称密码算法，这意味着在通信时应该有两个密钥参与，即公钥和私钥。有一些简单的步骤可以解决 RSA 算法中的问题。

## 示例-1

**第一步：选择两个质数** `p` 和 `q`
让我们取 `p = 3` 和 `q = 11`。

**第二步：计算** `n` 和 `φ` 的值，给出如下：

```
n = p * q
φ = (p-1) * (q-1)
```

这里的例子中，
`n = 3 * 11 = 33`
`φ = (3-1) * (11-1) = 2 * 10 = 20`

**第三步：找到** `e` **（公钥）的值**
选择 `e`，使得 `e` 应该是互质的。互质意味着它不应乘以 `φ` 的因子，也不应除以 `φ`。

`φ` 的因子是，`20 = 5 * 4 = 5 * 2 * 2`，所以 `e` 不要乘以 `5` 和 `2`，不要除以 20。

所以，质数是 3，7，11，17，19…，当取 3 和 11 时，选择 `e` 作为 7。

因此，`e = 7`。

**第四步：计算** `d` **（私钥）的值**
条件如下：
`gcd(φ, e) = φ * x + e * y = 1`，其中 `y` 是 `d` 的值。

要计算 `d` 的值：

1.  形成一个有四列的表格，即 `a`、`b`、`d` 和 `k`。
2.  初始化 `a = 1`，`b = 0`，`d = φ`，`k = -`（第一行）。
3.  初始化第二行的 `a = 0`，`b = 1`，`d = e`，`k = φ / e`。
4.  从下一行，应用以下公式计算下一个 `a`、`b`、`d` 和 `k` 的值，如下所示：
    *   `a_i = a_{i-2} - (a_{i-1} * k_{i-1})`
    *   `b_i = b_{i-2} - (b_{i-1} * k_{i-1})`
    *   `d_i = d_{i-2} - (d_{i-1} * k_{i-1})`
    *   `k_i = d_{i-1} / d_i`

一旦 `d = 1`，停止该过程并检查以下情况：

```
if (b > 0) {
    d = b;
} else {
    d = b + φ;
}
```

对于给定的示例，该表将是：

| a | b | d | k |
| --- | --- | --- | --- |
| 1 | 0 | 20 | – |
| 0 | 1 | 7 | 2 |
| 1 | -2 | 6 | 1 |
| -1 | 3 | 1 | – |

如上表 `d = 1` 所示，停止该过程并检查为 `b`。
`∴ b = 3` 给出的条件。

要验证 `b` 是否正确，应满足上述条件，即
`gcd(φ, e) = φ * x + e * y = (20 * -1) + (7 * 3) = 1`。因此 `d` 是正确的。

**第五步：进行加密和解密**
加密如下：
`c = t^e mod n`
解密如下：
`t = c^d mod n`

举个例子，假设 `t = 2`，那么
加密就是 `c = 2^7 mod 33 = 29`。

解密是 `t = 29^3 mod 33 = 2`。

因此在最终结果中，`p = 3`、`q = 11`、`φ = 20`、`n = 33`、`e = 7` 和 `d = 3`。

## 示例-2：[GATE CS-2017(Set 1)](https://www.geeksforgeeks.org/gate-gate-cs-2017-set-1-question-44/)
在 RSA 密码系统中，特定的 A 使用两个素数 `p = 13` 和 `q = 17` 来生成她的公钥和私钥。如果 A 的公钥是 35。那么 A 的私钥是？

1.  `p = 13` 和 `q = 17`
2.  计算 `n = 13 * 17 = 221` 和 `φ = (13-1) * (17-1) = 12 * 16 = 192`
3.  `e = 35`（公钥）
4.  计算 `d`（私钥）

| a | b | d | k |
| --- | --- | --- | --- |
| 1 | 0 | 192 | – |
| 0 | 1 | 35 | 5 |
| 1 | -5 | 17 | 2 |
| -2 | 11 | 1 | – |

`∴ d = 11`（私钥）
