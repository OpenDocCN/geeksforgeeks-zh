# 机器学习即服务(MLaaS)

> 原文: [https://www.geeksforgeeks.org/machine-learning-as-a-service-mlaas/](https://www.geeksforgeeks.org/machine-learning-as-a-service-mlaas/)

先决条件 – [机器学习(ML)](https://www.geeksforgeeks.org/machine-learning/)

在这个科技的数字世界里，我们每天都在听机器学习和人工智能。像已经进入技术市场的[人工智能即服务](https://www.geeksforgeeks.org/what-is-artificial-intelligence-as-a-service-aiaas-in-the-tech-industry/)一样，机器学习即服务也出现在科技行业。最近，福布斯预测全球机器学习市场将在未来 4 年从 73 亿美元增长到 306 亿美元。为此，[数据科学家](https://www.geeksforgeeks.org/overview-of-data-science/) & ML 人员必须继续构建模型，以满足客户的动态业务需求。但是仅仅建立模型是远远不够的，他们还必须考虑到许多过程，如：

1.  Check new ideas
2.  Maintain these models.
3.  Improve sexual function

对于数据科学家来说，以一定的速度满足需求几乎是不可能的任务。在机器学习即服务的帮助下，他们可以轻松地处理这些问题。

## 机器学习即服务(MLaaS) :

机器学习即服务(MLaaS)是提供机器学习(ML)工具的一组服务，作为[云计算](https://www.geeksforgeeks.org/cloud-computing/)服务的组成部分。机器学习即服务中的服务提供商提供诸如[深度学习](https://www.geeksforgeeks.org/introduction-deep-learning/)、数据可视化、预测分析、识别等工具。它还可以在消除基础设施问题方面发挥作用，如模型培训 & 评估、预处理等。

## MLaaS 实际上是如何工作的？

*   `MLaaS` 是构建在云架构上的，该架构由容器和 `Kubernetes` 组成，并为功能即服务 (`FaaS`) 和软件即服务 (`SaaS`) 的解决方案奠定了基础。公司可以提供从一开始就微调的 `ML` 模型，而不是提供一整套工具。
*   其算法用于在数据中搜索模式，因为这些模型构建的数学模型有助于预测新数据。
*   该平台提供模式识别-概率推理，并根据公司的需求创建不同的方法来完成定制的工作流程。

## 我们能从 `MLaaS` 平台上期待什么

1.  **`Time saver`：** `MLAAS` 为数据科学家提供了一种快速启动 `ML` 的方法，无需一些烦人的软件安装过程。
2.  **`Ease of use`：** `MLAAS` 提出了提供商的数据中心，他们可以在那里直接进行实际计算，这在业务的每个阶段都非常方便。
3.  **`Access to ML tools`：** `MLAAS` 提供商提供诸如医疗保健、人脸识别和情感分析等 `API`。对于业务，他们还提供数据可视化和预测分析。
4.  **`Data management`：** 许多公司将其数据从需要有效组织的存储保存到云存储。`MLA` 在这里提供云存储，并提供在几次 `ML` 实验中管理这些数据的方法，例如数据管道。

## 使用 `MLaaS` 的好处：

*   借助 `MLaaS`，开发人员可以轻松访问预构建的模型 & 算法，这是耗时的，也就是说，现在他们可以专注于、检查和改进项目的不同部分。
*   同样，通过使用 `AI`，服务业务可以提高其向客户提供服务的潜力，与客户的定期和有效互动可以提供更准确的业务策略。
*   带来我们希望从 `MLaaS` 设置中获得的结果 & 关注收入的激增可以在业务中发挥作用，否则组建一个拥有关键技能和知识的开发人员和工程师团队将是有限且成本高昂的。
*   `MLaaS` 平台可以负责管理和存储中小企业的数据，这些企业在存储海量数据的能力方面严重缺乏。

## 市场上一些流行的 `MLaaS`：

1.  **`Amazon ML`：** `Amazon Web Services (AWS) ML` 提供高级自动化功能，包括 `CSV` 文件、从多个服务器加载数据、`Amazon redshift` 和 `Amazon RDS`。该服务可以通过一些排序过程确定有价值的数据处理方法。
2.  **`Azure ML`：** `Microsoft Azure Studios` 提供的 `ML` 适用于 `AI` 初学者和专业玩家。`MLaaS` 提供了一个简单的基于浏览器的环境，并提供了拖放机制而不是编码。它还提供了大量的算法和许多方法供开发人员使用。
3.  **`IBM Watson ML`：** `Watson` 机器学习是 `IBM` 的 `Bluemix` 提供的一项广泛服务，旨在满足数据科学家和开发人员的需求。该服务拥有自己的可视化模型工具，旨在快速识别它们并获得有价值的见解，以便他们能够在业务中更快地做出决策。
4.  **`Google Cloud Machine Learning Engine`：** 它是一种用户友好的方式，可以为任何大小的数据构建 `ML` 模型。它还为自然语言处理提供 `ML` 服务，不仅用于图像 & 视频识别，还用于语音 & 翻译。

## 何时不使用 `MLaaS`：

服务 `MLaaS` 有很多优点和应用，它总是试图通过每天提供更好的服务来改变我们的生活，让我们的生活更加轻松。尽管如此，组织仍需要在某些点上避免多层自动分析，即：

*   如果数据需要保护和内部部署，我们应该防止使用多层自动分析。
*   如果数据在未来需要高水平的优化，则可能不需要 `MLaaS`。
*   如果需要优化复杂算法的服务成本，那么我们可能会采用内部基础设施。
