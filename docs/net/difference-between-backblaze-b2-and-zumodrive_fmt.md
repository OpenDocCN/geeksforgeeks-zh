# Backblaze B2 和 ZumoDrive 的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-backblaze-b2-and-zumodrive/](https://www.geeksforgeeks.org/difference-between-backblaze-b2-and-zumodrive/)

## 1. Backblaze B2
`Backblaze B2` 是由 `Backblaze` 公司提供的云存储服务。它与 `Amazon S3` 和其他云存储服务非常相似。它是一种对象存储服务。它由 `Backblaze` 公司于 2015 年推出。它提供 10 GB 的可用存储空间。它还为高级帐户提供了无限的存储空间。它也被称为 `B2` 云存储。

## 2. ZumoDrive
`ZumoDrive` 是一个文件托管服务、文件同步服务和基于云的文件存储服务，最初由 `Zecter Incorporation` 提供，但后来归 `Motorola Mobility` 所有。由 `Zecter` 公司开发，适用于 `Windows`、`macOS`、`iOS`、`webOS`、`Linux` 和 `Android`。它提供了 2 GB 的免费存储空间，付费时可扩展至 500 GB。它于 2009 年推出。

## Backblaze B2 和 ZumoDrive 的区别

| S.No | B2 | ZumoDrive |
| :--- | :--- | :--- |
| 1. | 它由 `Backblaze Corporation` 所有。 | 最初由 `Zecter` 公司拥有，但后来被 `Motorola Mobility` 拥有。 |
| 2. | 它于 2015 年推出。 | 它于 2009 年推出。 |
| 3. | 它提供 10 GB 的可用存储空间。 | 它提供了 2 GB 的可用存储空间。 |
| 4. | 它是由 `Backblaze Inc.` 开发的。 | 由 `Zecter` 公司开发。 |
| 5. | 它有无限的最大存储空间。 | 提供的最大存储大小为 500 GB。 |
| 6. | 它不使用增量编码技术。 | 它使用了增量编码技术。 |
| 7. | 它仍在使用中。 | 这项服务于 2012 年暂停。 |
| 8. | 文件未在本地加密。 | 文件在服务器上加密。 |
| 9. | 它不提供智能预测缓存。 | 它提供了智能预测缓存。 |
| 10. | 它提供了更高的数据安全性。 | 它提供的数据安全性相对较低。 |
