# 如何设置局域网？

> 原文：[https://www.geeksforgeeks.org/how-to-set-up-a-lan-network/](https://www.geeksforgeeks.org/how-to-set-up-a-lan-network/)

局域网（LAN）是一种数据通信网络，在本地连接工作站、服务器、路由器等网络设备。在一个小区域内共享资源，如一个建筑或校园。工作站之间建立物理或无线连接以共享资源。`Ethernet`和`Wi-Fi`是[**【LAN】**](https://www.geeksforgeeks.org/local-area-network-lan-technologies/)最重要的技术。家庭、学校、办公室等的个人网络都是局域网的例子。这些通常是私有网络。

## 设置局域网的要求

*   **工作站/个人设备：** 笔记本电脑、计算机、手机等。
*   **网络设备：** 路由器、交换机、调制解调器（如果路由器中未集成）。
*   **共享资源：** 打印机、磁盘驱动器等。
*   **线缆：** 以太网线缆，用于连接其他设备（在有线局域网情况下）。
*   **互联网连接：** `Wi-Fi`（在无线局域网情况下）。

### 设置局域网的说明

设置局域网应遵循以下步骤：

1.  **识别服务：** 识别将在工作站之间共享的网络服务，例如打印机、磁盘驱动器、数据等。
2.  **识别设备：** 识别将连接到网络的计算机、手机、笔记本电脑等设备，这些设备应具有唯一地址。
3.  **规划连接：** 通过布置线缆或在设备间建立无线连接来设计网络。有线局域网通过使用`Ethernet`线缆建立，而无线局域网则用于在没有任何物理连接设置的情况下连接网络设备。有线局域网比无线局域网更安全，但重新部署较困难。
4.  **选择网络设备：** 选择具有足够端口的交换机或路由器，以连接网络中的所有工作站。网络设备的选择基于网络需求。
5.  **配置端口：** 根据`ISP`（互联网服务提供商）提供的信息配置`WAN`端口。此外，配置有线路由器的`LAN`端口，以便网络中的所有工作站都有足够的可用地址。有线路由器充当`DHCP`（动态主机配置协议）服务器，并自动为连接到网络的所有设备分配地址。
6.  **建立连接：** 用线缆连接所有设备并配置局域网。使用标准`Ethernet`线缆连接工作站和服务器，而使用`Ethernet`交叉线缆通过将交换机的标准端口连接到路由器的`LAN`端口来连接交换机和有线路由器。对于无线局域网，使用路由器或交换机提供的`SSID`（服务集标识符）将所有设备连接到无线网络以配置局域网。
7.  **测试网络：** 测试连接到网络的每个工作站，以确保每个工作站都能访问网络服务。

### 设置提示

1.  在进行实际连接之前，对连接做一个全面的计划，以避免混淆。
2.  仔细确定网络的要求和规模，并进行相应的规划。
3.  明智地选择为网络提供更多灵活性的网络设备。
4.  确保线缆长度不超过 100 米。
5.  除非防火，否则避免在风道中铺设线缆。
6.  在网络建立后执行详细测试，以分析网络的实际性能。

## 局域网的应用

1.  **资源共享：** 局域网允许连接到网络的工作站共享打印机、扫描仪和光驱等资源。这降低了网络建立的成本。
2.  **软件共享：** 局域网允许在连接到网络的工作站之间共享一份许可软件的副本，而不是为每台计算机单独购买软件。
3.  **互联网共享：** 局域网便于所有连接到网络的设备共享互联网连接。
4.  **数据共享：** 局域网允许不同的工作站共享数据和文件。它还允许访问存储在中央服务器上的数据。
5.  **通信：** 连接到局域网的设备可以相互通信。

## 局域网的优势

1.  在小的地理区域内，这是一种简单且廉价的通信方式。
2.  通过将工作站连接到中央服务器，可以轻松扩展网络。
3.  可以轻松地从中央服务器管理资源和数据。
4.  它更安全，因为数据存储在中央服务器上，该服务器会拒绝非法访问数据。
5.  高数据传输速率。

## 局域网的缺点

1.  初始设置成本高。
2.  它侵犯了网络用户的隐私，因为管理员可以访问他们的所有数据和文件。
3.  如果中央服务器未得到适当保护，可能会面临安全问题。
4.  需要定期维护以处理软件安装、硬件故障、线缆干扰和其他问题。
5.  它限制了网络的规模。
