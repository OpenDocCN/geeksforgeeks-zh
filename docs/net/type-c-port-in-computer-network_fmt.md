# USB Type-C 端口详解

> 原文：[https://www.geeksforgeeks.org/type-c-port-in-computer-network/](https://www.geeksforgeeks.org/type-c-port-in-computer-network/)

`Type-C`（或 `USB-C`）是一种 24 针的 `USB` 连接器系统，其特点是具有双重旋转对称性。`USB Type-C` 规范 1.0 由 `USB Implementers Forum` 发布，并于 2014 年 8 月定稿。需要注意的是，实现了 `USB-C` 的设备不一定同时支持 `USB Power Delivery`、`USB 3.1` 或备用模式。

`USB` 连接器用于连接主机和设备，它旨在取代各种旧的 `USB` 连接器和电缆，成为一种面向未来的标准。`USB-C` 是一个 24 针的双面连接器，尺寸比 `micro-B` 连接器稍大，约为 8.4 毫米（0.33 英寸）乘 2.6 毫米（0.10 英寸）。该插座包含四个电源引脚和四个接地引脚、两对用于高速 `USB` 数据的差分对、四对用于增强型超高速数据的屏蔽差分对（其中两对发送，两对接收）、两个边带使用（`SBU`）引脚和两个配置通道（`CC`）引脚。

插头端通常只有一个高速差分对，其中一个 `CC` 引脚被 `VCONN` 取代，用于为电缆中的电子器件供电，而另一个 `CC` 引脚则用于实际传输配置通道信号。这些信号用于确定电缆的方向，并承载 `USB Power Delivery` 通信。要将旧设备连接到带有 `USB-C` 插座的主机，需要使用一端为 `USB-A` 或 `USB-B` 插头/插座、另一端为 `USB-C` 插头的电缆或适配器。

## C 型端口的优势

1.  **尺寸小巧**：该端口的宽度是标准 `USB` 连接器的一半，高度是三分之一，仅略大于苹果的 `Lightning` 或 `micro-USB` 连接器。
2.  **可逆插拔**：与旧的 `USB` 电缆不同，`C` 型插头没有正反面之分，可以任意方向插入。
3.  **高速传输**：`C` 型端口可以使用最新 `USB` 协议的高速传输数据，速率最高可达每秒 10 千兆位（`10 Gbps`）。
4.  **供电能力强**：新的连接器可以提供高达 100 瓦的功率，足以给笔记本电脑充电。
5.  **兼容性灵活**：通过适配器，用户能够将旧设备连接到 `C` 型端口，这意味着即使是使用非 `USB` 技术的设备（如 `DisplayPort`、`VGA` 和以太网）也能连接。

## C 型端口的缺点

1.  **设备支持不全**：作为最通用的连接器，仍然有一些设备不支持 `USB Type-C`。
2.  **存在安全风险**：近期的一些研究表明，新的 `USB Type-C` 端口在安全性方面较为薄弱。像 `BadUSB` 这样的恶意软件可以通过电缆中的固件在用户不知情的情况下感染笔记本电脑。
