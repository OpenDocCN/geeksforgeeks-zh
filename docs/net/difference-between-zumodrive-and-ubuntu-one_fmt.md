# ZumoDrive 和 Ubuntu One 的区别

> 原文：[https://www.geeksforgeeks.org/difference-between-zumodrive-and-ubuntu-one/](https://www.geeksforgeeks.org/difference-between-zumodrive-and-ubuntu-one/)

## 1. ZumoDrive
ZumoDrive 是一个文件托管服务、文件同步服务和基于云的文件存储服务，最初由 `Zecter Incorporation` 提供，但后来归摩托罗拉移动所有。由 `Zecter` 公司开发，适用于 `Windows`、`macOS`、`iOS`、`webOS`、`Linux` 和 `Android`。它提供了 `2 GB` 的免费存储空间，付费时可扩展至 `500 GB`。它于 `2009` 年推出。

## 2. Ubuntu One
Ubuntu One 是由 `Canonical Limited` 提供的文件托管服务、文件同步服务和云服务。这是一个基于 `OpenID` 的单点登录服务。它提供了在云上存储文件的服务，也提供了从云上传输音乐的服务。由 `Canonical` 有限公司开发，适用于 `Windows`、`OS X`、`iOS`、`Ubuntu`、`Android`。它提供了 `5 GB` 的可用存储空间。它于 `2009` 年推出。

## ZumoDrive 和 Ubuntu One 的区别

| ZumoDrive | Ubuntu One |
| --- | --- |
| 最初由`Zecter`公司拥有，但后来被`Motorola Mobility`拥有。 | 它由`Canonical Limited`所有。 |
| 它于 `2009` 年推出。 | 它也于 `2009` 年推出。 |
| 它提供了 `2 GB` 的可用存储空间。 | 它提供了 `5 GB` 的可用存储空间。 |
| 由`Zecter`公司开发。 | 由`Canonical Limited`开发。 |
| `2012` 年暂停。 | 这项服务于 `2014` 年暂停。 |
| 它有免费软件版本。 | 它也有免费软件版本。 |
| 存储在这里的文件被加密了。 | 存储在 Ubuntu One 中的文件没有加密。 |
| 这是一个文件托管和商业文件同步服务。 | 这是一个商业文件同步服务。 |
| 它得到了 `Windows`、`OS X`、`Linux`、`Android`、`iOS` 和 `webOS` 的支持。 | 它得到了 `Windows`、`OS X`、`Android`、`Ubuntu` 和 `iOS` 的支持。 |
