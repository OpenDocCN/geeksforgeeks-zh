在 C 中“`break`”和“`continue`”语句的上下文中，选择最佳语句。
**(A)** “`break`”可以用在“`for`”、“`while`”和“`do-while`”循环体中。
**(B)** “`continue`”可用于“`for`”、“`while`”和“`do-while`”循环体。
**(C)** “`break`”和“`continue`”可用于“`for`”、“`while`”、“`do-while`”循环体和“`switch`”体。
**(D)** “`break`”和“`continue`”可以用在“`for`”、“`while`”和“`do-while`”循环体中。但只有“`break`”才能用在“`switch`”身上。
**(E)** “`break`”和“`continue`”可以用在“`for`”、“`while`”和“`do-while`”循环体中。此外，“`continue`”和“`break`”可以用在“`switch`”和“`if-else`”体中。

**回答:****(D)**

**说明:**按照 C 标准，“`continue`”只能用于循环体。“`break`”只能用于回路体和开关体。这就是为什么正确答案是 D.
