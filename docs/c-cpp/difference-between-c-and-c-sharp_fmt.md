# C 和 C#的区别

> 原文: [`https://www.geeksforgeeks.org/difference-between-c-and-c-sharp/`](https://www.geeksforgeeks.org/difference-between-c-and-c-sharp/)

## [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)
[`C语言`](https://www.geeksforgeeks.org/c-programming-language/)是一种中级编程语言，由丹尼斯·里奇于 1972 年在贝尔研究实验室开发。[`C语言`](https://www.geeksforgeeks.org/c-programming-language/)结合了低级语言和高级语言的属性。因此它被认为是一种中间编程语言。[`C`](https://www.geeksforgeeks.org/c-programming-language/)可能是一种高级经典编程语言，允许你开发计算机代码和可移动应用程序。[`C语言`](https://www.geeksforgeeks.org/c-programming-language/)总共使用了 32 个关键词。这是开发计算机代码系统的完美语言。

## [`C#语言`](https://www.geeksforgeeks.org/csharp-programming-language/)
[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)是一种面向对象的编程语言，发音为 `C-Sharp`。这种语言是由微软在安德斯·海杰伯格和他的团队的领导下开发出来与 Java 竞争的。[`C#语言`](https://www.geeksforgeeks.org/csharp-programming-language/)总共使用了 86 个关键词。在[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)中，指针仅在不安全模式下使用。

## [`C`](https://www.geeksforgeeks.org/c-programming-language/)和[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)的区别

| S.NO | [`C`](https://www.geeksforgeeks.org/c-programming-language/) | [`C#`](https://www.geeksforgeeks.org/csharp-programming-language/) |
| --- | --- | --- |
| 1. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)支持过程编程。 | 而[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)支持面向对象编程。 |
| 2. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)支持指针。 | 而在[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)中，指针只在不安全模式下使用。 |
| 3. | 在[`C语言`](https://www.geeksforgeeks.org/c-programming-language/)中，垃圾收集不是。 | 而在[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)中，垃圾收集由 [`公共语言运行时(CLR)`](https://www.geeksforgeeks.org/common-language-runtime-clr-in-c-sharp/) 管理。 |
| 4. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)可以跨平台执行。 | 鉴于[`.NET框架`](https://www.geeksforgeeks.org/introduction-to-net-framework/)需要执行[`C#语言`](https://www.geeksforgeeks.org/csharp-programming-language/)。 |
| 5. | 通过使用[`C语言`](https://www.geeksforgeeks.org/c-programming-language/)，我们可以实现低层次的抽象。 | 而通过使用[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)我们可以实现高度的抽象。 |
| 6. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)更注重函数。 | 而[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)更注重设计。 |
| 7. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)给出了一流的性能。 | 而[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)给出了标准性能。 |
| 8. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)总共使用了 32 个关键词。 | 而[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)中总共使用了 86 个关键字。 |
| 9. | [`C语言`](https://www.geeksforgeeks.org/c-programming-language/)主要用于商业行业和工程。 | 而[`C#`](https://www.geeksforgeeks.org/csharp-programming-language/)用于软件形成和其他网络相关目标。 |
