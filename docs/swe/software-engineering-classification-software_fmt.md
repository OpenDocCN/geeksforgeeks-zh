# 软件工程 | 软件分类

> 原文: [https://www.geeksforgeeks.org/software-engineering-classification-software/](https://www.geeksforgeeks.org/software-engineering-classification-software/)

该软件广泛应用于医院、银行、学校、国防、金融、股票市场等多个领域。它可以分为不同的类型:

## 在应用的基础上:

### 1. System Software –
`System Software` 是管理计算机资源和支持应用程序执行所必需的。像操作系统、编译器、编辑器和驱动程序等软件都属于这一类。没有这些，计算机就无法运行。操作系统用于将程序的机器相关需求与运行它的机器的能力连接起来。编译器将程序从高级语言转换为机器语言。

### 2. Networking and Web Applications Software –
网络软件为计算机之间以及与数据存储设施之间的交互提供必要的支持。当软件运行在计算机网络（如万维网）上时，也会使用网络软件。它包括所有网络管理软件、服务器软件、安全和加密软件，以及用于开发基于Web的应用程序的软件，如 `HTML`、`PHP`、`XML` 等。

### 3. Embedded Software –
这种软件嵌入到硬件中，通常作为大型系统的一部分存储在只读存储器（`ROM`）中，用于在受控条件下支持特定功能。例如，用于仪器和控制应用的软件，如洗衣机、卫星、微波炉等。

### 4. Reservation Software –
预订系统主要用于存储和检索信息，并执行与航空旅行、汽车租赁、酒店或其他活动相关的交易。它们也提供对巴士和铁路预订的访问，尽管这些并不总是与主系统集成。这些系统也用于在酒店业中为用户传递计算机化信息，进行预订并确保酒店不会超订。

### 5. Business Software –
这类软件用于支持商业应用，是使用最广泛的软件类别。例如库存管理、会计、银行、医院、学校、股票市场等软件。

### 6. Entertainment Software –
教育和娱乐软件为教育机构，特别是那些从事幼儿教育的机构提供了强大的工具。娱乐软件种类繁多，如电脑游戏、教育游戏、翻译软件、地图软件等。

### 7. Artificial Intelligence Software –
像专家系统、决策支持系统、模式识别软件、人工神经网络等软件都属于这一类。它们涉及复杂的问题，这些问题不受使用非数值算法的复杂计算的影响。

### 8. Scientific Software –
科学和工程软件满足科学或工程用户执行特定企业任务的需求。此类软件是使用特定领域的原理、技术和公式为特定应用编写的。例如 `MATLAB`、`AUTOCAD`、`PSPICE`、`ORCAD` 等软件。

### 9. Utilities Software –
属于此类别的程序执行特定任务，在大小、成本和复杂性方面与其他软件不同。例如防病毒软件、语音识别软件、压缩程序等。

### 10. 文档管理软件 –
文档管理软件用于跟踪、管理和存储文档，以减少文书工作。这种系统能够记录不同用户创建和修改的各种版本（历史跟踪）。它们通常提供存储、版本控制、元数据、安全性以及索引和检索功能。

## 在版权基础上:

### 1. 商用 –
它代表了我们从软件公司、商用电脑商店等购买的大部分软件。在这种情况下，当用户购买软件时，他们会获得一个许可证密钥来使用它。不允许用户复制软件。该程序的版权归公司所有。

### 2. Shareware –
`Shareware` 软件也受版权保护，但购买者被允许制作和分发副本，条件是测试软件后，如果购买者决定使用它，则必须付费。

在上述两种类型的软件中，都不允许对软件进行更改。

### 3. Freeware –
一般来说，根据 `freeware` 软件许可证，可以制作软件副本用于存档和分发，但分发不能用于盈利。允许并鼓励对软件进行衍生作品和修改。即使没有版权所有者的明确许可，也允许对程序代码进行反编译。

### 4. 公共领域 –
在公共领域软件的情况下，原始版权所有者明确放弃软件的所有权利。因此，软件拷贝既可以用于存档，也可以用于分发，并且对分发没有限制。软件的修改和逆向工程也是允许的。
