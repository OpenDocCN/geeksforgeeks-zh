AJAX 是一种技术，而不是一种被开发者用来使网站像桌面应用程序一样运行的编程语言。它在客户端运行，用于创建异步 web 应用程序。AJAX 是一组使用多种网络技术来创建一套网络开发技术的技术。

jQuery 为 AJAX 功能提供了各种方法。

有一些 jQuery AJAX 方法用于使用 HTTP Get 和 HTTP Post 从远程服务器请求文本、HTML、XML 或 JSON。
