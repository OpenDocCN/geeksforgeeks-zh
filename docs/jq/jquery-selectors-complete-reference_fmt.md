# jQuery | 选择器完整参考

> 原文: [https://www.geeksforgeeks.org/jquery-selectors-complete-reference/](https://www.geeksforgeeks.org/jquery-selectors-complete-reference/)

jQuery 选择器用于选择 HTML 元素，并允许您以我们想要的方式操作 HTML 元素。它在一个变量参数上选择 HTML 元素，如它们的名称、类、id、类型、属性、属性值等。
