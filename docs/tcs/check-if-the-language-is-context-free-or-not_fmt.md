# 检查语言是否与上下文无关

> 原文: [https://www.geeksforgeeks.org/check-if-the-language-is-context-free-or-not/](https://www.geeksforgeeks.org/check-if-the-language-is-context-free-or-not/)

## 先决条件
- [抽取引理](https://www.geeksforgeeks.org/theory-of-computation-pumping-lemma/)
- [如何识别一种语言是否正规](https://www.geeksforgeeks.org/how-to-identify-if-a-language-is-regular-or-not/)

我们通常会面临一些问题，以识别哪些给定的语言是上下文无关的。对于常规语言来说，回答这个问题相对容易，但是对于上下文无关语言来说，这个问题有时很棘手。Pumping Lemma 为我们提供了执行否定测试的能力，即如果一种语言不满足 pumping lemma，那么我们可以肯定地说它不是上下文无关的，但是如果它满足，那么该语言可能是上下文无关的，也可能不是上下文无关的。泵引理更多的是一种数学证明，需要更多的时间，并且将其应用于上下文无关语言是一项乏味的任务，并且为复杂的语言表达式找到反例并不困难。基于常见的观察和分析，我们可以非常快速地解决这个问题:

### 1. 每种常规语言都是上下文无关的。
**示例–** `{a^m b^l c^k d^n | m, l, k, n >= 1}` 是上下文无关的，因为它也是常规的。

### 2. 给定一个表达式，如果可以在字符串中找到一个中心或中点，那么我们可以使用堆栈对左右子部分进行比较。
**示例 1 –** `L = {a^n b^n | n >= 1}` 是上下文无关的，因为我们可以推入 a，然后为每个出现的 b 弹出 a。

**示例 2–** `L = {a^m b^n c^(m+n)}` 是上下文无关的。我们可以改写为 `{ a^m b^n c^n c^m }`。

**示例 3–** `L = {a^n b^(2n)}` 是上下文无关的，因为我们可以为 b 的每次出现推两个 a 并弹出一个 a。因此，我们在这里也得到一个中点。

**示例 4–** `L = {a^n b^n c^n}` 并非上下文无关。

### 3. 给定的表达式是多个具有中点的表达式的组合，并且每个子表达式独立于其他子表达式，那么它是上下文无关的。
**示例 1 –** `L = { a^m b^m c^n d^n }` 是上下文无关的。它包含多个表达式，每个表达式都有一个中点。

**示例 2–** `L = {a^m b^n c^m d^n}` 并非上下文无关。

### 4. 给定的表达式由一个操作组成，在这个操作中可以找到中间点以及中间的一些独立的正则表达式，从而得到上下文无关的语言。
**示例–** `L = {a^m b^i c^m d^k}` 是上下文无关语言。
这里，我们有 `b^i` 和 `d^k` 作为中间独立的正则表达式，这不会影响堆栈。

### 5. 没有形成可以使用堆栈进行线性比较的模式的表达式不是上下文无关语言。
**示例 1–** `L = { a^m b^(n^2) }` 不是上下文无关的。
**示例 2–** `L = { a^n b^(2^n) }` 不是上下文无关的。
**示例 3–** `L = { a^(n^2) }` 不是上下文无关的。
**示例 4–** `L = {a^m | m 为素数}` 不是上下文无关的。

### 6. 涉及三个或更多变量独立计数和比较的表达式不是上下文无关语言，因为堆栈一次只允许比较两个变量。
**示例 1 –** `L = { a^n b^n c^n }` 不是上下文无关的。

**示例 2–** `L = { w | na(w)= nb(w)= NC(w)}` 不是上下文无关的。

**示例 3–** `L = {a^i b^j c^k | i>j>k }` 不是上下文无关的。

### 7. 需要记住的一点是，在下推自动机中，计数和比较只能与堆栈顶进行，而不能与堆栈底进行，因此表现出涉及与堆栈底比较特征的语言不是上下文无关语言。
**示例 1 –** `L = { a^m b^n c^m d^n }` 不是上下文无关的。
推入 a 然后推入 b。现在，我们将无法将 c 与 a 进行比较，因为堆栈顶是 b。

**示例 2–** `L = { WW | W 属于{a，b}* }` 不是上下文无关的。
人们可能会想画一个非确定性的下推自动机，但这没有帮助，因为第一个符号将在堆栈的底部，当第二个 W 开始时，我们将无法将其与堆栈的底部进行比较。

### 8. 如果我们能够以非确定性的方式在表达式中找到中点，那么它是上下文无关语言。
**示例 1 –** `L = { W W^r | W belongs to {a, b}* }` 是上下文无关语言。

**示例 2–** `L = {a^i b^j c^k d^l | i = k 或 j=l }` 是上下文无关语言。
