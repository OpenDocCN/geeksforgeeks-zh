# SASS | 数字函数

> 原文: [https://www.geeksforgeeks.org/sass-numeric-functions/](https://www.geeksforgeeks.org/sass-numeric-functions/)

就像 SASS 字符串函数一样，SASS 给出了一组简单的函数来更改`数值`，这些值按照您希望的方式工作。请注意，除了`percent()`之外，可以给每个函数赋予 CSS 值描述符，如`px`或`rem`。在这种情况下，SASS 将忽略这些单位，它们不会显示在结果中。

下表列出了 Sass 中的所有数值函数:

| 功能 | 描述 | 例子 |
| --- | --- | --- |
| `percentage(数字)` | 此方法将无单位数字转换为百分比。意思是，乘以 100。 | `percentage(2.5)`<br>**输出**: `250`<br>`percentage(4em)`<br>**输出**: 语法错误 |
| `round(数字)` | 此方法将数字舍入到最接近的整数。 | `round(2.25)`<br>**输出**: `2`<br>`round(2.5)`<br>**输出**: `3` |
| `ceil(数字)` | 此方法将数字向上舍入到最接近的整数。 | `ceil(2.25)`<br>**输出**: `3` |
| `floor(数字)` | 此方法将数字向下舍入到最接近的整数。 | `floor(2.25)`<br>**输出**: `2` |
| `abs(数字)` | 此方法返回数字的绝对值。 | `abs(2)`<br>**输出**: `2`<br>`abs(-2)`<br>**输出**: `2` |
| `min(数字)` | 此方法返回数字列表中的最小值。 | `min(4, 5, -7, 8, 2)`<br>**输出**: `-7` |
| `max(数字)` | 此方法返回数字列表中最大的值。 | `max(4, 5, -7, 8, 2)`<br>**输出**: `8` |
| `random()` | 此方法返回一个范围为 `[0, 1]` 的随机数。 | `random()`<br>**输出**: `0.4965` |
| `random(极限)` | 此方法返回一个在 `[1, limit]` 范围内的随机整数。 | `random(4)`<br>**输出**: `2.4652` |
