# jQuery UI 可选完整参考

> 原文：[https://www.geeksforgeeks.org/jquery-ui-selectable-complete-reference/](https://www.geeksforgeeks.org/jquery-ui-selectable-complete-reference/)

**jQuery UI 可选小部件**用于执行选举事件，该小部件使用鼠标单独或成组选择元素。这个小部件中有很多选项、方法和事件，它们都在下面列出和分类。

## jQuery UI 可选小部件选项

*   `attach`选项
*   `autoRefresh`选项
*   [`cancel`选项](https://www.geeksforgeeks.org/jquery-ui-selectable-cancel-option/)
*   [`classes`选项](https://www.geeksforgeeks.org/jquery-ui-selectable-classes-option/)
*   `delay`选项
*   [`disabled`选项](https://www.geeksforgeeks.org/jquery-ui-selectable-disabled-option/)
*   `distance`选项
*   `filter`选项
*   `tolerance`选项

## jQuery UI 可选小部件方法

*   `destroy()`方法
*   `disable()`方法
*   `enable()`方法
*   `instance()`方法
*   `option()`方法
*   `refresh()`方法
*   `widget()`方法

## jQuery 用户界面可选小部件事件

*   [`create`事件](https://www.geeksforgeeks.org/jquery-ui-selectable-create-event/)
*   `selected`事件
*   `selecting`事件
*   `start`事件
*   `stop`事件
*   `unselected`事件
*   `unselecting`事件
